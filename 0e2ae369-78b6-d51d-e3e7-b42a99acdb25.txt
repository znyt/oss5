　　洪荒之中，灵神界的兴起，不知道会带给洪荒怎样的变化，其他的不说，单单就各种族之间的实力，便会增强一倍不止。
<br/>　　历史的车轮永远向前不断驶去，不会停止。不管多了一个灵神界，还是少了某个人，依旧不能让它变动分毫。岁月悠悠，终究是要向前看的。
<br/>　　时间长河，奔流不息，转瞬间，洪荒已经过去了五百年。
<br/>　　洪荒，还是以前那个洪荒，没有改变多少，变化的只是，洪荒之中的高手增多了，各族之间的摩擦，也变多了。
<br/>　　因为有着灵神界的缘故，洪荒万族之间的战斗，终究是影响到了洪荒之中的势力，灵神界的因果，被牵引到了外界了结。导致洪荒之中，各族都蠢蠢欲动，战斗不止。
<br/>　　但这些都没有改变什么，人族依旧是天地主角，洪荒第一大族，有伏羲衍八卦，尊人族共主之位，治世天下，人族之中，高手不断涌现，人族也变得越来越强盛了，不是一般种族可以招惹的。
<br/>　　此时，是一个夜晚，如往常一样，洪荒之中，一轮明月高高升起，月如银盘，月华垂下，洒下苍茫大地，巨大的至尊太阴星高高挂起，给人一种宁静的气息。月辉似水，银雾挥洒，倾泻而下，如一片片薄纱，淡雅朦胧，给人一种祥和而宁静之感。
<br/>　　人族祖地，首阳山上，最高峰，武宫玄天殿中。
<br/>　　唰！
<br/>　　忽然，首阳山震动起来，风云狂舞，海量的精气散发出磅礴的光辉，一阵灵气如同潮汐般卷起，灵气如海，不断涌动着，排山倒海而来，将此地给淹没了。
<br/>　　武塔之中，玄天双眼睁开，如神芒透射而出，里面似乎酝酿着混元的意境，玄奥不可言语。
<br/>　　“岁月轮，生死轮，命运轮，此时不融，更待何时？！”
<br/>　　只听玄天一声冷喝，双眼中神光爆闪，更有一丝坚定，如同一阵雷霆轰鸣，在他体内轰然响起。
<br/>　　在玄天的脑后，突然缓缓浮起三道神轮，每一道神轮都仿佛是一轮昊阳，悬浮于他的身后。正头顶上的神轮带着妖异的红色，有无尽梵唱声响起，像是在歌颂，又仿佛在祝福，隐隐约约可以听到一阵阵神秘的声音响起：
<br/>　　“伟大的宿命，永恒的旋律，它的光，无量普照，世界的每一个角落！”
<br/>　　“永恒的命运，在黑暗之中徘徊，无人能够脱离它的掌控，他是永恒的主人，违背宿命之人，将接受它的制裁！”
<br/>　　“命运的力量，它凌驾于诸天之上，主宰着众生的命运！”
<br/>　　“命运之下，无人可逃，亿万虚空，微尘之数，无量世界，无法逃脱的命运纠缠，永远注定的宿命纷争……”
<br/>　　“……”
<br/>　　此乃命运的力量，天地间最神秘的力量之一。
<br/>　　命运，因果，时间，空间，轮回，造化等几种法则，都是天地间最强的几种法则。
<br/>　　空间为王，时间为尊，命运不出，因果称王，造化无双，轮回无限！
<br/>　　这可不是说说而已，而是确有此事。道无大小，但法则却有强弱。
<br/>　　命运、因果等法则，却是天地间最强的几种法则。
<br/>　　此红色神轮乃命运轮，便是带着一丝命运法则的神轮。试问天地间没有真正超脱大道者，有谁能逃过命运的掌控？这便是命运的强大，他无处不在，无人可逃脱。
<br/>　　左边的是乳白色的神轮，带着一股沧桑的气息，有无尽岁月在里面流转，好似轻轻一划，便能带走无数时光，无声无息，真正的大恐怖！
<br/>　　此为岁月轮，带有一丝时间法则的神轮，岁月无声无息，洪荒之中，玄天也没有听说过有谁能逃得过时间的侵蚀。
<br/>　　任你风华绝代，傲绝古今，但依旧逃不过时间的力量，岁月一样会在你身上留下痕迹，更甚者，让你直接成为一具红粉骷髅。
<br/>　　这便是岁月的无息，岁月的恐怖！
<br/>　　右上方的是一轮黑白双色的神轮，散发着生与死的力量，生死互相转化，生之极致是死，同样的，死之极致是生，生死乃是贯穿轮回的法则，只要不超脱，没人能逃得过生死。
<br/>　　人生一世，草木一秋，不过生死二字而已。
<br/>　　这是生死轮，带着浓浓的生死法则，生死无限，苍茫无极。
<br/>　　命运轮，岁月轮，生死轮，乃是玄天的三大转轮。
<br/>　　月色依旧，光辉不变，清风拂过，明月照江，树枝摇曳，发出嗽嗽的响音，如同一道莫名的仙乐，似混元在吟唱，杂乱而含有深意。
<br/>　　三道神轮在缓缓靠近，它们在不断的融合，紫金光华绽放，散发出凛然不可侵犯的威势，如生灵，不可冒犯！
<br/>　　“轰！”
<br/>　　也不知过了多久，最终三道神轮终于完美的融合在了一起，变成一道全新的神轮。三者相互补足，神轮变得完美起来，散发出无比璀璨的紫金光芒。
<br/>　　紫金色的神轮悬浮于他身后，宛若神灵傲世，神圣不可侵犯！
<br/>　　“吾之武道，不需要多少，只需要一道便可以了。一道破万道，一法破万法！任你何等厉害，我自一道镇压。所以，三道神轮必须融合为一，才算圆满。”
<br/>　　“接下来，便是天、地、人三道洞天合一了。”
<br/>　　玄天喃喃自语道，他的眼神之中，闪过非常坚定的神色，那是对道的执着，对道的追求！
<br/>　　不成道，毋宁死！
<br/>　　此刻，玄天的头颅之中，散发出金色光芒，隐隐有神秘的空间在里面出现。随后，便听到一阵阵神秘的吟唱声传出，宛若诸神在吟唱，使得他的大脑一阵清明，元神一阵明朗。
<br/>　　紧接着，在他的胸部，依旧有一道神秘的空间浮现，在里面，好似能够看到刀耕火种，燧人点火，茹毛饮血，结网捕鱼，驱赶妖兽等人族先贤显现的画面，那里面蕴含的是，人道的气息，人道的文明。

<br/>　　此乃武道洞天尊者境时开辟的天、地、人三洞天。
<br/>　　玄天白衣出尘，盘坐在武塔之内，黑发飞舞，全身闪光绽放，肌体晶莹，脊骨若龙，他像是一尊神王，祥和而平静，英武而无双，引动这虚空的力量，化为神锤，淬炼己身。
<br/>　　三道浩大的神秘洞天，就仿佛三方世界，在他体内浮浮沉沉，轮转不休。
<br/>　　“轰隆隆！”
