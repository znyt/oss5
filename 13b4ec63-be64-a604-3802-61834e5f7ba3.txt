　　第199章
<br/>　　蛤蟆寨，大帐内。
<br/>　　李善言和齐德强刚才差点填了粪坑，此时才算是被请进来，依旧惊魂未定。
<br/>　　大帐内，秦蛮，林少羽都握着枪，关鱼，小白狼几人攥着刀柄。程大雷把斧头搁在桌上，有一下没一下的打量着斧刃。
<br/>　　“你说这斧头能刮胡子么？”
<br/>　　“不如大当家试试。”徐神机。
<br/>　　“我试什么呀，我又没胡子。”程大雷撇撇嘴，忽然眼睛一亮：“李军师，您的胡子我看着可有点长……”
<br/>　　“程寨主取笑了，程寨主取笑了。”李善言忙摆手。
<br/>　　“老程这个人一直都这么风趣。”齐德强：“这一点我是有发言权的。”
<br/>　　“当然，谁不知道齐将和程寨主亲如兄弟。”李善言扶须道。
<br/>　　“你们究竟来干嘛？”程大雷磨着斧刃。
<br/>　　“程寨主这次力挽狂澜，坚守青牛山，大败戎族。程寨主英雄人物，实在是帝国架海紫金樑，擎天白玉柱。”
<br/>　　“李军师你的胡子实在有些长。”程大雷拎着斧头要站起来。
<br/>　　“程寨主，程寨主……”李善言忙道：“黑石，落叶，赤沙，铁河四城中，程寨主可以任选其一，由我家主公上奏朝廷，封给程寨主。”
<br/>　　程大雷把斧头放下：“什么个意思？”
<br/>　　“程寨主不居功，我家主公难道还能不惦记着，总是不能让程寨主流血又流泪。”李善言呵呵笑着：“明日，就在明日，我家主公准备在风雨亭设宴，幽州境的乡绅富豪，兵丁将官都会到。也要让他们知道，是谁救了他们，是谁护住了他们身家性命。”
<br/>　　李善言大手一挥：“诸家兄弟可是都要去，幽州百姓都等着看诸位英雄呢。”
<br/>　　“又是风雨亭？”程大雷皱起眉头。
<br/>　　“当然，明日之前，程寨主最好确定是那座城，我家主公也好早作安排。”
<br/>　　李善言和齐德强起身告辞，临走前还拍拍这个的肩膀，捶捶这个的胸口，说些『帝国虎将』『少年英雄』『栋梁之才』之类的话。
<br/>　　二人走后，蛤蟆寨诸人凑在一起，嘀嘀咕咕议论着今天的事情。
<br/>　　“却是没想到，这一仗我们并不白打，杨龙停会送给我们一座城。”徐神机。
<br/>　　“大当家你觉得，咱们要那座城最合适？”
<br/>　　“黑石城已被屠尽，肯定不在考虑之中。”
<br/>　　“那就只剩下落叶、赤沙、铁河三城。”
<br/>　　“不如就选落叶城，我们对落叶城还熟悉些。”
<br/>　　众人议论纷纷，皆为今天的消息而兴奋。做城主自然比做山贼有前途得多。起码名义上就顺了，不像山贼一样人人喊打。拥有一座城池，就可以堂堂正正的做生意，潜心攀科技树。失了一座山寨，却得到一座城，这基本盘比之前壮大许多。
<br/>　　这便是塞翁失马焉知非福的道理呐。
<br/>　　黑石，铁河，落叶，赤沙在这四座城池中，程大雷更属意的是落叶城，落叶城有港口，船来船往也方便。北地也不是一点特产也没有，如兽皮，人参，来自戎族的马匹都是南地稀罕的。而南地的茶叶，香料运过来也是一本万利。
<br/>　　山寨中，诸人议论纷纷，讨论着明日该选那座城合适，而程大雷心里大约已经有了主意。
<br/>　　次日，天还没亮，蛤蟆寨的山贼就集结在一起，出发前往风雨亭。
<br/>　　大家心中明白，今日离去怕就是永远告别此地，即便再回来，也绝不是以山贼的身份。
<br/>　　如果可以的话，谁愿意顶着一个山贼的身份呢。
<br/>　　千余人沿着官道出发，程大雷骑马走在最前方，一路有说有笑。
<br/>　　“守法朝朝忧闷，强梁夜夜欢歌，损人利己骑马骡，正值公平挨饿。修桥补路瞎眼，杀人放火儿多，我到西天问我佛，佛说：我也没辙。”
<br/>　　正行在路上，打前方路上迎面走过来两个老道，一大一小，且行且歌，老道士手中晃悠着一张白幡，上书四个大字：看破人间。
<br/>　　程大雷认得这老道，姓郭名烦人的就是，程大雷不是很待见对方，双方擦肩而过，谁也没有说话。
<br/>　　“喂，你要去哪儿？”老道走过几步，忽然停住，回头喊了一声。
<br/>　　“我往前面走啊。”程大雷也停住。
<br/>　　“别走了，回去吧，前面走不通。”
<br/>　　“怎么走不通，前面修路呢？”
<br/>　　老道晃悠着来到程大雷身边，一个马上一个马下。
<br/>　　“老道粗通义理，略晓阴阳，云游天下，与人算命为生。程寨主可愿算让老道替你算上一算。”
<br/>　　看着他肩上的白幡，程大雷道：“你会什么啊，摸骨相面，八字流年，阴阳命理，测字星相？”
<br/>　　“呵呵……这些……”老道扶须微笑。
<br/>　　“都会？”
<br/>　　“一概不会。”
<br/>　　“那塔罗星座，紫微斗数，唤灵请神，实在不行你给我来段相声得了。”
<br/>　　“……”老道。
<br/>　　“那你究竟会什么？”
<br/>　　“流年大运，向来不会，相面摸骨，一窍不通。贫道只会一件事……”老道一晃手中白幡：“看破人间，仙人指路。”
<br/>　　“我的路你也能指么？”
<br/>　　“略知一二。”老道手中白幡一晃：“向前走，已然无路，继续走，便是死路。”
<br/>　　“向后走呢？”
<br/>　　“向后走，乃是绝路，九死一生。”
<br/>　　“那麻烦仙长，我的生路在何处？”
<br/>　　白幡一晃，划过周围：“四方神州，八方乾坤就是程寨主的生路。”
<br/>　　程大雷皱眉深思，老道扶须微笑，目光落在程大雷身上，看上去心怀悲悯，仙风道骨，实在的陆地神仙，三清真人。
<br/>　　程大雷突然从马上跃下来，一拳砸在老道头上。
<br/>　　“啊，我打！”
<br/>　　“哎呦，哎呦。”
<br/>　　“让你丫不好好说话，让你丫装。”
<br/>　　“哎呦，程寨主不要打了，程寨主手下留情。”
<br/>　　“啊，我打！”
<br/>　　“不要打了，不要……”老道忽然吼了一声：“姓程的，你死到临头，还不自知么？”
<br/>　　程大雷一顿，伸出去的拳头收回去，此刻是气喘吁吁。老道现在是鼻型脸肿，顶着个熊猫眼，胡子也被扯掉几缕。
<br/>　　“现在会好好说话了？”
<br/>　　“哼，姓程的，你即将大祸临头，可怜你身在梦中，呼呼大睡。”
<br/>　　
<br/>(本章完)
