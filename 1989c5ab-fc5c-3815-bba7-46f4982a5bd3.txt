　　萧逸一行人休息了半个时辰左右，再次出发。
<br/>　　雷鸣荒泽内，果真是妖兽横行，萧逸都不记得有多少妖兽死在林飞的箭下和铁达大叔的刀下了。
<br/>　　铁达大叔等人的乾坤袋都装满了妖兽尸体。
<br/>　　正常情况下，武者是不敢这样疯狂猎杀妖兽的。
<br/>　　很多时候都是完成了任务便赶紧离开，只有遇到无法避免的战斗，才会和妖兽斗上一番。
<br/>　　否则，一旦消耗光真气，在这雷鸣荒泽是非常危险的，很可能没命走出去。
<br/>　　但现在有萧逸给他们炼制隐气丹和驱兽丹，便没了这些后顾之忧。
<br/>　　黄源拍了拍自己的乾坤袋，笑道，“啧啧，和易霄兄弟一起出任务，这感觉贼爽。”
<br/>　　林飞也打趣道，“像这般酣畅淋漓的在雷鸣荒泽随意战斗，以前可不敢想象。”
<br/>　　“好了。”铁达大叔认真地说道，“你们的真气都消耗了七八成以上吧，赶紧吞服隐气丹和驱兽丹，这段时间我们都不要战斗了，争取在赶到洞府前恢复所有真气。也不知洞内会有什么样的危险。”
<br/>　　.....
<br/>　　经过一番安全的赶路，恰好在众人体内真气都恢复后，铁达大叔说道，“我们到了。”
<br/>　　“到了？”众人面露疑惑。
<br/>　　萧逸看了眼周围，除了密布的树木外，根本没有洞府。
<br/>　　“呵呵。”铁达大叔神秘一笑，往前走了几步。
<br/>　　忽然，他的身影在众人的眼中凭空消失。
<br/>　　“怎么回事，铁达大叔呢？”林飞等人吓了一跳。
<br/>　　萧逸先是皱眉，而后眉头一松，惊疑道，“难道是传说中的灵气屏障？”
<br/>　　“灵气屏障？”
<br/>　　众人听到这四字，立刻明白了过来。
<br/>　　铁达大叔又忽然出现，笑道，“聪明，就是灵气屏障。”
<br/>　　众人跟随铁达大叔的脚步，向前走了几步，眼前的景象瞬间大变。
<br/>　　一个恢宏的洞府赫然出现在萧逸眼前。
<br/>　　远远看去，方圆十里，皆在洞府的范围内，洞府内的一座座宫殿，极为雄伟。
<br/>　　萧逸不禁赞叹道，“不愧是狂血玄君，竟然用灵气屏障将整个洞府都包裹了起来。”
<br/>　　洞玄境武者，已经能掌控天地灵气，像慕容墨，挥挥手就能控制方圆数百米的天地灵气禁锢别人。
<br/>　　狂血玄君乃是最强大的破玄境武者，他对天地灵气的掌控，自然恐怖得骇人。
<br/>　　破玄境，比洞玄境整整高了一个大境界。
<br/>　　方圆十里的天地灵气，竟都被他调动起来，形成一个屏障，将整个洞府包裹住。
<br/>　　屏障外，外人就算近在咫尺，也不会看到这里有任何东西。
<br/>　　屏障内，则是别有洞天。
<br/>　　武道九大境界，凡境、后天、先天、洞玄、破玄、地元、天元、地极、天极。
<br/>　　铁达大叔笑道，“可不是嘛，以狂血玄君的本事，就算死了，他的洞府也非常人能够发现的。”
<br/>　　“要不是那猎妖师运气好，误打误撞跑进来这里，恐怕永远也不会有人发现他的洞府。”
<br/>　　铁达大叔沉声说罢，众人往洞府而去。
<br/>　　洞府内，引入眼帘的，首先是一个巨大的广场。
<br/>　　经过广场，再里面才算是一座座宫殿，狂血玄君死后长眠的地方。
<br/>　　然而，广场内，竟已有许多猎妖师，看这人数，不下千人。
<br/>　　“其他猎妖队早就来了。”萧逸道。
<br/>　　铁达大叔笑道，“我都能收到消息，其他猎妖队自然也是能的。”
<br/>　　“再者，事关狂血玄君的洞府，那第一个发现的猎妖师想瞒也瞒不住。”
<br/>　　这时，一群人快速走来，为首的是一个中年男子。
<br/>　　“铁达。”中年男子率先打了个招呼。
<br/>　　“王羽。”铁达大叔似乎与此人交情不错。
<br/>　　王羽，破玄城有名的猎妖师之一，先天五重实力，比铁达大叔还要厉害。
<br/>　　一旁的杜新急道，“队长，别叙旧了，先进去吧，不然宝物都被人抢光了。”
<br/>　　“哈哈。”王羽笑道，“别担心，这里只是洞府前的广场，宝贝都在宫殿里。大家都怕宫殿内有危险，这几天都没人敢进去呢。”
<br/>　　“不过。”王羽顿了顿，道，“今天大家都耐不住了，准备进入。毕竟是狂血玄君留下来的宝物，个个都狂热得很。”
<br/>　　“对了。”王羽继续道，“铁达，我过来，一是打个招呼，二则是告诉你个坏消息，巨斧猎妖队也来了。”
<br/>　　“巨斧猎妖队？”铁达大叔顿时面露杀意。
<br/>　　王羽说道，“铁达，我知道你们的过节。你看，要不我们两支猎妖队联手，一来可以在宫殿有个照应，二来你也不用担心那群家伙找你麻烦了。”
<br/>　　“找我麻烦，哼，我才不怕。”铁达大叔冷冷地说道。
<br/>　　“至于联手，还是算了。就怕在宫殿里找到宝物，我俩倒是没所谓，可是我们还有队员，因此而产生纠纷就不好了。”
<br/>　　王羽叹了口气，道，“既然如此，也罢，你小心些。”
<br/>　　说罢，王羽转身离去。
<br/>　　萧逸问道，“铁达大叔，那巨斧猎妖队过你有过节？”
<br/>　　铁达大叔点点头，但没多说什么。
<br/>　　“诶呀。”一旁的杜新看到已经有猎妖队进入宫殿了，顿时急了。
<br/>　　“队长，我们也先进去好不好。”
<br/>　　铁达大叔思索了一下，点头道，“过些天，肯定会有越来越多的猎妖师和武者来这里争夺宝物，我们也快些进去吧。”
<br/>　　众人越过广场，来到宫殿前。
<br/>　　数十座宫殿整齐的排列着，每座宫殿，又有十数道门。
<br/>　　同样，这些宫殿被一层灵气包裹着。
<br/>　　萧逸等人随便挑了间宫殿，随便推开一门，走了进去。
<br/>　　这是一个很大的房间。
<br/>　　嗖，一道黑影忽然袭来。
<br/>　　“小心。”铁达大叔眼疾手快，一刀将黑影劈成两半，是一头荒鳄。
<br/>　　“怎么回事，这里又不是沼泽，哪来的荒鳄...”
<br/>　　话还未说完，数道黑影从旁边袭来，攻击的是林飞等人，同样是荒鳄。
<br/>　　“孽畜，找死。”林飞一箭射出，黄源则一拳轰出。
<br/>　　然而，意料中的荒鳄被杀死并没有出现。
<br/>　　反而是这两人瞬间被击飞。
<br/>　　“噗。”两人猛地吐出一口鲜血，不可置信道，“好强。”
<br/>　　两人被击飞，荒鳄朝萧逸和浅沫攻来。
<br/>　　“哼。”萧逸冷哼一声，一剑劈出。
<br/>　　锵的一声，激起一阵火花，荒鳄毫发无损，反倒是萧逸被震退了几步。
<br/>　　恰在此时，铁达大叔反应过来，反手一刀砍过，将数头荒鳄毙命。
<br/>　　“不对劲，这些荒鳄的实力很强，起码是后天九重。”萧逸沉声说道，他一试便试出了刚才那几头荒鳄的实力。
<br/>　　正常的荒鳄只有后天五重实力，这到底怎么回事？
<br/>　　同一时间，房间内竟齐刷刷地出现了数十头荒鳄。
<br/>　　众人的脸色瞬间难看起来。
<br/>　　数十头荒鳄张开血盆大口，嘶吼着，仿佛一群被困了无数年，饿坏了的噬人恶兽。
<br/>　　“该死，保护好自己，我会用最短的时间杀光它们。”
<br/>　　铁达大叔吩咐一声，拔刀而上。
<br/>　　这些荒鳄都是后天九重实力，林飞等人根本不是对手。
<br/>　　铁达大叔一人力敌数十头荒鳄，竟不落下风。
<br/>　　但，荒鳄的数量毕竟多，有一头荒鳄越过了他，径直朝萧逸等人而来。
<br/>　　林飞、黄源和杜新瞬间吞服下巨力丹，将修为暂时提升到后天八重左右，三人联手拦下了这头荒鳄。
<br/>　　果然，这些荒鳄都是后天九重的实力。
<br/>　　三人联手，竟无法击杀它，仅仅是牵制住它。
<br/>　　然而，这时，越来越多的荒鳄越过铁达大叔，往这边过来。
<br/>　　铁达大叔毕竟只有一人，根本无法一次性拦下所有荒鳄。
<br/>　　“糟了。”林飞等人惊呼一声。
<br/>　　他们三人已经被两头荒鳄逼到了墙角，根本无法施以援手。
<br/>　　“快救易霄和浅沫。”林飞大喝一声。
<br/>　　然而，铁达大叔被十数头荒鳄包围住，根本无法脱身。
<br/>　　谁都料想不到，这才刚刚进入宫殿，已经遇到了这么大的危机。
<br/>　　“吼。”荒鳄们发狂似地攻来。
<br/>　　“易霄，怎么办？”浅沫也急了。
<br/>　　“该死。”萧逸暗暗皱眉。
<br/>　　要不是主脉受损，他杀这些荒鳄如杀鸡般简单。
<br/>　　“上来。”萧逸对浅沫喝了一声。
<br/>　　“阿？”浅沫微微一愣。
<br/>　　“阿什么，想被吃掉吗？”萧逸喝了一声，一把背起浅沫。
<br/>　　“吼”，荒鳄一口咬来。
<br/>　　嗖，萧逸的身影瞬间消失在原地。
<br/>　　他杀不了这些荒鳄，但不代表这些荒鳄杀得了他。
<br/>　　房间内，铁达大叔快速地斩杀着荒鳄，林飞三人苦苦支撑。
<br/>　　还有一道火红色身影，在这巨大房间中快速跳跃着。
<br/>　　“好快的速度。”林飞瞳孔一缩，自己的眼睛，竟丝毫跟不上萧逸的速度。
<br/>　　“切。”杜新不屑道，“他一个后天一重，战力这么弱，要没有点保命的手段早死了。”
<br/>　　另一边，铁达大叔见萧逸和浅沫没事，松了口气。
<br/>　　萧逸快速在荒鳄间腾转挪移。
<br/>　　刚躲开一只荒鳄，背后又是一只袭来。
<br/>　　“阿，易霄，后面有荒鳄。”浅沫惊呼一声。
<br/>　　萧逸眼光一凛，脚步一动，踩着后面荒鳄的脑袋瞬间跳跃开。
<br/>　　嗖嗖嗖，火红色身影的速度快得吓人。
<br/>　　“易霄，后面又有荒鳄了。”浅沫再次惊叫。
<br/>　　“浅沫，闭嘴行不行，我耳朵被你吼得很疼。”萧逸的身体柔软地再次躲过荒鳄的追击。
<br/>　　“我怕嘛，阿，又来了。”浅沫吓得双手乱动。
<br/>　　荒鳄那血盆大口，一口就能把她那小身板吞了。
<br/>　　“喂，别乱动，捂我眼睛了，看不到路，放开。”萧逸忽然被挡住视线，顿时大惊。
<br/>　　如此危急之时，稍有差池，后果严重。
<br/>　　“吼”，不知何时起，一头荒鳄从侧面咬来。
<br/>　　萧逸堪堪反应过来，但为时已晚，只能下意识地用手臂抵挡。
<br/>　　嘶的一声，荒鳄锋利的牙齿直接在手臂上划出一道巨大的伤口。
<br/>　　霎那间，手臂鲜血狂喷，贱湿了他的脸庞，染红了他的衣衫。
