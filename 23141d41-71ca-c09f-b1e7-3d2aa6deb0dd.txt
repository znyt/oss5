　　第374章
<br/>　　对方出奇的外表引起了程大雷注意，是个男人，金色的头发粘在一起，嘴角挂着淤血，双手被麻绳绑着拴在马后。
<br/>　　这很明显不是中土人士。程大雷在草原待这么久，也听说过一些传言。
<br/>　　传言在琴川关外，跨过沙漠和山地有个极西之国。那里人的相貌，风土人情和帝国完全不同，他们被称之为夷族。
<br/>　　经常也有夷族的商队到这里来，所以此地对于夷族并不陌生，银眸体内就有一半夷族的血统。
<br/>　　程大雷对这传说中的极西之国还是很好奇的，只不过在这里待了这么久，一次夷族商队也没碰见，今天还是第一次见到夷族的人。
<br/>　　他目光落在那伙戎族身上，问：“你叫什么名字？”
<br/>　　“黑犬部乌角。”这人以手摁胸膛。
<br/>　　程大雷目光落在那夷族身上，道：“这个人我要了，你拿我的箭去琴川关换粮。”
<br/>　　程大雷抛出一支弩箭，乌角抬手接住，笑道：“原本就打算给程当家送过去，程当家的粮食，我们可不好意思拿。”
<br/>　　“好啦，好啦，你们戎族什么时候也学得这么客套。”程大雷挥挥手：“这个人怎么来的？”
<br/>　　“草原上碰到的，被抓到后一句话也没说。”乌角。
<br/>　　这夷族被送到程大雷面前，程大雷挥剑将他手上的麻绳斩断，目光打量着他。
<br/>　　“我是琴川关程大雷，喏，你什么来历？”
<br/>　　男人看着程大雷，眼神中有不安、困惑、恐惧，同时也有好奇。
<br/>　　“怎么，不会说帝国话？”
<br/>　　虽然在和戎族的战争中，帝国屡战屡败，但毫无疑问帝国的文化是强大的。戎族没有自己的语言、文字，虽然某些小部落有古怪的方言，但大多还是以帝国话为通用语。
<br/>　　至于极西之国的情况，程大雷并不了解，说不得他们就不会这边语言呢。
<br/>　　“诺顿王国凯恩·道尔，见过程当家。”男人单手微微低头。帝国话说得还很标准，只是口音有些怪。
<br/>　　极西之国只是帝国对人家的称呼，在人家那里，自称为诺顿王国。
<br/>　　本来打算今天和樊梨花出来散心，却是没想到遭遇凯恩，好好的散心如此也就毁了。程大雷带着凯恩回到琴川关，先令人与他收拾一下，身上的伤口做简单处理。
<br/>　　程大雷对这个极西之国本身还是很好奇的，据传言说，极西之国擅长奇技淫巧，有许多千奇百怪的小玩意，比如那千里镜就来自于极西之国。
<br/>　　等凯恩收拾好后，徐神机领着他来将军府。程大雷正坐在大厅悠哉悠哉的喝茶，见到徐神机领着凯恩进来，程大雷不由得一愣。
<br/>　　这人当真好俊的相貌，金黄色的头发披在肩上，蔚蓝的眼睛就像海水一般深邃，身体强壮高大，像那古希腊的雕塑一样健美。
<br/>　　“多谢程大雷救了我，我向您致以崇高的敬意，您的恩情我永远不会忘记。”
<br/>　　“好啦，好啦，不要这么客套。”程大雷摆摆手，让凯恩坐下。
<br/>　　“你是怎么到这里来的？”
<br/>　　“我是随商队出发，只是中途遭遇了风暴，和大家走失，最后落到那群野蛮人手中，如果不是程当家救了我，怕我已经被那群野蛮人折磨死。”
<br/>　　说到这里时，凯恩咬牙切齿，看来不仅在帝国眼中，在诺顿王国，戎族也是野蛮人。
<br/>　　程大雷重点还是向凯恩了解诺顿王国的情况，程大雷一直想要了解这世界的完整模样。如今好不容易来了个外邦人，又会说帝国语，程大雷怎么可能放过。
<br/>　　据这凯恩说，从琴川关出发，穿过草原、荒漠、山地就能进入诺顿王国。诺顿王国疆域极大，据这凯恩吹嘘，从南到北有十万公里。
<br/>　　当然，程大雷觉得这凯恩吹牛的成分极大。就像帝国人也喜欢吹牛说有九万里江山，事实上怎么可能嘛。
<br/>　　不过诺顿王国很大是事实，如今他们的皇帝是诺顿九世，似乎风评也不好。
<br/>　　皇帝下面有总督，各地有封王，职位和帝国的城主也差不了多少。
<br/>　　以这个时代信息的滞后，也不可能有一套彻底的中央集权制度，无论是诺顿王国还是帝国都是采取诸侯自治的方式。某种程度上，帝国的制度甚至更先进些。
<br/>　　不过，诺顿王国有一种帝国没有的东西，那就是：教廷。
<br/>　　诺顿王国的神话传说是这样的：世界是天神在睡梦中创造，天神的灵魂化作神明，治理人间。
<br/>　　教廷在王国的权力极大，他们在各地建立神殿，买卖奴隶，侵占土地，甚至王国的王位传承都由教皇决定。
<br/>　　而对于王国的异教徒，教廷更是赶尽杀绝，抓到后就是绞刑，砍头，火刑。
<br/>　　而这凯恩不巧，刚好就是一名异教徒。刚开始他就对程大雷撒谎了，他哪里是过来经商的，他根本就是过来逃难的。
<br/>　　所以，诺顿王国也有鸡贼的人呐。
<br/>　　程大雷看得出来，这凯恩是想要留在这里的。相比之下，帝国要比他的家乡繁华一些，帝国把四外叫做蛮荒也并不是没有道理。
<br/>　　何况，凯恩现在已经是走投无路。
<br/>　　程大雷本也有收留他的意思，至少能开阔一下视野吧，只不过凯恩一开始对程大雷撒谎了，程大雷也不介意端端架子。
<br/>　　“凯恩阁下信仰什么啊？”
<br/>　　“我是一名吟游诗人，信仰自由的风之精灵。”
<br/>　　“喔，那就等于什么都不会了。”
<br/>　　怪不得要逃出来呢，王国不养闲人，琴川关也不养闲人啊。
<br/>　　程大雷顺便阅读以下他的信息。
<br/>　　姓名：凯恩·道尔（来自异国的顶级吟游诗人）
<br/>　　年龄：27
<br/>　　技能：语言精通
<br/>　　隐藏属性：风之信仰者
<br/>　　啧，倒是个拥有隐藏属性的稀有人物，只不过不知道这『风之信仰者』有什么用。
<br/>　　程大雷熬得凯恩实在受不了，终于对程大雷说了实话。
<br/>　　他们这一伙人，大概有五六十位，都是来自王国的异教徒。如今他们的人陷在一处地方，派凯恩出来是探路的。
<br/>　　“还有人么。”
<br/>　　程大雷动了念头，这波人里会不会有自己想要的人才。
<br/>　　
<br/>(本章完)
