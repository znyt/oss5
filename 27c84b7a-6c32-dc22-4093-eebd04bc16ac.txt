　　半个月后的清晨，天墉城西侧后山，盘坐在山崖上，慕清霄嘴中吐出一口浊气，星眸中光泽流转。
<br/>　　“不愧是高级仙武位面，真是浓郁的天地灵气，即使没有机遇，靠凝丹堂的灵药，半年内也能突破练虚六重。”
<br/>　　修为到了练虚期，如果是普通修士，光靠吸收天地间的灵气，哪怕是数十载突破一次也是极为正常的事情。
<br/>　　比如天墉城的普通长老们，他们所修炼的法决，甚至拥有的资源，都限制了他们的成长。
<br/>　　但慕清霄不同，不管是他修炼的心法，或者是血脉，又或者是资源，都不是其他人能比的。
<br/>　　若说天墉城，谁拥有的资源最多，除了掌门涵素真人，执剑长老紫胤真人，凝丹长老还虚真人，就属慕清霄拥有的资源最丰富。
<br/>　　而普通修士，吸收天地间的灵气，数十载才能突破一重，他依靠灵药，配合灵气，半年内就能突破。
<br/>　　当然，修为越高，需要的时间也会随之增长。
<br/>　　站在山崖上，慕清霄望着周围，宛如仙境般的景色，叹道：“剧情已经开始，看来得找个理由，下山一趟。”
<br/>　　现在，他的身上有一半玉衡，是慕清霄当初在琴川，隐秘山谷中的琵琶精身上所获得。
<br/>　　至于另外一半，如果信息中的记载没错，那就是在天墉城以北，碧山上一座名为自闲山庄的地缚灵身上。
<br/>　　任务自然是尽早完成，心里就越踏实，迟则生变，并不是他所希望的结果。
<br/>　　想到此处，慕清霄就化作一道剑光，向执剑堂而去。
<br/>　　执剑堂最外围的斩妖殿，此处是天墉城平日里最热闹的地方，不管是内门，或者外门弟子，都聚集在这里。
<br/>　　斩妖殿在天墉城执剑堂已经有很长一段时间，最起码也有百年之久。
<br/>　　自紫胤真人来到天墉城后，便在执剑堂外设立斩妖殿，以此来警示门下弟子，斩妖除魔，救济苍生，本就是修士的职责所在。
<br/>　　天墉城方圆百里内的城镇，一旦有普通人解决不了的事情，都会以草书的形势，呈现在斩妖殿内。
<br/>　　而弟子们完成任务后，不仅能获得城主所给的奖励，表现优异者，还能天墉城的奖赏。
<br/>　　比如上次，掌门涵素真人让慕清霄炼制的丹药，就是用来奖励给表现优异的弟子。
<br/>　　因此，天墉城门下颇有实力的弟子们都会来此，接受任务，下山斩妖除魔，不管是为了奖励，或者是为了荣誉。
<br/>　　慕清霄来此，自然也是为了挑几个任务，好找理由下山。
<br/>　　他虽是凝丹长老，但想要下山也需要十足的理由，否则频频下山，难免会有人嚼舌根。
<br/>　　“凝丹长老...”
<br/>　　“凝丹长老...”
<br/>　　“凝丹长老早...”
<br/>　　哪怕是一大清早，斩妖殿中都汇聚了不少的弟子，有强有弱。
<br/>　　慕清霄一袭白衣，面如冠玉，仙风道骨，来到斩妖殿后，与周围身穿道袍的弟子，显得格格不入，瞬间就引来所有弟子的注意。
<br/>　　认出他来的弟子，纷纷上前打招呼，慕清霄报以平淡的微笑，很快来到立在左侧的一块信息栏前。
<br/>　　信息栏上，挂着的一般都是最新的任务，抬头望去：
<br/>　　“临安城内，惊现盗贼，权贵人士与城主府都频频失窃，哪怕增加护卫，也追查不到蛛丝马迹，望仙人们伸出援手。”
<br/>　　“临安城附属城市，九旬老太裸死街头，死相极为凄惨，怀疑妖魔作祟......”
<br/>　　“稻草村内，母猪夜半频频惨叫......”
<br/>　　望着信息栏上的一个个任务，慕清霄眼角忍不住抽了抽，忽然剑眉一皱，落在最下方的一条信息上。
<br/>　　“碧山翠萍村内，最近频频有年轻男子死去，死者身上都出现一个现状，事发当天都浑浑噩噩......”
<br/>　　看到碧山两字，慕清霄嘴角牵起一丝笑意，碧山翠萍村，距离自闲山庄并不远。
<br/>　　在几十年前，自闲山庄是个占地面积很大的山庄，而山庄外就是翠萍山。
<br/>　　可在山庄内的人被屠杀后，翠萍村里的人觉得晦气，于是就搬迁到了山脚下，而自闲山庄则渐渐成为乱葬岗。
<br/>　　想到此处，慕清霄直接将草书揭下，向斩妖殿内部走去。
<br/>　　来到斩妖殿内，只见一名身穿道袍的青少年坐在其中，正帮来此弟子登记着任务信息。
<br/>　　看到青少年的瞬间，慕清霄微微一愣，后者可不就是与他一起进入天墉城，紫胤真人的首徒陵越么。
<br/>　　当慕清霄来此，正排队登记的弟子们，纷纷给他让开一条路，表现的非常尊敬。
<br/>　　慕清霄也不客气，来到队伍的最前面，道：“陵越，几年不见，都已经筑基期了，不错。”
<br/>　　抬起头来，看到慕清霄，陵越面露微笑，连忙站起身来，就要行礼。
<br/>　　慕清霄摆了摆手，将记载信息的草书递了过去，道：“无需多礼，我来此是准备下山一趟，你替我登记一下。”
<br/>　　“嗯，好。”
<br/>　　接过草书，陵越微微点头，开始帮他登记。
<br/>　　这时，慕清霄面露好奇的问道：“你平日都在此帮弟子登记信息？”
<br/>　　“这里的事情一般都是执剑堂的侍女处理，今天待女出了点事，我便来此顶替一天。”
<br/>　　“原来是这样，对了，半个月前，紫胤真人带回来的孩子，现在怎么样了？”
<br/>　　闻言，陵越微微一笑，道：“那孩子貌似记不起以前的事情了，正在堂中修养，至于往后会如何，需要等师尊出关后再做决定。”
<br/>　　根据陵越口中的信息，看来紫胤真人还未出关，韩云溪不记得以前的事情，想必是紫胤真人提前将他的记忆封印。
<br/>　　乌蒙灵谷被灭族，对于一个十岁的孩子来说，未免太过沉重。
<br/>　　“凝丹长老，认识那孩子？”
<br/>　　“半个月前，紫胤真人将他带回来的时候，我正好在场。”
<br/>　　几分钟后，陵越停笔，道：“长老，已经帮您登记好了。”
<br/>　　“嗯，这枚‘地灵丹’给你，想必你很快就能用上。”
<br/>　　微微点头，慕清霄从储存空间中取出一颗土黄色的丹药递了过去。
<br/>　　听闻地灵丹，周围弟子纷纷忘了过来，眼中满是羡慕嫉妒的光芒。
