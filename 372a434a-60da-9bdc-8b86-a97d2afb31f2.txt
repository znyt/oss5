        “下个月，我的战队要打半决赛，你要不跟我一起去北城？”明司寒一边放慢步伐，一边问道。
<br/>
<br/>        连星儿连连点头，微笑道：“好呀！那你给我安排离赛场最近的VIP专座，我要看我师父打比赛！”
<br/>
<br/>        明司寒微微偏头睐着她，看着她笑靥如花，心里只觉很甜，嘴角也不禁挂起了一抹浅笑。
<br/>
<br/>        他想好了，等他拿了比赛冠军，他要亲手拿着奖杯送给她当师徒第一次正式见面的见面礼。
<br/>
<br/>        “好，一定。”他郑重地允诺。
<br/>
<br/>        话题到这儿，连星儿忽然眨巴着眼睛，一脸好奇地看着明司寒问道：“你到底有几个家？”
<br/>
<br/>        “你指的是房子吗？”明司寒反问。
<br/>
<br/>        连星儿点了点头。
<br/>
<br/>        他在临海城里也有单独住的地方，这会儿他又要跟她一起回滨江城。
<br/>
<br/>        她记得，他在滨江城好像也有一套别墅来着。
<br/>
<br/>        “不知道诶！七七八八的加起来，应该有十来套吧！反正都在不同的城市里。”明司寒不痛不痒地说。
<br/>
<br/>        连星儿一脸不可思议地接着问道：“你自己买的啊？”
<br/>
<br/>        “嗯，有问题吗？”明司寒反问。
<br/>
<br/>        连星儿立马摇头，很是崇拜地看着明司寒：“你真的好厉害哦！我到现在还得靠我爹地养着！不过，等我毕业了，我也可以自己赚钱了。”
<br/>
<br/>        “你毕业了，我也可以养你啊！”明司寒下意识地接话。
<br/>
<br/>        连星儿笑道：“我很贵的啊！你养不起的！”
<br/>
<br/>        “贵才好啊！这样，我也就有赚钱的动力了。”明司寒会心一笑。
<br/>
<br/>        连星儿怔了一下，忽然意识这话题扯远了，立即闭上了嘴。
<br/>
<br/>        明司寒也跟着愣了一下，话锋一转，接着说道：“我可没有要娶你的意思啊！你别自作多情了！”
<br/>
<br/>        “……”连星儿汗颜，硬生生地回怼了一句，“我也没有要嫁给你的意思，你也别自作多情了。”
<br/>
<br/>        “……”这回轮到明司寒无言以对。
<br/>
<br/>        两人开始闹起了别扭，一路从临海城别扭到滨江城。
<br/>
<br/>        下了大巴车后，明司寒打算去取自己的私家车先送连星儿回家，谁知连星儿自己拦了一辆出租车，说还有其他事要去做便先走了。
<br/>
<br/>        明司寒没拦着，只是淡定地送她上了出租车，然后跟她挥手道别：“平安到家后，给我回个信。”
<br/>
<br/>        “好！”连星儿坐在副驾驶座上点了点头，摇上车窗前，她向他挥了挥手，“拜拜。”
<br/>
<br/>        “嗯，拜拜。”明司寒微微一笑。
<br/>
<br/>        反正到了滨江城就是傅夜沉的地盘，他无需为她连星儿担心。
<br/>
<br/>        连星儿上了出租车后，让司机往郊区开去，一直到了一条路的尽头，前方不再是柏油路后，她才付了车费下了车。
<br/>
<br/>        出租车一走，连星儿便往旁边的一条小路上走去。
<br/>
<br/>        穿过一片树林后，她来到了一张高合金智能门前，按响了门铃。
<br/>
<br/>        连星儿等了一会儿，门内没动静，她又按了一次。
<br/>
<br/>        还是没动静，当她准备按第三次的时候，门却自动开了。
<br/>
<br/>        而这个时候，门内的那条平日里被日光灯照得亮如白昼玄关隧道里却是一片漆黑，恍若深不见底。
<br/>
<br/>        连星儿站在门口打算从手提包里掏手机时，忽然听到隧道深处传来一阵疾步的脚步声以及粗喘声。
<br/>
<br/>        下一秒，漆黑的隧道里突然出现两个金黄色上下快速跳动的圆点。
<br/>
<br/>        连星儿怔愣了一下，还未缓过神来时，那两个快速跳动的金色圆点忽然间变成了一双兽眸，紧接着是一对尖锐的前爪从黑暗中冲了出来。
<br/>
<br/>        一只浑身灰白黑色兽毛，足以有一个成年人那么高大的狼型巨兽腾空跃起。
<br/>
<br/>        连星儿顿时瞠目结舌，当她看到一张长着獠牙的血盆大口时，吓得尖叫了起来：“啊——”
<br/>
<br/>        下一个瞬间，她闭上了眼睛，只觉肩头传来被撕裂的刺痛感，让她痛晕了过去。
<br/>
<br/>        连星儿浑浑噩噩地感觉自己一直在做梦，梦到自己小时候被一只流浪狗追的情形，不，那不是一只流浪狗，而更像是一只狼。
<br/>
<br/>        “救命！救命！救命！”
<br/>
<br/>        连中凯听到连星儿的呼救声后，立即冲到床边按住了她的肩头，安抚道：“乖孙儿，我的乖孙儿，是外公！外公在这儿啊！”
<br/>
<br/>        连星儿惊魂未定地在床上翻来覆去后，猛然睁开双眼坐起身来，额头上已是大汗淋漓。
<br/>
<br/>        连中凯见状，连忙拿起放置在一旁床头柜上的干净毛巾，替连星儿擦拭着汗水。
<br/>
<br/>        “外公、外公，我看到一只狼了！一只有人那么大的狼！”连星儿连忙拉住连中凯的手腕，一边摇晃着，一边情绪激动地说道。
