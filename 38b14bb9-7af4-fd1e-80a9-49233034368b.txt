夏夕绾看着李玉兰，“李玉兰，你们把爷爷藏到哪里去了，你们是不是心虚了，怕爷爷醒了将十多年前的真相说出来？”
<br/>
<br/>        “真相？夏夕绾，我听不懂你在说些什么，十多年前明明就是你将老爷子推下楼的，我们不让你见老爷子也是为了老爷子的安全着想，况且这件事是你爸一手操办的，我劝你不要将主意打到老爷子的身上，赶紧走吧！”这一次李玉兰的口风很紧，一口咬定了是夏夕绾将老爷子推下楼的，还下了逐客令。

<br/>        夏夕绾也没有再多做纠缠，她转身就下楼了。

<br/>
<br/>        夏夕绾从女佣身边过的，她停下了脚步，看向那碗药。
<br/>
<br/>        李玉兰面色一变，她骂骂咧咧的就下了楼，还打了那个女佣一巴掌，“小贱人，谁让你把药端出来的，赶紧端回去！”
<br/>
<br/>        女佣都要哭了，“是，夫人。”
<br/>
<br/>        夏夕绾再次看向李玉兰，这一次是带了探究的意味，“李玉兰，你生病了？这样吧，我是医生，给你免费把个脉吧！”
<br/>
<br/>        李玉兰吓了一跳，整个人都往后退，不让夏夕绾把脉，“夏夕绾，你快点走吧，你再不走的话我就让人将你轰出去了！”
<br/>
<br/>        “李玉兰，那你保重身体，我先走了。”夏夕绾离开。

<br/>        夏夕绾出了别墅，她停下脚步回头，刚才她嗅到了那碗中药的味道，如果没有猜错的话，那是一碗安胎药！
<br/>
<br/>        李玉兰怀孕了！

<br/>
<br/>        没想到李玉兰都这么大的年纪了，这生儿子的心还没死，现在她都快五十岁了，竟然顺利怀上了三胎。
<br/>
<br/>        看得出来李玉兰十分在意这一胎，大概怕她搞破坏，都不愿意让她知道，怪不得这段时间夏家这么安静，看来他们都是想安胎生儿子来翻身了。
<br/>
<br/>        这时“叮”一声，夏夕绾的手机响了，来微信了。

<br/>        是双双发来的。

<br/>        夏夕绾点开了照片，真的是夏振国，不过夏振国不是一个人，他身边还有一道青春倩丽的身影，看的十分的眼熟，夏夕绾想起来了，竟然是夏小蝶以前那个闺蜜孔真儿！

<br/>        夏振国是带着孔真儿去逛商场的，夏振国手里大包小包的，看来刷了不少卡，孔真儿亲昵的挽着他，还亲了他一下。

<br/>        双双好家伙啊，绾绾，你老爸竟然在外面包养了一个女大学生。

<br/>        夏夕绾这段时间有点忙，没空管到夏振国，没想到夏振国的日子过得这么滋润舒坦，情人都养起来了。

<br/>        这家里的红旗在怀孕养胎，外面的彩旗飘飘，他也不怕自己翻车了。
