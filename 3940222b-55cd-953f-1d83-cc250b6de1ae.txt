　　第540章
<br/>　　程大雷撇撇嘴，接下来的的动作令刘发财难以置信。只见程大雷将剑扎在地上，摊开双手道：“我的人头就在这里，看你有没有本事来取。”
<br/>　　刘发财也不是酒囊饭桶，他知道武器对武者的重要性。想要赤手空拳空手夺白刃谈何容易。
<br/>　　程大雷竟敢将剑抛开，难道他的拳脚功夫也一样了得。
<br/>　　两个情人都被程大雷杀死了，苏离对程大雷的恨意可想而知。一见程大雷手中没了兵器，她怎么会放过这机会，身子腾起，决绝一剑向程大雷扎过来。
<br/>　　在苏离、剑恨南、李越英三人中，苏离的本事是最强的。能娶两个男人的女人怎么会简单。她的本事不仅三才剑阵，还有这一招离火一剑。
<br/>　　其势如火，侵略万物，将全身劲力凝于一点，如一头愤怒的火狮子，有摧枯拉朽之势。
<br/>　　那一刹那，周围的气机似乎都被苏离掠去，即便这一剑不是攻向刘发财，刘发财都觉得有些窒息。
<br/>　　程大雷抬起了右手。
<br/>　　一瞬间，气机溃散，愤怒的火狮子被摁住脑袋，周围令人窒息的气氛又恢复如常。
<br/>　　只见那柄离火剑，被程大雷夹在两指之间，一切云淡风轻，宛若什么都没有发生。
<br/>　　刘发财睁大了眼睛，苏离也难以置信，目瞪口呆的看着程大雷。
<br/>　　程大雷的拳脚功夫其实并不如何，无非是有些力气而已。但他除了阿飞快剑外，同样还有灵犀一指。
<br/>　　天下武功，唯快不破；灵犀一指，能接天下快剑。
<br/>　　也不知道程大雷的灵犀一指，能不能接住他的快剑。
<br/>　　眼前刘发财愣住了，苏离也愣住了。但程大雷可没有发愣，他几乎没有任何停顿，上前一步捏住苏离的手腕。然后重重一拳，砸向苏离的侧腰。
<br/>　　即便苏离本事了得，但也并非铜皮铁骨，一拳就砸得她肚子里翻江倒海，五脏六腑像裂开一般。程大雷的攻击没有停止，拳打脚踢，一身力气泄在苏离身上。
<br/>　　苏离花容失色，嘴角裂开，已完全失去了身体的控制权。但就算在这个时候，她也用愤恨的眼神盯着程大雷。
<br/>　　程大雷与她的目光对视，这才发现苏离今年三十有余，但却也有些姿色。如今杜鹃挂血，更是惹人生怜。
<br/>　　“快些走吧，走得急了，还赶得上。”
<br/>　　咔嚓一声，程大雷掐断了她的喉咙，苏离的身体软绵绵垂在地上。
<br/>　　程大雷轻出一口气，将匹夫剑捡起来收入鞘中，拍了拍身上的土。
<br/>　　刘发财颤颤巍巍站起来，不敢与程大雷的眼神接触，他愣了半晌，忽然笑道：“哈，程当家果然好本事，名不虚……”
<br/>　　程大雷回过头，目光冷冷的盯着他。刘发财瞬间闭嘴，不敢说话。
<br/>　　“再有下一次，我便不会留你了。”
<br/>　　“是是是。”刘发财忙不迭的点头，后背已被冷汗浸湿。
<br/>　　二人牵上座骑，继续上路。一路上，刘发财都不断提醒自己，这是个人渣，这是个魔头，自己千万不要被他的外表欺骗了。
<br/>　　“程当家是不是要寻一个大夫？”刘发财大概猜出程大雷的意思，故意显眼的骑着黑牛，惹得江湖高手来杀他，就是为了打探消息。
<br/>　　这样的主意，自己是该佩服他艺高人胆大呢，还是该说他脑袋有些秀逗。
<br/>　　“嗯呐，你有这方面的情报？”程大雷问。
<br/>　　“没有。”刘发财顿了顿：“不过，有一个地方，该有程当家要的情报？”
<br/>　　“哪里？”
<br/>　　“琅琊万卷书院。”刘发财吐出一个地名。
<br/>　　程大雷微微皱眉，在脑海中搜索着关于万卷书院的情报。
<br/>　　程大雷也派出人手搜集江湖上的情报，但凉州和东海一东一西，相距太远，刘发财忽然提起这名字，程大雷还需好好想想。
<br/>　　万卷书院座落在徐州琅琊城，靠山观海，主人名叫孟夫子。孟夫子不慕权贵，不追名利，一心搜集天下书籍。据他所说，功名利禄，皇权富贵最终的结局都是被黄土所埋，只有先人留在书中的智慧，方才能流传千年。
<br/>　　据说万卷书院的藏书不下万卷，从上古先秦时期到如今李家掌国，经史子集，医卜星相藏书无所不包。当然，也不是什么书都有资格进入万卷书院。帝国多少读书人，呕心泣血写本书，毕生愿望就是摆在万卷书院的书架上，以此为荣。
<br/>　　所以这孟夫子就算不是天下读书人的领袖，但也至少是精神领袖。琅琊万卷山俨然成为士子心中的圣地，孟夫子更是桃李满天下，帝国朝廷多少达官贵人，在孟夫子面前以弟子自称。
<br/>　　人脉广了，消息就不可能不广。况且自己要寻的华氏一族就在东海，如果想要打探他们的情报，怕就要去问问这地头蛇。
<br/>　　“好，去东海。”程大雷道。
<br/>　　刘发财再没有逃跑的心思，确实是不敢再有，乖乖的和程大雷一起，去向徐州琅琊山。
<br/>　　几乎程大雷刚走不久，有两道人影出现在刚才发生战斗的地方。
<br/>　　李婉儿看着地上的三具尸体，心中默默祷告，最后长长叹了一句。
<br/>　　“这是谁，下手如此凶狠，帝国连年征战，路旁遗骨无人埋，我们将他们尸骨埋葬了吧。”
<br/>　　不用她吩咐，李莫愁已开始动手。现在李莫愁的打扮十分诡异，身后交叉背着两柄铁锹，看上去有些滑稽。倒是有了些蛤蟆寨的气质，又蠢又强，蠢强蠢强的。
<br/>　　这也是没有办法的事，如果李莫愁不帮忙，李婉儿就会自己动手。还不知要到什么时候，李莫愁动手的话还能结束的快些，好继续赶路。
<br/>　　关键是，这李婉儿其实没有多少力气。本是十指不沾阳春水的千金公主，什么时候干过这种埋人的事情。她挖了两铁锹就没了力气，大半工作是李莫愁完成的。
<br/>　　李婉儿擦擦额头的汗珠，看着李莫愁的模样露出一抹笑容。
<br/>　　“能做一点事，便做一点事，谁说没有用处，看你不就是被我影响了么。一个人改变十个人，十个人改变一百个人，然后就是千千万万人。”
<br/>　　李莫愁一铁锹拍死她的心情都有。
<br/>(本章完)
