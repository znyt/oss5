　　第641章
<br/>　　那混江龙怪叫一声，提着鱼叉便扑了上来，口中怪叫一声：
<br/>　　“贼人，去死吧。”
<br/>　　铛！
<br/>　　两人的兵器在空中重重相撞，单雄信暗吃一惊，心道：此子竟然如此了得。
<br/>　　混江龙心中更是震惊，他擅长水下功夫，步下的功夫其实比不得单雄信。如此双方刚一交手，他心中便怯了。
<br/>　　传说那程大雷手下强将如云，也不知面前是哪一位，那程大雷今天到了没有？周围乱糟糟的，也分辨不出敌人有多少。可混江龙知道，再打下去自己必定吃亏。
<br/>　　说时迟那时快，眨眼间双方已经交手数合，混江龙勉强撑着不败。忽然，他后跃跳出战团，指着远处道：“我家大当家去那里逃了！”
<br/>　　单雄信向他手指的方向看了一眼，黑洞洞却也看不清楚。回过头来，眼前已不见了混江龙的身影。
<br/>　　单雄信却也不是傻瓜，只是这些机灵鬼道的东西他并不擅长。此刻他已意识到自己上当，哇呀呀一声大叫，提着兵器寻找混江龙的踪迹。可周围一片混乱，又哪里寻得到。
<br/>　　有怎样的首领便必定有怎样的手下，混江龙知难而退，他的手下其实也好不到哪里去。战斗进行没有多久，便已接近尾声。倒不是说蛤蟆城的兵有多强的战斗力，这自然是一方面，另外一方面则是因为程大雷、银眸、单雄信三人。
<br/>　　程大雷自不必说，单雄信也是单打独斗的好手，而银眸力大无穷，是可以拎着敌人身体当武器的存在。黑风寨控制不住这三人，便注定了他们的失败，更何况，他们的指挥官混江龙更是临阵脱逃，根本没人能将他们组织起来。
<br/>　　于是，一面倒的形势便出现了，程大雷一方几乎在全员无损的情况下结束了战斗。最后歼敌数百人，俘虏敌人七八百众，剩下的人便从各个方向，做鸟兽散了。
<br/>　　在黎明时分，程大雷令杨国忠清点这次战斗的战利品。兵器若干，粮草不计其数，另外还有几十艘大小船只……
<br/>　　黑风寨在此地盘桓多时，看似强大，但却一攻便破，战斗的顺利程度委实也是他没有想到的。
<br/>　　“大当家，这些俘虏怎么处理？”单雄信做出凶狠表情：“干脆杀了，不留后患。”
<br/>　　程大雷白他一眼，其实在如今忠诚度是个很稀缺的品质，一般谁给碗饱饭吃，他们就会跟着谁干。混江龙这批手下，未必没有他从别处劫来的，现在他们跟了程大雷，他们也并不会有什么意见，也许还会庆幸抱了一棵大树。
<br/>　　“俘虏粮食武器能带上的都带走，剩下的山寨，一把火烧掉干净。”程大雷干脆直接的下达命令。
<br/>　　“是。”阿喜痛快答应一声，又道：“大当家，目前属下还有一件小事，不知该如何处理？”
<br/>　　“什么事呀？”
<br/>　　程大雷奇怪的看了他一眼，发现阿喜目光闪烁，欲言又止，似乎有些话不方便当着众人讲。他点点头：“走，带我去看看。”
<br/>　　阿喜带着程大雷到一个屋子前，打开门恭恭敬敬冲程大雷摆摆手：“大当家请。”
<br/>　　程大雷踱步而入，又腾地一下跳出来：“什么情况？”
<br/>　　屋里有七八个女人，都是衣衫不整，披头散发。
<br/>　　阿喜咳嗽一声，道：“属下打听过，那混江龙好色如命，这些人当都是他掳来的女子，她们都是从床上被拉下来的，属下把她们关在这里，所以她们现在都是这个样子。大当家，这些人该如何处理？”
<br/>　　“这……”
<br/>　　程大雷委实有些为难了，他走到今天，也算是大战小战无数，但委实没有收获过女人，并没有处理这方面事情的经验。
<br/>　　“一般……都是怎么处理的？”程大雷好奇问。
<br/>　　“大当家可以讲她们据为己有，传出去也不算什么。”阿喜有句话没说出口：反正程大雷现在的名声也不差这些了。
<br/>　　程大雷心中一荡，委实有些动心，不过他也算是见过世面的人物，这些庸脂俗粉还不会被他看在眼里。
<br/>　　“有没有其他处理的法子？”
<br/>　　“也可以将她们赏给手下弟兄，或者是将他们遣送回家。”阿喜顿了顿：“不过这些人，恐怕都无家可归，即便回到父母身边，对他们来说也未必是幸事。”
<br/>　　程大雷点点头，明白阿喜话中的道理。这个时代，女人的身份是很低的，和牛羊骡马这样的畜生也没太大区别，都可以被看做男人的私产。许多诸侯王妻妾无数，身边美女如云，奖励手下女人的事情更是经常发生。
<br/>　　这样一个世道，男人活下去都艰难，更何况女人。程大雷沉思片刻，道：“这件事你来办好了，就赏给这次的有功之人。”
<br/>　　乱世如潮，谁不是这洪流中几根浮尘微沫。就连程大雷也是身不由己，他更不可能改变几个女人的命运。随波逐流，却也是没有办法的办法。
<br/>　　一场大火从黑风寨燃起，借着晨风之威，火势越来越大。这个时候，程大雷已经赶着船队，满载货物踏上了返程的路。
<br/>　　大门被推开，混江龙从山寨中跑出来，一头扎进水中，大口大口的饮水。
<br/>　　回过头，整座山寨已成了火海，辛苦多年打下的基业已经成了一片灰烬。
<br/>　　昨夜他见机得快，逃入卧房的暗道中，如此才逃脱一条性命在。但那暗道并不是向外的，只不过是卧室下一座地窖。听得外面动静小了，他才从地窖中逃出来，可山寨已经化作火海。
<br/>　　“大当家，大当家……”
<br/>　　草丛中涌出一个个人，他们都是昨夜逃走的，其实也没有逃走多远，都藏在山寨外围，直到此刻才敢现身。
<br/>　　黑风寨三千弟兄，此刻面前剩下的也不过就五六百人罢了。其他人死的死逃的逃，要么就成了程大雷的俘虏。
<br/>　　“大当家，咱们接下来该去哪里？”绿头蝇过来问。
<br/>　　“程大雷，此仇不报我誓不为人！”
<br/>　　混江龙咬牙切齿，向天咆哮，第一个念头就是找程大雷报仇。但其实也报不了仇，他想了想，无可奈何的挥挥手：
<br/>　　“走！”
<br/>(本章完)
