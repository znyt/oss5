　　陈少爷结束完最后一轮引诱剂的引爆后，便从后方跑到了前线找到楚涵：“老大，怎么了？”
<br/>　　“绝命狙击枪带着的吧？”楚涵没头没脑的问道。
<br/>　　“带着呢，但是现在已经是第三代了，之前和上又帮我改良了一下。”陈少爷回道。
<br/>　　楚涵点头，解决周围丧尸的动作却不停，竟然没让身边的陈少爷收到一丝波及：“很好，去指挥台空地待命。”
<br/>　　“是。”陈少爷下意识照做的动作一顿，摸了摸脑袋不解问道：“额，待命内容？”
<br/>　　“一会儿你就知道了，该打的就打。”楚涵不再多说，开始专注于杀丧尸，一会儿的功夫那些七阶甚至八阶的高阶丧尸，就死伤一片，而后被他一只只踹下了高墙。
<br/>　　陈少爷回到了高台空地上，虽然不明白到底怎么回事，但还是习惯性的对自己的装备开始例行检查。
<br/>　　何峰瞥了眼前方高墙的场景，顺便也看了下陆续撤回的空军队伍，紧接着声音极轻的自言自语道：“巧了，正好在新武器能量耗尽的时候……还真是祸不单行。”
<br/>　　高少辉也发现了狼牙战团的变动，一直没有任何交流的他不得不将天照3000降落在高墙，还顺带压死了好几头高阶丧尸。
<br/>　　“楚涵，飞行丧尸都杀光了。”高少辉有些激动，跑上来就兴奋开口。
<br/>　　楚涵没有任何意外的发问：“一共杀了多少？”
<br/>　　高少辉想了想回道：“几万吧，现在空间都撤了回来，飞行丧尸还会出现吗？”
<br/>　　楚涵摇了摇头：“不确定，但从比例来说出现的飞行丧尸数量已经过了头，应当不会再有，空军的战绩很强。”
<br/>　　高少辉裂了裂嘴：“我一个人就杀了好几千呢！”
<br/>　　楚涵笑看了他一眼，道：“你还有别的任务，回段氏基地运物资。”
<br/>　　“啊？”高少辉愣住，紧接着不满的喊道：“你搞毛，天照3000的空间这么点，怎么运？”
<br/>　　“你不会让段氏基地的人派直升机运过来？”楚涵没有多说，只是直视着高少辉：“还有一份特殊物资，你去找和上拿。”
<br/>　　“好吧好吧，看在本少爷心情好的份上。”高少辉没有太多意见，只是临走前忽然又在楚涵耳边来了一句：“帮我在何峰面前多说说好话！”
<br/>　　天照3000眨眼间消失在天际，楚涵这里的心情越是越发沉重，身旁不断战斗的三战队成员也大致如此。
<br/>　　因为他们所有人都知道，防守队控制的那批新武器能量即将耗光，很快这个基地就将失去最有力的第一道防线，还有大几百万的丧尸在基地外徘徊，它们将会在新武器失去作用的第一时间，一次性的扑进这个基地！
<br/>　　之前的两天不过是这场大尸潮战役的前菜，塞牙缝都不够！
<br/>　　真正你死活我的绝地反击战，还没开始！
<br/>　　——————
<br/>　　南署基地与段氏基地的距离本就不远，更是同属于一条线上的华夏边境地带，天照3000飞过去耗时仅仅几分钟，近乎就是高少辉刚和楚涵说完话，哼了首曲后就抵达了目的地。
<br/>　　鲁初雪等一干段氏高层此时还并不知晓南署基地爆发的情况，毕竟距离最后一批狼牙战团的队伍离开还没几天，而眼下正处于他们所知的大尸潮出现的时间，所以并不知道狼牙安排以及没有收到任何消息的鲁初雪，不禁有些担忧，决定派出直升机前往南署基地一探。
<br/>　　只是当她刚下达这一命令，段氏基地的直升机还没来得及起飞的时候，天照3000出现的消息就一下子在整个基地炸开。
<br/>　　此时段氏基地的人还没听说过这种飞行物，而鲁初雪也是第一次见到这种玩意儿，当下完全没能做出任何反应，只能呆傻的看着天照3000大摇大摆的飞入基地，绕了一圈后稳稳的停在了中心处的一片空地上。
<br/>　　段氏基地的高层们连忙跑出去，鲁初雪本人也赶紧冲出屋子，在这本就紧张的几天里，异变怎么不断发生？
<br/>　　“什么东西？这是飞机还是啥？”
<br/>　　“不会是外星……”
<br/>　　很多段氏高层都吓的只能站得远远的不敢靠近，当下在已经习惯了末世中丧尸和异种的他们，很自然的想到了这世上还会出现的其他可能。
<br/>　　“让让！”这时候的鲁初雪冲出人群，快步朝着那天照3000而去。
<br/>　　“中将？！”有人大惊。
<br/>　　而就在这时候，天照3000的机舱门打开，高少辉一脚就踏了出来。
<br/>　　见到高少辉一瞬间的鲁初雪脚步一停，有些愕然：“怎么不是楚涵上将？”
<br/>　　其余段氏高层在看到天照3000内走出的是个人之后，也终于放松了警惕，随着鲁初雪的步伐逐渐靠近。
<br/>　　高少辉有些惊讶的看着鲁初雪：“你认识天照3000？楚涵开过它？”
<br/>　　鲁初雪一愣后回答：“没有，我只是听丁思尧说过。”
<br/>　　“哦！”高少辉并不知道丁思尧是哪个家伙，于是开始进入正题道：“你是哪位？”
<br/>　　刚跑过来的段氏基地高层们一靠近就听到了这句话，顿时看向高少辉的眼神就颇为不友好，这是哪来的家伙竟然对中将说话如此不客气。
<br/>　　但下一刻让他们出乎意料的是，鲁初雪颇为规范的朝着高少辉敬礼，道：“属下为段氏基地鲁初雪中将。”
<br/>　　高少辉身为神秘家族的嫡系少爷，觉得这一切都理所应当，于是点头道：“楚涵让我过来传句话，让你们送批物资过去。”
<br/>　　对大尸潮目前情况一无所知的鲁初雪下意识问道：“南署基地吗？”
<br/>　　“废话。”高少辉觉得眼前这女人蠢爆了，语气也不耐烦起来：“大尸潮战役都打了两天了，不送到南署基地难道还送到狼牙去啊？”
<br/>　　此话一出，四周一片哗然！
<br/>　　“两天？已经开始了！？”
<br/>　　“尸潮战已经开始两天了？！”
<br/>　　“我们这里怎么一点风声都没有？”
<br/>　　鲁初雪也没想到大尸潮竟然体现到来，瞬间紧张异常：“当下战况如何？我需要根据战况安排物资类别分配。”
<br/>　　高少辉哪懂这些专业知识，但想了想还是尽力回答道：“杀了七八百万丧尸了吧？还有一半呢！狼牙这里……恩，基地内一线战区和迷宫战区的情况我不知道，只知道空军死了三分之一。”
