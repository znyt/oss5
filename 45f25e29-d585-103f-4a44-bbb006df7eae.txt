　　洪荒天地初开，一元复始，万象更新。万物起始，生灵迭出，灵性智慧，因果轮回，天道纲常。
<br/>　　天地初开，荒古世纪，大道归隐，天道孕育。天地神机，道气横生，生育众神。众神含盘古开天福运，演化灵智，生而为神，拥有大智慧，大神通，大法力。
<br/>　　开天是功，但同时也是劫。功德福运泽蕴苍生，但劫煞之气却会破灭苍生。功德在开天之初分散各处众神，但盘古看天时所杀的混沌魔神残余的晦气、怨气、怒气、恨意等各种负面所形成的劫煞之气却孕育出各种凶兽。
<br/>　　凶兽天生与众神不和，只知道破坏洪荒天地。天地之初，孕育出五大凶兽王与众多凶兽，凶兽王中最为强大的是神逆，其余四者分别为吞天噬地之饕鬄、善恶颠倒之穷奇、是非不分之混沌、顽固不化之梼杌。
<br/>　　五大凶兽以破灭洪荒天地为乐，在这新生的天地之间掀起无穷杀机。同时，凶兽之间也相互杀伐，完全不信任任何人。
<br/>　　后来，兽皇神逆以最强姿态横扫其他四大凶兽，建立了空前强大的凶兽王朝，带领众多凶兽在洪荒世界大兴杀伐。
<br/>　　于此，应运而生的还有五位大神，他们钟天地气运而生，为了守护洪荒天地，是五大凶兽的敌人。这五位大神分别是金祖白帝、木祖青帝、水祖黑帝、火祖赤帝、土祖黄帝。同时应运而生的，还有其他众神，他们隐隐以五帝为首。
<br/>　　为了守护洪荒，众神与众凶兽开启了持续一千多元会的荒古大战，直至一方覆灭为止。
<br/>　　……
<br/>　　洪荒不周神山顶天立地，支撑起整个洪荒天地，是洪荒第一神山，乃盘古脊骨所化。
<br/>　　在不周神山之东不知多远的距离，也有一座神山，山脉巍峨壮阔，气势磅礴，连绵而恢弘，宛若天地脊背，横亘在那里，一条有一条交织在一起，又像是蛰伏在地上的蛮龙，在沉睡。
<br/>　　它壮阔而雄浑，无与伦比，为洪荒第二神山昆仑神山。
<br/>　　一股苍茫与雄浑的气息铺天盖地而来，山中勃勃生机，到处散发着烂烂光泽。烟霞散彩，日月耀光。时而仙鹤唳，时而青鸾翔。白鹿老猿时而隐现，青狮白象偶有隐藏。乃天堂福地也。
<br/>　　昆仑神山为众神之乡，分东西两座，灵秀多集中在东昆仑，如此机缘只因此地孕育盘古元神三清。
<br/>　　东昆仑神山深处，一座钟灵秀气，灵气浓郁的山谷之中，谷内草木郁郁葱葱，仙花散漫，霞光四射，灵气纵横。按理说，这里灵气如此浓郁，应该有很多强大妖兽。但，事实完全相反，此地非但没有强大妖兽，反而一个动物都没有。
<br/>　　只因为此地被一座大阵覆盖，大阵保护着里面的东西，谁来谁死，当然没有妖兽了。其大阵内，灵气更加浓郁，在大阵的中间要三道青色清气，正是盘古元神清气，也就是所谓的三清。
<br/>　　三道元神以三才之势排布，于此地的山川之势组成一座天然的三才大阵，既有保护了道元神之用，又有汇聚天地各种灵气之效。
<br/>　　三道太清元神，第一道元神之上，透露着一股冷淡无为的气息，其上一座三十三层金色小塔缓缓旋转，汇聚更多灵气注入元神之中。
<br/>　　第二道玉清元神之上，则透露着一股高贵的气息，仔细看就可以知道，元神最深处有一朵白色火焰在熊熊燃烧着，将外面的元神清气点点炼化着。
<br/>　　第三道上清元神之上，透露着一股桀骜不驯，奋发向上的凌厉气息。
<br/>　　三道元神在缓缓吸收着山谷之中的天地灵气，滋养元神，以期待早日意志苏醒。其中，玉清元神吸收速度最快，太清第二，上清第三。
<br/>　　……
<br/>　　“嗯？这是……？盘古元神？玉清气？元始天尊！”元释的意识慢慢苏醒过来，发现自己所在的地方正是盘古所化的元神上，而且还还是玉清元神，后世的大名鼎鼎的元始天尊。
<br/>　　“果然，我还是逃过了大道的注意，度过了这混沌劫，而且利用因果线牵引到盘古元神之中，转生到这洪荒天地之中。再不用担心户口问题了。”
<br/>　　听了这话，像是元释早有预料。不错，元释是早有预料，他从开天时就打得是这个注意。再知道盘古要开天时，元释就知道自己摆在自己面前的路有两条，自己必须要做出选择。
<br/>　　两条路，一条是继续做混沌魔神，参悟大道因果法则，直至证道不朽，另一条是放弃混沌魔神之身，转生至洪荒新天地之中继续修行。经过深思熟虑，元释选的是第二条路。
<br/>　　继续做混沌魔神，做因果魔神是很强，但元释看不到一点自己的前途。因果大道法则自己已经参悟到了混元十二重，之后想要证道不朽，就需要掌握一整条因果大道法则本源。若是要如此做，就是要与整个大道作对，混沌之中只有自己一人，元释真看不到一点前途。
<br/>　　至于第二条路，转生到洪荒世界，这就意味着自己要放弃之前的种种，重新开始修炼。但同时，元释也就可以参悟他为自己准备的第二条更有潜力的道——元始本源之道。这一条大道，元释没有参考，需要自己慢慢摸索，但他有一个别人都没有的造化——鸿蒙元始印！
<br/>　　元释选择了第二条路，但想要进入洪荒，他就需要有一个新的身份。因为只有洪荒生灵才能在洪荒之中肆无忌惮的做事，不是洪荒生灵，那就是变数。变数在开天之初，天道不显的时候，还不重要，但是一旦天道出世，变数就会被天道第一时间注意到，之后毁灭。
<br/>　　所以，元释从决定道路之后，就在算计着洪荒之中的新身份了。在洪荒之中，与大气运、大机缘、大富运的人，也就那么几个。但他们中每一个都与盘古有着或多或少的联系。
<br/>　　三清、十二祖巫、妖族等都是盘古造化众生所化的生灵。换句话说，他们都是盘古的后代。
