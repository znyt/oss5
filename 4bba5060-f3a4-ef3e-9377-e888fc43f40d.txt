　　岁月悠悠，不知过了多久，洪荒之中，人族却是迅速发展起来，大小部落遍布大半个洪荒，真正算得上是洪荒第一种族了。
<br/>　　却说人族部落之中，有一部落名为风氏部落。部落之中，有一女子，出生于华胥山之边，便称之为华胥氏。此女相貌清秀，颇有点大家闺秀的模样，心地善良，于人族之中甚有好评。
<br/>　　风氏部落之西，有一雷泽，雷泽之中有雷神，人身龙首，鼓其腹，犹如奔雷。部众古老传言，雷神乃是居住在天上的神仙，执掌天雷，惩罚人间的一切恶人，是以，雷泽乃是部落的神圣禁地，无论任何人都不能私自进入雷泽，否则将被驱逐出部落！
<br/>　　华胥氏便是从小听着雷神的故事长大的，所以她对雷神非常好奇，只是她不能违背族规，只好将它深深的埋在心中。
<br/>　　一日，一道闪亮的电光划破天际，向着雷泽方向落去，恰巧被华胥氏看到。华胥看到那道亮光，顿时便想起了部落之中一直流传的雷神传说，不由得眼前一亮。华胥想要去雷泽看看，又怕被人知道后驱逐出部落。在一阵天人交战后，最终还是她的好奇心占了上风。于是，华胥悄悄向着雷泽方向行去。
<br/>　　如此走了半天，华胥终于来到了雷泽边上，看着近在咫尺的雷泽，华胥又有些犹豫了，毕竟自己一旦进去，被人知道后便要逐出部落。一番天人交战后，最终还是好奇心占据了上风，华胥心道：
<br/>　　“既然自己都已来到了此处，不进去看看实在是不甘心，而且，进不进去如今都已经没有多少区别了。”当下便不再犹豫，她迈步进了雷泽。
<br/>　　进入雷泽之后，华胥并没有看到什么雷神，迷雾倒是有好大一片。她带着好奇心走了进去，一路之上做下标记，防止迷路。华胥在雷泽中又是行了半个刻钟，顿时见到前方出现一大片闪闪发光的光晕。
<br/>　　华胥大为好奇，快步走了过去，走到近前，华胥才惊讶的发现前方不远处，有着一小把闪烁着白色光晕的玉琴，那玉琴在光华之中不断流转，起起伏伏，旋转不断，好似在吸收着周边的灵气。
<br/>　　雷神没有见到，反而见到一把小琴，这是怎么回事？难道这就是传说中的灵宝吗？华胥心中颇为好奇，便走近了过去。
<br/>　　“嗖！”
<br/>　　正在此时，一道光彩闪烁，华胥还没来得及闪避，那道光彩便瞬间冲进身体之内，消失不见了。华胥在光彩入体之后，并没有感受到什么不适，她心中顿时松了一口气，快速向四周查看几眼，在并没有发现其他人后，顿时就赶紧向着雷泽之外跑去。
<br/>　　华胥回到部落之后，并没有发生什么事，只是在两个月之后，她发现自己的腹中好像多了什么，而且之后，她的肚子一天比一天大了起来。周围人看向她的目光之中也带着浓浓的古怪，后来一阵探查后，才知道，自己这是怀孕了。
<br/>　　华胥未婚先孕，部落里的长老出面，向华胥问起了原因。此时华胥哪还敢有隐藏啊，她也顾不得其他，顿时将自己如何看到亮光，如何到达雷泽，如何见到玉琴之事，一一向长老讲明。
<br/>　　众人听了华胥所讲的事情的经过后，只觉得华胥怀了一个怪物，他们看向华胥的眼神之中顿时变得厌恶了。但念在华胥平日间为人和善，却是没有直接将她打杀，而是将她赶出部落，让她自生自灭。
<br/>　　华胥没了部落，一时之间大受打击，周围之人看向她的目光之中也带着浓浓的古怪，还有一丝厌恶。华胥走出部落，来到靠近部落的渝水便自己搭了一个小屋，一个人住在其中。平时部落中与她还算好的人，偶尔来看看她，给她送些食物，她还算能够活下去。
<br/>　　随着时间的流逝，华胥的肚子也越来越大了，而且已经影响到了她的步行，她也无法劳动了。部落中的亲人也无法经常来看她，于是，她不得不为自己今后的食物问题担忧。
<br/>　　就在华胥为食物的问题忧心之时，只见天上飞来一直麒麟，它口中叼着一片荷叶，荷叶之中，有一些果子与食物。华胥看见麒麟，心中甚是害怕，躲在屋中不敢出来，直到麒麟远走后，她才敢出现将食物取走。如此几番之后，华胥也就习惯了麒麟来给她送食物，她心中也就不再害怕了，便慢慢在此生活下来。
<br/>　　……
<br/>　　昆仑山中，通天教主正在为他的徒弟们开讲大道，他口吐大道之言，妙语连珠，道韵显化，龙飞凤舞，造化天地，玄之又玄。
<br/>　　讲道数日之后，通天教主便不再讲道，遣散了众人，之后留下自己最小的一个真传弟子——龟灵圣母。
<br/>　　通天教主座下有四大真传弟子，大弟子多宝道人，二弟子金灵圣母，三弟子无当圣母，四弟子便是这位龟灵圣母了。至于其他的，便都是一群普通弟子或者记名弟子。
<br/>　　龟灵圣母虽然最小，却最得通天教主喜爱。她的前身便是通天教主在补天之时斩下的巨鳌，因为补天有功，得到无数功德护身，通天帮助她转世之后，今生拜入通天门下。虽然她是后来者，但因为身具功德，颇有福缘，修为增长却是最为快速，如今已是大罗金仙了，确实不凡。
<br/>　　“龟灵，人族之中，有一部落，名为风氏，不久之后，人族之皇便会降落于此，到时，你便将其收为弟子，好好教导，你可明白？”
<br/>　　“弟子谨遵老师法旨！”
<br/>　　龟灵圣母连忙答道。于是，之后龟灵连忙出了昆仑山，向着人族部落而去。
<br/>　　这一日，龟灵无意中听到，风氏部落中，有人谈起华胥之事，她顿时觉得奇怪，心中觉得此事甚是蹊跷，恐怕没那么简单。之后，她向人问清华胥氏的住处后，便飞了过去。
<br/>　　可就在她靠近华胥氏之时，突然风起云涌，九天之上，电闪雷鸣，蓦然之间，光华大涨，紫气冲天，星辰之中，一道道紫气从天而降，华胥氏的屋子之中，霞光大作，异象满屋，隐隐间有神龙缭绕。
<br/>　　紧接着，一声嘹亮的哭泣声传出，传遍天地间，传入众人的脑海之中。
<br/>　　同时，洪荒之中的几位圣人，也都知道了人族圣皇降生了。
<br/>　　龟灵圣母从天而降，落到华胥氏屋子前，华胥氏大惊，连忙抱起怀中的孩子，她害怕别人要伤害她的孩子。
<br/>　　“你便是华胥氏？”只听龟灵圣母问道。
<br/>　　“仙姑乃是何人？”华胥氏抱着手中的孩子，警惕的问道。
<br/>　　龟灵圣母微微一笑，毫不在意华胥氏的态度，她笑道：“你不必担忧，贫道乃上清圣人截教通天教主座下四弟子，龟灵圣母是也，姑娘可是华胥氏？”
<br/>　　华胥一听，心中顿时大惊，圣人门下，她自是知晓，一个个都是神通广大的神仙，她不敢怠慢，连忙回道：“人族华胥，拜见仙姑！”
<br/>　　龟灵连忙将她扶了起来，看着华胥手中的孩子，她掐指算了起来，不过并没有算到什么，只是算到眼前的孩子将来是人族大贤。而就在这时，她又听到了通天教主的传音，她突然一愣，说道：
<br/>　　“姑娘，贫道欲收你的孩子为徒，你可否愿意？”
<br/>　　听了龟灵圣母的话后，华胥氏心中一愣，之后便是大喜，连忙拜谢道：
<br/>　　“愿意，愿意，多谢仙姑！”
<br/>　　龟灵微微一笑，道：“不必多礼，对了，你的孩子可有名字？”
<br/>　　“风氏伏羲！”
<br/>　　华胥微微一愣，她怎么知道叫什么？她还没有想好呢？不过，就在此时，冥冥之中好似有什么在干预着她，让她不由自主的说出这样一个名字。
<br/>　　“伏羲？！”
<br/>　　龟灵圣母心中一惊，她可是知道，巫妖时代，伏羲乃是妖族大圣，女娲的哥哥。
<br/>　　而华胥手中的孩子，在听到这个名字之后，却也是目瞪口呆。
<br/>　　……
