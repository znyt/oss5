        但是这也不能怪傅夜沉吧！
<br/>
<br/>        又要爬傅夜沉的床，又不愿意给他傅夜沉生孩子，谁叫她童筱颖自己喜欢作死呢？
<br/>
<br/>        落以柔一心向着傅夜沉，却忍不住给童筱颖出起了馊主意：“我说你这吃什么事后药啊！还不如去医院里上环，那就不用担心会怀上孩子了。而且也不会太伤身体。”

<br/>        她这话一落，童筱颖眼皮子抬了抬，目露凶光地瞪向落以柔，毫不客气地呵斥：“说完了吗？说完了，你可以走了！”
<br/>
<br/>        “是，总裁夫人，我就不打扰你沐浴更衣了！”
<br/>
<br/>        落以柔鄙夷地睨了童筱颖一眼，说完，转身就走。
<br/>
<br/>        童筱颖心情十分糟糕地目送落以柔的背影往楼下走去，待那个多嘴的女人彻底离开自己的视野后，她才抱着衣服正准备从床上下来时，双腿间隐隐透着一丝的灼痛感。
<br/>
<br/>        他对她的狠，让她一想起来就感到不寒而栗。
<br/>
<br/>        忍着痛去了浴室，冲洗掉傅夜沉留在她身上的那些污秽，重新换好了衣服，她连午饭都未吃，便直接回到设计部里继续去工作。
<br/>
<br/>        下午下班前，她把审核好的图纸和审核记录，一并打包发给了部长孙莉莉，算是完成了今天的工作任务，接着又向孙莉莉要了明天要审核的设计图。
<br/>
<br/>        童筱颖继续回到办公桌前工作，中途还给自己点了份外卖，一直加班到晚上十点，仍旧没有要下班的意思，办公室里的其他同事都已经离开了。
<br/>
<br/>        她不想回家，不想去面对傅夜沉，更不想沦为他生孩子的工具。
<br/>
<br/>        既然他派人监视着她，那好，她索性哪里都不去，就坐在这办公室里。
<br/>
<br/>        结果，她才熬到下半夜，就有些熬不住了。
<br/>
<br/>        就在她昏昏欲睡的时候，忽然感到身子腾空而上。
<br/>
<br/>        她倏然睁开双眼，映入眼帘的是傅夜沉那对炯亮如猎豹般的黑眸，令她心悸得差点喘不上气来。
<br/>
<br/>        童筱颖瞬间挣扎起来，手脚并用地推搡着傅夜沉。
<br/>
<br/>        他刚把她抱起，经不起她的折腾，不得不将她放了下来。
<br/>
<br/>        童筱颖刚站稳脚，就马不停蹄地往后退，直到后背靠到墙角，待他长腿挪步靠近之际，她不知道从哪里摸出一把丁字尺来，直指傅夜沉的胸膛，呵斥：“你别过来！”
<br/>
<br/>        上午，他愤怒地强要了她的那件事，算是给她造成了心理阴影。
<br/>
<br/>        没有哪个女人愿意被自己的丈夫在婚内强暴，不是吗？
<br/>
<br/>        童筱颖鼻子一酸，凝视着傅夜沉的那双眸子，瞬间被眼泪充盈，而潸然泪下。
<br/>
<br/>        “筱颖……”傅夜沉心疼地凝望着这个哭得像泪人儿一样的小女人，试图往前一步。
<br/>
<br/>        他真的受不了她的眼泪，她一哭，他的心痛得就像被什么东西给死死地揪住。
<br/>
<br/>        童筱颖心悸地歇斯底里道：“我叫你别过来，听到没有？你走开！我不想再见到你！”
<br/>
<br/>        “上午的事情，我向你道歉。”傅夜沉垂下眼帘，驻足深沉道。
<br/>
<br/>        童筱颖却微微摇了摇头，持着丁字尺的手，因为害怕而不禁颤抖了起来：“我不需要你的道歉，你别碰我，我求你走开。”

<br/>        这条尺子，已经丈量出了她对他心理上产生的距离。
