　　萧逸与萧若寒的打赌，是萧逸能否凭借自身抢得进入紫云洞修炼的名额。
<br/>　　所以，他现在要上场应战，而要对战的，自然是那些已经取得了名额的凡境六重子弟。
<br/>　　当萧逸喝出一声‘谁敢来战’时，那些自认天资卓绝，高人一等的凡境六重子弟，纷纷露出了难看的脸色。
<br/>　　“竟然敢跟我们叫板，不自量力。”
<br/>　　“虽然我不屑于和他打，但我却不介意撕了他的嘴，看他还敢不敢这么狂妄。”
<br/>　　忽然，一道身影跳上比武台。
<br/>　　来人竟是萧星扬。
<br/>　　“萧逸，你还没那个资格和最出色的子弟对战，就让我来会一会你吧。”箫星扬说着，眼中露出了不屑之色。
<br/>　　“你？”萧逸眉头一皱。
<br/>　　这时，眼尖的他发现，不远处，九长老就竟站到了五长老等人身旁。
<br/>　　“原来如此，五长老真够阴险的。”萧逸眼睛一眯，瞬间明白过来。
<br/>　　进入紫云洞修炼，除了凡境六重的弟子外，还有两个名额可以直接被获得，而不用进行比武。
<br/>　　分别是萧若寒的名额和自己的名额。
<br/>　　萧若寒作为家族第一人，自然没人敢和他抢。
<br/>　　但自己的嘛....
<br/>　　一旦自己今天败了，打赌输掉，那么名额就会让出来。
<br/>　　那么，箫星扬作为凡境五重那一轮的比武第一，仅次于凡境六重的弟子，将得到这个名额。
<br/>　　难怪半月前，五长老等人会给出那样的选择给自己，要么不要少家主之位，要么不要修炼名额。
<br/>　　无论自己选择哪一样，都会陷入五长老早就埋好的陷阱。
<br/>　　不要少家主之位，五长老自然更容易进行下一步算计。
<br/>　　不要修炼名额，五长老将因此而拉拢到九长老，加上之前拉拢的七、八两位长老，他即将拥有半数以上的长老支持。
<br/>　　到时候，他同样可以很轻易开展后续的算计。
<br/>　　“这老乌龟，一次又一次地算计我，真是可恶。”萧逸心里暗骂着。
<br/>　　不过，他还不惧。有三长老在背后护着，他不需要有其他顾虑。
<br/>　　三长老是当年家主失踪前钦定的代理家主，名正言顺，自然受到大部分族人的支持。
<br/>　　单论家族里的权利，三长老一人就顶得上数位长老。
<br/>　　箫星扬见萧逸沉默，以为他怕了，当即笑道，“怎么，小废物，你怕了？放心，我不会要你的命，最多切磋一番。”
<br/>　　其实他心里想的是，不会要你的命，但我会将你打残，彻底废掉。
<br/>　　萧逸耸耸肩，道，“既然你自己当出头鸟，那就别怪我了。”
<br/>　　一旁的裁判见状，当即道，“好，既然箫星扬出战，萧逸应战，那么比武开始。但记住了，同族比武，点到即止，不可伤人。”
<br/>　　“是。”箫星扬恭敬地应答一声，但其实他早就想好了，比武嘛，难免会有意外。
<br/>　　“小废物，去死吧。”箫星扬瞬间祭出了蛮火牛武魂，力气陡然间增大，一拳朝萧逸砸去。
<br/>　　“呵，这一拳下去，那小废物死定了。”
<br/>　　“区区凡境一重，箫星扬就算不用武魂，萧逸也必败无疑。”
<br/>　　五长老和萧若寒仿佛已经看到萧逸被吐血击飞的场景。
<br/>　　然而，‘啪’的一声。
<br/>　　在所有族人不可置信的目光中...
<br/>　　萧逸竟然不闪不躲，轻轻一抬手，正面接下了箫星扬的拳头。
<br/>　　“正面挡下箫星扬的拳头？这怎么可能！”
<br/>　　箫星扬可是凡境五重阿，萧逸不过是凡境一重，彼此的差距应该极大才对。”
<br/>　　确实，凡境中，每一重的差距都极大，力量起码在数百斤以上。
<br/>　　两三个凡境四重加起来，都不一定打得过一个凡境五重。
<br/>　　“不对。”一些眼尖的族人忽然惊呼道，“萧逸不是凡境一重，观他刚才接箫星扬的一拳，力度绝对达到了凡境五重的水准。”
<br/>　　“什么？萧逸达到凡境五重了？”大部分族人和年轻子弟们都露出了不可思议的表情。
<br/>　　比武台上，裁判认真地打量了萧逸一眼，随即点了点头，“好小子，果然是凡境五重。”
<br/>　　“什么！凡境五重？”五长老和萧若寒顿时大惊失色。
<br/>　　“凡境五重，逸儿竟然是凡境五重的修为。好，很好。”三长老欣喜若狂，他甚至没有注意到，自己激动下，竟然将角落里的一块石头捏得粉碎。
<br/>　　比武台上，听着族人们惊讶的表情，箫星扬原本得意的脸色变为阴冷，“凡境五重又怎样，哼，在凡境五重中，我是无敌的。只要这废物不达到凡境六重，他就不是我的对手，修炼名额是属于我的。”
<br/>　　想罢，他加大了力度，再次一拳朝萧逸打去。
<br/>　　“哼。”萧逸冷哼一声，在他眼中，箫星扬的攻击处处都是破绽。
<br/>　　轻轻一甩头，躲过箫星扬的拳头，而后一把反扣，将箫星扬整个倒摔。
<br/>　　同为凡境五重，萧逸有一百种办法打败箫星扬。
<br/>　　而对付箫星扬，他甚至全力出手都不需要。
<br/>　　‘嘭’的一声，箫星扬重重地摔在了地面上。
<br/>　　“虎形。”萧逸眼中一冷，双掌成爪，势大力沉的攻击打出，一爪扣住了箫星扬的咽喉，一爪扣住了箫星扬的脑门。
<br/>　　双爪成虎噬之势，只要他愿意，顷刻间就能划破箫星扬的咽喉或者脑门。
<br/>　　同时，他的脚狠狠踩在箫星扬胸膛上，让箫星扬动弹不得。
<br/>　　若箫星扬还敢反抗，他将双爪合攻，将箫星扬的脑袋捏爆。
<br/>　　形意五绝可不是单纯死板的攻击，彼此间可千变万化。萧逸完全可以根据不同的战斗情况，发挥出最适合的攻击，一招败敌。
<br/>　　“你最好别动，否则，我不介意现在就杀了你。”萧逸冷冷地说道。
<br/>　　感受到萧逸眼中的杀意，以及咽喉上的疼痛，箫星扬顿时不敢动弹。
<br/>　　台下族人中，早已引起了轩然大波。
<br/>　　“好厉害，萧逸竟然一招败敌。”
<br/>　　“出手的时机竟然拿捏得如此准确，恰好是箫星扬出拳未收，无法回房之际。”
<br/>　　“还有他的攻击，竟然瞬间扣住了箫星扬的要害，让箫星扬再无反抗的可能。”
<br/>　　族人们纷纷赞叹，但有些不敢相信自己的眼睛，什么时候萧逸变得如此厉害了。
<br/>　　台上，萧逸冷冷地看着箫星扬，道，“记住了，辱人者人恒辱之，以后管好你的嘴巴。”
<br/>　　说罢，萧逸重重一脚踢出，将箫星扬踢下了比武台。
<br/>　　箫星扬的身躯重重地跌落台，摔了个狗吃屎。
<br/>　　萧逸傲立于比武台，再次看向凡境六重的子弟们，冷冷道，“现在，我有资格和你们对战了吧。接下来，谁来一战？”
<br/>　　充满挑衅意味的话语，一瞬间让这些子弟们脸色大变。
<br/>　　他们不是傻瓜，单看刚才的战斗，萧逸能瞬间打败箫星扬，代表着萧逸完全拥有着凡境六重的战力。
<br/>　　他们可不想随便当出头鸟，更不想随意在场上被萧逸打败，作为家族最出色的子弟，他们可不愿意丢脸。
<br/>　　这时，不远处的五长老和萧若寒对视了一眼，并不着痕迹地点了点头。
<br/>　　“小废物，就算你达到凡境五重了又如何，和凡境六重比，你还差远了。”萧若寒阴冷地想着。
<br/>　　同时，他隐晦地对着一个子弟打了个眼色。
<br/>　　那个子弟点了点头，而后站了出来，跳上比武台。
<br/>　　“萧逸，我来会一会你。”
<br/>　　来人是萧子木，凡境六重，在家族年轻一代的实力排名中，足可排前五。
<br/>　　萧子木祭出了火旋棍，甩了漂亮的棍花后，一指萧逸，道，“萧逸，遇到我，算你倒霉，三棍之内，你必败。”
<br/>　　其实，萧若寒早就对他施压，并让他收拾萧逸。
<br/>　　萧逸怡然不惧，淡淡道，“就凭你，还没那个实力胜我。”
<br/>　　“猖狂。”萧子木一棍朝萧逸袭去。
