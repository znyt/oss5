　　清晨，初生的暖阳从窗外照射进来，将屋内的昏暗驱散一空。
<br/>　　在一片温香软玉中醒来，鼻尖环绕的是那令人心情愉悦的肉香，只是那股少女身上特有的体香，稍稍破坏了刘哔还算不错的心情。
<br/>　　“阿嚏！”
<br/>　　打了个喷嚏，刘哔下意识的想要揉揉鼻子，却发现，有些不太对劲。
<br/>　　手臂动不了了，被人给……压住了。
<br/>　　眉头微微一皱，刘哔将手臂从白小飞身下抽出，随后拿开了环在他腰间的雪白藕臂，动作轻盈的下了床。
<br/>　　这是晓佳的单身公寓，只有一张床，本来他们原本提议打地铺，但连多出来的被褥也都没有。
<br/>　　除了小胖，因为体型原因，被无情的赶去沙发上，剩下的五个人，都挤在了同一张床上。
<br/>　　索性晓佳的床还算大，几人又都不怎么胖，也勉强能睡得开，只不过有些拥挤罢了。
<br/>　　趁着众人还没醒，刘哔从空间戒指中取出几件装备，戴上之后，开门走了出去。
<br/>　　片刻，他站在了阳光医院门前的深坑边。
<br/>　　没有那些累赘，他自己一个人的速度快上了许多，这么远的距离，也只花费了一两分钟的时间。
<br/>　　“主人，您来了。”
<br/>　　有些畏缩的声音响起，刘哔‘恩’了一声，转过身来，看到身披白大褂的博士。
<br/>　　昨天他杀掉的，只是顺手从医院拽过来的一个尸兄而已。
<br/>　　博士的试验品那么多，即使被他清除了一些，也还剩不少，找一个体型特征差不多的尸兄，并不是难事。
<br/>　　“给你交代的事情，进展怎么样？”
<br/>　　抬脚往医院内部走去，刘哔和博士攀谈着。
<br/>　　“前期准备已经进行的差不多了，现在就缺一些实验品，虽然尸兄也可以，但速度估计要慢上很多。”博士的声音有点迟疑，随后继续说道：“要是有机会的话，还是用人体或动物做实验比较好。”
<br/>　　点了点头，刘哔没说什么，他要看一看具体的实验成果，再做决定。
<br/>　　片刻，两人来到了一间地下研究所内，里面摆放着几罐营养液，泡着一些尸兄的躯体。
<br/>　　这些尸兄，都是经过特殊改造的，但似乎效果并不是特别明显，和外面那些尸兄比起来，也大体没有多少区别。
<br/>　　“就这样吗？”
<br/>　　刘哔停下了脚步，语气有些不满。
<br/>　　“强化药剂对已经尸变的个体，起不到什么效果，这样都是我用一些别的手段催化，才有机会发生变异的。”
<br/>　　博士脸上的表情也有些无奈，毕竟试验品太少了，哪怕他再天才，也不可能再没有试验品的情况下完成研究。
<br/>　　毕竟，在这样的地界，活人和动物并不常见，况且就算是见到了，也很难有机会将人给抓回来。
<br/>　　病毒已经肆掠了不短时间了，现在还能活着的，都不是傻子。
<br/>　　“你就不能脑子转一些么。”刘哔斜了一眼博士，语气有点无奈：“比如，发个广播，说明这里是安全点，免费提供食物和淡水，这样还能吸引不了人来？”
<br/>　　博士闻言一愣，陷入了沉思，似乎在考虑计划的可行性。
<br/>　　叹了口气，刘哔心中有些嘀咕，这些热衷于研究人员，是不是在其他方面都是死脑筋。
<br/>　　这么简单的方法，诸如之类的还有很多，博士就不知道转动一下。
<br/>　　“我懂了，接下来我就去办，如果顺利的话，第一批货就能出来了。”
<br/>　　博士语气稍稍有些兴奋，刘哔给他提议的办法，为他打开了一座崭新的大门，在这个建议上面，他也想到了很多自己的想法。
<br/>　　说到底，这只是个概念问题，在此之前，博士根本没有这样的概念，但有了这个概念之后，以他的智商，自然能想到很多更为完善的事情。
<br/>　　“恩，那就先这样吧，这边暂时教给你，我估计很长一段时间，都不会过来了。”
<br/>　　刘哔点了点头，仔细想了一下，确定没有什么问题之后，从地下研究所内走了出去。
<br/>　　……
<br/>　　几分钟后，刘哔额头上带着一层细密的汗水，敲响了晓佳公寓的大门。
<br/>　　“诶，你回来了呀，正好，饭做好了，快来吃吧。”
<br/>　　小惠打开大门，轻轻侧了侧身，脸颊上飘起两朵红云。
<br/>　　她还记的昨天洗澡的事情，这还是第一次，有男人看光了她的身体。
<br/>　　“喂，老兄，我们看见了你留的字条。”白小飞坐在沙发上，手中把玩着电视遥控器：“不过话说回来，外面这么危险，你都要去晨跑吗？”
<br/>　　顺手用挂在脖子上的毛巾擦了擦汗，刘哔爽朗的笑了一声，坐到餐桌上：“十多年的习惯了，一时半会改不了，要是不跑的话，浑身难受。”
<br/>　　颇为理解的笑了一声，白小飞关上电视，从沙发上站了起来。
<br/>　　“话说，还不知道你的名字呢，总不能一直叫你老兄吧？”
<br/>　　听到白小飞的问题，其他几人都竖起耳朵，她们也都十分好奇。
<br/>　　“哦，说起来，我们的名字差不多呢。”刘哔看了看面前的鸡蛋面，眉头微不可查的一皱：“我叫王飞，朋友们都叫我老王。”
<br/>　　众人的表情都有些古怪，白小飞大笑一声，没心没肺的道：“老王啊，真是个好名字呢，哈哈。”
<br/>　　“额，和隔壁老王没关系，我的爱好也不是人妻。”摊了摊手，对于白小飞的调笑，刘哔并没有丝毫在意。
<br/>　　反正他本来就不叫王飞。
<br/>　　“好啦，快吃饭吧，吃过饭，我们还要出去帮小鹿找家人呢。”
<br/>　　小惠打了个圆场，贤惠的将筷子分发给众人，随后，几人面对香喷喷的鸡蛋面，大快朵颐了起来。
<br/>　　虽然只是很普通的面条，但是在这样的环境下，却给人一种难以言表的温馨。
<br/>　　热气腾腾的饭菜，单纯可爱的熊孩子，温柔贤惠的女人，还有自幼相交的好友。
<br/>　　这样的生活，很容易让人产生一种家的归属感。
<br/>　　如果……能一直这样下去，世界末日真的不算什么吧。
<br/>　　脸上带着阳光的笑容，白小飞暗下决定，哪怕是付出生命，也一定会守护住这片美好。
