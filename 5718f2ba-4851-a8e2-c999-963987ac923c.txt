　　而此时，
<br/>　　全球各大论坛、网站之上，已经越来越多的人，加入到观看直播的行列，听到叶尘突然开口说话，顿时彻底炸开了。
<br/>　　“他说的好像是……华夏语？”
<br/>　　“原来不是外星人，他竟然是华夏人！”
<br/>　　“难道是华夏国培养出来的超人吗？太可怕了！”
<br/>　　“他这是想做什么？”
<br/>　　……
<br/>　　所有人望着国会山上方，那道犹如天神般的金色身影，尽管隔着屏幕，都感到一股难以形容的压迫感。
<br/>　　米国总统普朗克，颤颤巍巍从人群中走了出来，躬身行礼道：
<br/>　　“尊敬的叶先生！我是米国总统普朗克，我代表联邦政府，真诚的向您请求休战！”
<br/>　　哗！
<br/>　　普朗克此话一出口，无论是在场的众多议员，还是周围的各部士兵，以及通过网络直播观看的全球网民，顿时尽皆哗然。
<br/>　　堂堂全球霸主，当世第一大国的总统，竟然在向这个华夏少年求和！
<br/>　　这绝对是米国自建国数百年以来，从来都没有出现过的事情！
<br/>　　众议员以及周围的米国士兵，个个一脸惨然之色，简直无法接受此等奇耻大辱。
<br/>　　可是，他们也都清楚，不接受也得接受！
<br/>　　连核弹都奈何不了此人，除了求和之外，还能如何？
<br/>　　除非拼个鱼死网破，让整个华盛顿乃至整个米国一同为其陪葬！
<br/>　　然而，
<br/>　　任谁也没有想到，面对普朗克忍受着奇耻大辱，主动求和，叶尘却冷冷一笑，
<br/>　　“现在求和？未免太晚了些吧！”
<br/>　　若非之前他反应够快，在那百万吨级的核弹轰炸之下，就算不死恐怕也得脱层皮，此事叶尘自然不可能就此作罢。
<br/>　　普朗克和众人听到这话，顿时个个脸色再次狂变！
<br/>　　对于米国总统的主动求和，这家伙竟然还拒绝了！
<br/>　　“阁下虽然实力强大，可做事还是不要太过嚣张跋扈的好！我们之所以主动求和，是为了联邦子民的安全着想，可并不是因为怕了你！”
<br/>　　众议员之中，突然战出一名身穿军装的白人男子，仰头冲着叶尘，冷然暴喝道。
<br/>　　叶尘两眼微微一眯，
<br/>　　“是吗？你们，不怕？”
<br/>　　说完这话，叶尘直接随手一挥，
<br/>　　轰！
<br/>　　一道长达近千米的紫色剑芒，自空中狠狠劈落下来！
<br/>　　彭！
<br/>　　轰隆隆！
<br/>　　代表着米国最高权威的国会大厦，竟然被那道恐怖的剑芒，一剑劈成了两半！
<br/>　　甚至连同整个国会山，都狠狠晃动了一下，犹如随时都要倒塌一般。
<br/>　　下一刻，
<br/>　　普朗克等人无比震惊的发现，不但国会大厦瞬间成为了一堆废墟，连同他们跟前，也出现了一道足足数十米深的巨大鸿沟！
<br/>　　“嘶~~”
<br/>　　所有人纷纷倒吸了一口凉气，而且之前说不怕的那名议员，顿时吓得浑身瑟瑟发抖，直接没能忍住，竟“噗通”一声，跪倒在了地上，再也不敢多说一个字。
<br/>　　倘若方才叶尘这一剑，对准的是他们，他丝毫不怀疑，他们这些人此刻，恐怕早已经全部死在了对方剑下！
<br/>　　若非亲眼所见，他实在想象不到，对方一击之威，竟然恐怖到如此程度，简直不弱于一枚小型核弹，而且关键对力量的控制，简直精准的可怕！
<br/>　　其余众人，同样也被叶尘这一手，震惊的无以复加，原本仅存的那一丝反抗念头，此刻也顿时荡然无存。
<br/>　　他们这些人，虽然个个身居高位，可越是身居高位，越往往怕死！
<br/>　　更何况，想要杀掉眼前这个人，除非他们抱着同归于尽的决心，赔上整个华盛顿数百万的人口！
<br/>　　“阁下到底想要怎样才愿意罢手？还请明言！”
<br/>　　最终，国防部长拉姆斯，深吸了一口气，上前一步，沉声开口道。
<br/>　　叶尘轻哼一声，
<br/>　　“很简单！我要的不是求和，而是无条件投降！”
<br/>　　轰！
<br/>　　叶尘此话一出口，国会山周围的众人，顿时个个群情激愤，而此时正在观看直播的全球网民，也全都尽皆愕然。
<br/>　　无条件投降！
<br/>　　也就是说，米国要彻底放弃抵抗，完全交由眼前这个华夏少年，来主宰他们一国的命运。
<br/>　　这已经不是奇耻大辱，而是要亡国了！
<br/>　　“阁下这个要求，未免太过分了些吧？”
<br/>　　拉姆斯强忍着心头的恐惧，抬头怒声道。
<br/>　　叶尘却一脸云淡风轻，
<br/>　　“若不投降，那便战吧！”
<br/>　　说完这话，叶尘再次举起手中的紫琼剑，
<br/>　　轰！
<br/>　　上千米的恐怖剑芒，再次出现在众人面前。
<br/>　　米国国会数百米议员，顿时个个脸色煞白，有的甚至已经直接吓晕了过去。
<br/>　　他们丝毫不怀疑，叶尘这一剑落下，他们所有人肯定必死无疑！
<br/>　　最终，
<br/>　　米国总统普朗克叹息了一声，缓缓道：
<br/>　　“倘若我们投降，阁下能否不要伤害无辜的市民？”
<br/>　　叶尘淡淡道：
<br/>　　“我从华夏至此，可曾伤过一名无辜市民？”
<br/>　　众人顿时愕然，仔细想想，还真是如此！
<br/>　　叶尘自太平洋一路杀到这里，的确没有杀一名普通米国公民。
<br/>　　“那投降之后，阁下准备如何处置？”
<br/>　　米国总统再次追问道。
<br/>　　所谓的无条件投降，战败国没有任何谈判的资格，必须完全接受对方的处置，就像当初的岛国一样，至今都在米国的控制之下。
<br/>　　叶尘略微沉吟了片刻，方才道：
<br/>　　“两个条件！”
<br/>　　“第一，今后米国军队不得再踏入亚洲半步！”
<br/>　　“第二，10万亿美元的战争赔款！”
<br/>　　呼！
<br/>　　听到这两个条件，普朗克和国会众议员，纷纷松了一口长气。
<br/>　　叶尘所提的这两个条件，并不算怎么苛刻，倘若真是如此，倒也不是不能考虑。
<br/>　　只不过，他们一旦投降，就要永久退出亚洲，并且全球霸主的地位不保，而他们，也将会被永远钉在历史的耻辱柱上。
<br/>　　其实，他们又哪里知道，叶尘之所以非要逼迫米国政府投降，除了要为自己出一口恶气之外，还想以此来威慑全球。
<br/>　　他要让所有人都知道，连米国这样的当世第一强国，都被他叶尘踩在脚下，看今后谁还敢再找他家人的麻烦？
<br/>　　“给你们一个小时的时间考虑！”
<br/>　　“一个小时之后，要么，交出投降书，要么，死！”
<br/>　　“当然，你们也可以选择，将你们的所有核弹引爆，让三亿米国公民为你们陪葬！”
<br/>　　撂下这句话，叶尘双手负于身后，闭上双眼，不再开口说话。
