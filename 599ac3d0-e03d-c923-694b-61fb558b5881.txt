　　其中一顶轿子里，坐着真正的后卿和女魃。
<br/>　　“我觉得，你不该亲自来的，反正一切都安排好了，你来了未必是好事。”女魃有点担心地说道，“林三生那家伙诡计多端，我怕他万一看穿，你会有危险。”
<br/>　　“正因为他诡计多端，我才更要亲自来，战局瞬息万变，我不来如何放心。”
<br/>　　女魃搂着后卿的胳膊，把脑袋靠上去，这个充满邪性的女人此刻在他身边却温顺地像一只小猫。
<br/>　　“等一切结束，我把摇光给你吧，让他服侍你我，她身子很好的……”女魃一脸坏笑地舔了下嘴唇。
<br/>　　后卿面无表情。
<br/>　　女魃佯怒道：“我就知道，你心里还想着那个谁。”
<br/>　　“等这一仗打完，你再慢慢吃醋吧。”
<br/>　　“好！”女魃咬了咬牙，“到时候你可拦不住！”
<br/>　　一天之后，尸族大军行进到山脉腹地。
<br/>　　女魃趴在后卿的腿上休息，后卿一只手抚摸着她脑后的长发，说道：“他们该来了。”
<br/>　　“万一他们不来呢？”
<br/>　　“那我们就杀过去，不过，这不可能。”
<br/>　　女魃突然侧过脸，双手抱住后卿的腰，从下面看着他的脸，喃喃说道：“夫君，我最近有时候会想，我们为什么一定要征服空界呢，就在零界里呆着也挺好，偶尔去一下人间，为什么，我们要不停地战斗？”
<br/>　　“我们，声来就注意要战斗的，零界暂时能够安生，也是打出来的，一旦空界强烈起来，他们势必要去剿灭我们，到那时候，我们想跑都没地方了。”
<br/>　　他伸手摸了摸女魃的脸。
<br/>　　“这一战如果赢了，我们可以修整很久，甚至以后都不用再打仗了。”
<br/>　　后卿透过黑布望着远处的山景，深吸了一口气，本来如止水一般的心中也泛起了小小的波动。
<br/>　　为了这一战，他准备了很久，与林三生斗智斗勇，费尽力气才突破层层防线，将大军开到这里来。
<br/>　　还差最后一步棋。
<br/>　　顺利走完这最后一步，就能一统空界，可以长时间修整一下，然后再徐图别的空间……
<br/>　　撕拉！
<br/>　　一阵尖锐的响声传来，走在山谷中最前面的尸兵本来走的好好的，突然撞在一张无形的网上，但后面的不明情况，还在往走，把最前面一批尸兵直接从网里挤了出去，切割成了一块块的碎肉。
<br/>　　等前面一批人没了，后面的又被挤到前面，当他们发现前面有网，想要站住，却又被后面不知情的同伴挤出去……
<br/>　　就这样最前面好几波尸兵被法阵所杀，惨叫声传到后面去，才越来越多的僵尸停下脚步……如果从高空看下去，整个尸族的队伍，就像一只在山谷里缓慢蠕动的大虫子。
<br/>　　不等这只“虫子”停下，埋伏在两边山上的联军便杀了出来，朝着山下这群完全懵逼的尸兵冲过去。
<br/>　　战斗一触即发，从局部蔓延开来。
<br/>　　黎山老母、摇光仙子，连同众阁派、蜀山派的宗师，各自带着本门弟子，在山顶飞行起来，直奔大军中间那顶黑色的轿子。
<br/>　　尸兵疯狂阻拦，阐教宗师们疯狂进攻，双方都拼了命。
<br/>　　好在投入战斗的联军士兵越来越多，前赴后继地开路，总算帮助这些大佬们冲到了黑轿子前。
<br/>　　“太上三清，御极真灵，急急如律令！”
<br/>　　黎山老母念咒，一边挥舞衣袖，一朵朵白色的梨花从袖子里飞出去，只听见一阵撕拉撕拉的声音，梨花飞处，将蒙在轿子上的黑布划开，中间……蹲伏着一道人影，周身被一层黑气覆盖着，看不到相貌。
<br/>　　黎山老母等人互相看了看，一起展开攻势，这黑影却哇啦一声怪叫，展开翅膀飞起来。
<br/>　　原来是一只飞僵！
<br/>　　大伙合力消灭，把战场交给联军士兵——这些士兵由林三生的提拔的一位将军统率——他们自己狂奔起来，前往另外一个峡谷。
<br/>　　战争是士兵的事，他们这几大门派的大佬，一共分成了四路，分别埋伏在尸族五路大军行进的山谷中，不干别的，只为了证明后卿不在这一路大军中。
<br/>　　这也是林三生的无奈之举，这几天来，他通过蛛丝马迹，确定了尸族主力最有可能会走的那条路，但也仅仅是最有可能，万一错了，后果不堪设想。
<br/>　　于是他集合了诸方大佬，在几条路上分别设伏，一旦确定后卿不在这路大军中，立刻就去邻近的那条峡谷增援，这就大大减少了判断失误带来的风险。
<br/>　　距离黎山老母他们几十里之外的一条峡谷中，几个佛宗大佬也掀开了一辆黑轿子的顶，这次遇到的是一只双头僵尸，皮糙肉厚的，几个老和尚费了半天劲才干掉他，接着通过信物传递信号，一起赶往相邻的峡谷。
<br/>　　呜……呜……呜……
<br/>　　一共三声。
<br/>　　林三生把形状如海螺一般的号角放下了，这种海螺叫龙号，是一种低级灵物，长在深海中，壳有灵性，彼此间就是通过壳来感知对方的位置，死亡之后，会有壳留下，是极为稀少的一种灵物，对人间法师来说，有相当多的作用。
<br/>　　林三生之前一直苦于如何在战斗中远程通信，距离近还好说，这都相隔几十公里，空界又没手机，后来老郭出了主意让他弄几个龙号的壳，这东西一般人弄不到，但橙子是鲛人公主，从深海挖几个东西还是没问题的，找来之后老郭加工了下，只要用灵力激活，就算在千里之外也能收到讯号，连位置也能感知到，跟道风的雪花玛瑙戒功能类似。
<br/>　　为了这次行动，林三生给每条线路的负责人都配了一只龙号，三声长就表示该线路没发现回后卿。
<br/>　　呜……呜……呜……
<br/>　　龙号里又传出信号声，第三条路也没有。
<br/>　　林三生深吸一口气，看来，这五路人马是一起到的。后卿也是够贼的，一锤子买卖，不给他们试错的时间。
<br/>　　“怎么样怎么样？”小青在一边追问道。
<br/>　　林三生还没回话，就有手下士兵飞奔来报，说尸族大军已经进入第四峡谷。
