　　天地悠悠，岁月匆匆，不知几许。
<br/>　　洪荒一片安然，各族强者、诸多大神通者们大多闭关悟道，只有人间处于动乱时期，有的修士想要借机寻找机缘，借此突破，纷纷做出各种谋划。
<br/>　　尼连禅河河水宽阔，大河涛涛，洋洋西去。
<br/>　　大日炎炎似火烧，河岸边的石子都被晒得无比滚烫，人踏在上面犹如大火炙烤般疼痛。
<br/>　　河畔边的菩提树苍劲而青翠的枝叶遮住了正午的阳光，热风吹动，树叶飒飒作响，泛起微微白光。
<br/>　　风中的梧桐树悄然无语，只是看着明净、展览的天空默默地展开它那纷繁茂盛的枝叶，一如它过往数千年岁月里所做的那样。
<br/>　　树下，一名瘦长的僧人双腿盘起，结跏趺而坐，僧人看上去大约三四十岁模样，年轻的身体因长期的苦行冥思而显得非常虚弱，偏袒的右半边身体处露出的肌肤苍白到近乎透明，头上淡栗色的卷发散乱如鸟巢，而且，好像还真有一对小鸟在乱发间筑起了鸟窝，生下几只雏鸟，忙碌地飞来飞去。
<br/>　　每当飞翔到阴影落下时，带来一股清凉的气流，僧人蓬松的短发中便会发出急促的叫唤声，几张嫩黄的小嘴迫不及待地张开又合上。
<br/>　　乔达摩?悉达多不言不动，他已经在这株古老的菩提树下整整静坐了九十天，不曾动过分毫，也不曾开口说过一句话。他在苦苦思索着道的真谛。
<br/>　　他已经在尼连禅河附近苦苦修行了六年了，六年中他深入极端苦行，摧残自己的身心，其他苦行者还以为他要就此丧命，但因为有着别人的帮助，他都度过来了。后来，他认为苦行无益，因为无法达到觉悟与涅槃，所以他放弃了。
<br/>　　他回忆起小时候在树下初禅的体验，认为这可能给他提供帮助，于是，他便来到了尼连禅河畔的菩提树下，静静参禅。
<br/>　　饿了，他有牧羊女苏耶妲的羊乳之供，脏了，他便在这尼连禅河中沐浴，一直坚持了三个多月。
<br/>　　雨季过去，旱季到来，灼热的热浪炙烤着大地，长期枯坐除了使自己的身体变得更加虚弱，他一无所获。
<br/>　　大地在强烈的阳光下干燥到了极点，干热的空气从地上升起，使地平线出现了轻微的扭曲。
<br/>　　这个时候，东方的地平线尽头却走来了一个人，那是一个道士，轻盈的脚步仿佛根本没有接触到地面，干燥的路上也没有扬起丝毫的尘埃。
<br/>　　那道士不急不缓的走着，很快便走到了菩提树前。
<br/>　　悉达多看清楚了此人的面貌，道士脚穿一双青鞋，手挽白色拂尘，数缕长须飘拂胸前，头顶挽着三个发髻，一副出尘的姿态。
<br/>　　长须道人走到悉达多面前，笑道：“大道须向变化中求之，怎会有枯坐灰身，闭心合智，便可明白至道之理？”
<br/>　　他手中白色拂尘，一挥击在悉达多肩头，悉达多豁然惊醒，睁开眼来，只见光影朦胧之中，一名相貌异丽道人背光而立，气度非凡，有种莫名的道韵散开，但他就是抓不住，差之分毫。
<br/>　　悉达多连忙起身双手合十问道：“老师来此，不知有何指教？”
<br/>　　道人大笑，并未回答他的话，只是将一个手指立起，上指苍天，在悉达多面前连晃数下。悉达多一见，浑身一震，喃喃道：
<br/>　　“我明白了，我明白了。”
<br/>　　悉达多面带微笑，盘腿坐下，道人朗声大笑，转眼便已消失不见。
<br/>　　菩提树下，僧人闭目坐下，不知过去多少时间，忽然睁开眼睛，眼前山仍然是山，水依旧是水，但在此刻悉达多看来，一草一木，一尘一沙，都在诉说着无尽的玄微奥妙。
<br/>　　道之三境，第一境看山是山，看水是水；第二镜，看山不是山，看水不是水；第三境看山仍是山，看水仍是水。如今悉达多便达到了这第三境界，返璞归真，回到最初。
<br/>　　悉达多一笑，天地山川万物尽皆同时微笑，僧人将头顶鸟巢轻轻取下，小心翼翼地放到菩提树上的枝干间，便要迈步而走，但因为多日不曾行走，身体处于极度虚弱状态，体力消耗几乎殆尽，导致自己身形不稳，差点晃倒在地。
<br/>　　对面走来一个牧羊女，正是苏耶妲，她看见悉达多满脸苍白，摇摇晃晃，便取下羊角挂着的陶罐，递给悉达多，道：“吃吧。”
<br/>　　“多谢。”
<br/>　　悉达多双手合十，接过陶罐，将满满地一罐羊奶一灌而尽，力气稍微恢复，便将陶罐递回，道：“多谢！”
<br/>　　牧羊女抿嘴一笑，接过陶罐，向着远处村庄走去。
<br/>　　悉达多缓缓走入尼连禅河，用河水洗去一身污垢，在波光涟漪的夕阳中，走上岸来，精神奕奕，大步而去。
<br/>　　与此同时，西方灵山的菩提树光芒四射，禅音缭绕，无尽道音响彻灵山，惊天动地。接引与准提见后，顿时面色大喜。
<br/>　　二十九岁出家，经历六年，三十五岁的乔达摩?悉达多终于在今日菩提树下成就正等正觉，大彻大悟。
<br/>　　从此，他在南北天竺的广阔大地上传经讲道，救助黎民，人们尊称他为释迦牟尼！
<br/>　　梵天王得知此事后，亲自下身前去请他讲解佛道，并且将佛道传遍世间，让人们去理解佛道，更是将佛教地位抬高到神教之上。
<br/>　　“轰——”
<br/>　　一声轰隆巨响自西牛贺洲方向传出，随后，道韵升腾而起，禅音响起，越响越亮，唱遍诸天，三界无尽大神通者顿时从闭关中惊醒，向西牛贺洲方向望去。
<br/>　　就连刚刚结束闭关状态的元始亦被这动静给转移了注意力，他抬头向着西方望去，顿时便看见西牛贺洲的灵山上，金色佛光光芒万丈，普照半个西牛贺洲，道音缭绕，无尽禅音响起，像是无数人在念经般，声音越来越大。
<br/>　　一株巨大无比的菩提树冲霄而起，绿意盎然，枝繁叶茂，金色佛光缭绕其间，擎天而立，镇压虚空，好似整个西牛贺洲的苍穹都要被这枝干支撑起来一般，巨树周围有无尽仙气霞光缭绕，瑞气祥鸣，无比神圣，给人一种宁静的感觉。
<br/>　　“嗯？佛教就要成立了吗？”
<br/>　　元始一眼便知道发生了何事，西方二位圣人要弃道立佛，开创西方佛教，证得无上佛道。
<br/>　　佛教的创立，代表着西方二圣独立于道教之外，自成一道，同时也代表着佛道之争的开端。
