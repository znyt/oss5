
<br/>　　<br/>　　    王惊蛰从厢房里出来的时候，冯天良正站在门口搓着手焦急的等待着，双方和谈后最紧张的莫过于他了，看见两方人相谈甚欢，语态平和的出来，冯天良顿时松了口气，事态看起来还是很美好的。<br/>　　<br/>　　    “王先生，有什么吩咐你尽管说，天良肯定不会推辞”冯天良诚恳又无比期望的说道。<br/>　　<br/>　　    一个成功的商人，这个人无疑也是非常精明的，坦白的说这时候王惊蛰告诉冯天良，你送我千八百万的他肯定都不带有一点犹豫的，甚至让他把这四合院送给自己，冯天良也能愿意。<br/>　　<br/>　　    能用一些身外之物结交到王惊蛰的话，从今以后两人明摆着就有了连接在一起的桥梁，对方稍微指点下冯天良他肯定都获益匪浅，因为有些东西你用钱来说是没法衡量的，就像易天逸之前就觉得，冯天良要是能交下王惊蛰的话，他都不用担心富不过三代这种事了。<br/>　　<br/>　　    “当年你父亲亲口答应过，我们来收刀的时候，但凡列出什么代价，你们都能一口应允·······”<br/>　　<br/>　　    冯天良直接点头说道：“我母亲说过，是有这么回事”<br/>　　<br/>　　    王惊蛰笑了：“我要是要你的万贯家财呢？”<br/>　　<br/>　　    冯天良当即就愣了，略微寻思了一下后，点头说道：“没问题，大不了我从头再来吧”<br/>　　<br/>　　    不管冯天良是真愿意还是假敷衍，但态度必须得端正，他现在真是怕了王惊蛰了。<br/>　　<br/>　　    大师爷和二师爷都笑了，冯天良可能不太了解，他们太知道了，王惊蛰是不可能要钱的，钱这东西对他们来说就是业障，得到的越多麻烦也就越多，肯定受不起的。<br/>　　<br/>　　    所以说嘛，世道上那些算卦，占卜的人一张口就说多少钱能破你的血光之灾，能让你财源广进什么的那都是扯淡，他们能有发财之道的话，还至于去摆摊算卦么，福布斯排行榜上，算命先生搞不好都得占一半了。<br/>　　<br/>　　    “我稍后给你个地址，在关中一带，你或者让人尽快启程赶过去，那里有户老宅，看门的是一位老人家，他看见我的字之后就知道是谁让你过去的了，你要做的也很简单，让人把那户老宅修缮一下，但有一点要记住了，就是老宅的原貌要保持，不能有太大的变动，翻新的过程也要小心一点，里面的东西绝对不能碰触到，能做到么？”<br/>　　<br/>　　    冯天良对于王惊蛰奇怪的要求有点迷糊了，他公司旗下有好几只施工队伍，随便派出哪一支来都能把这个小活给接下来了。<br/>　　<br/>　　    “就只有这个？”冯天良试探着问道。<br/>　　<br/>　　    “你能做好就行了”王惊蛰也没解释，一语带过了。<br/>　　<br/>　　    冯天良点头说道：“肯定不会让您失望，没问题的，那我这······”<br/>　　<br/>　　    冯天良一开口，后面的几人都顿时看向了王惊蛰，他的意思是我答应你了，但我身上这九星绝煞可得怎么破掉？<br/>　　<br/>　　    布局是门手段，同样的破局也是如此，人都说解铃还须系铃人，但有的时候这铃系的太死了，解开的话也未必容易。<br/>　　<br/>　　    王惊蛰也不搭话，迈步就走出了四合院，来到两扇朱红色的大门前，几人全都跟了出去后，就看见王惊蛰一脚踩在右边的那头镇宅狮子上然后腾空而起，跃到冯宅那块牌匾前方的时候，一巴掌就拍在了上面。<br/>　　<br/>　　    “嗖”被插在冯宅两字中间的菜刀应声而出，王惊蛰伸手就接了过来，当他“噗通”一声两脚落地的时候，冯天良人猛地就哆嗦了一下，头顶上萦绕的霉运瞬间就散开了。<br/>　　<br/>　　    易天逸豁然呆愣，不可置信的说道：“这就行了？”<br/>　　<br/>　　    “我拔刀能行，你拔就不行了”王惊蛰将菜刀收到包中说道。<br/>　　<br/>　　    易天逸一脸懵懂，还想再问的时候大师爷伸手拦了他一下摇了摇头，轻声说道：“懂点规矩，人家看家的本事是你随便问的么？”<br/>　　<br/>　　    易天逸脸一红嗯了一声，易苏苏撇嘴说道：“装腔作势！”<br/>　　<br/>　　    “记住你们先答应我的第一件事了吧？”王惊蛰问道。<br/>　　<br/>　　    “记得，到时候我亲自为你走一趟”大师爷抬头说道。<br/>　　<br/>　　    “好，你去份量还够用，那就回见吧”王惊蛰摆了摆手，招呼丁武和林羡鱼出来，三人就往胡同外面走去。<br/>　　<br/>　　    他们人影消失之后，大师爷就跟易天逸和易苏苏说道：“你们绕着四合院走一圈，去看下这里什么风水布局”<br/>　　<br/>　　    二师爷和大师爷两人背着手，同时仰头看着门上的牌匾，半晌过后才回过神来，冲着冯天良问道：“你家这个四合院是什么时候到手的，应该不是祖传下来的吧？”<br/>　　<br/>　　    “那到不是，是我父亲买下来的”冯天良回忆着说道：“我父亲接了二八五厂以后家里条件渐渐的就好起来了，也有了一些积蓄，之前听我妈说我父亲信了一个算命先生的话，交代他等家中条件稍微好转就购置一处四合院，最好还是在东城一带，我现在想想，让我爸买这栋院子的人应该就是当初那位送他菜刀的先生了”<br/>　　<br/>　　    大师爷和二师爷听完之后果不其然的点了点头，也没再解释，等了一会易天逸和易苏苏转了两圈回来了。<br/>　　<br/>　　    “这院子风水很好，是一处聚财聚运的地方······”<br/>　　<br/>　　    大师爷仰头感叹了一句：“赊刀人光这一手，可就拉我们相字脉太远了，我们做风水局风水阵，一局是一局，一阵是一阵，而赊刀人手下却是局局相扣阵阵相连着，真是拍马也赶不上啊”<br/>　　<br/>　　    易苏苏诧异的问道：“大师爷您是看出什么来了？”<br/>　　<br/>　　    “当年赊刀人点拨冯玉林的时候，是看出他运道要在哪一年起来了，所以冯玉林接了厂子以后家势就开始转好了，等好起来的这一年，冯玉林应该就在这时买了这栋四合院在手，那时候京城的四合院的年代比较久远了，而且建造的时候多数都请人看过，所以风水脉络都是不错的，加上京城里的皇气和贵气都很重，多少都影响了这栋院子的风水，聚财也聚运，所以冯玉林买了这栋院子后冯家的气运就和四合院连到了一起，自然就能发家致富了”大师爷又问冯天良道：“你家中人应该叮嘱过你，不管什么时候都不能卖了这院子吧？”<br/>　　<br/>　　    “上次给我母亲打电话的时候，她确实交代过，我父亲和她讲的，无论如何都不能卖了这栋四合院”其实冯天良之前也不清楚这点，他找人看着院子是因为京城四合院现在都很稀缺，有价无市，他完全是当固定投资来看待的，只要他没家破人亡，这四合院就是放一百年也不一定会卖。<br/>　　<br/>　　    易苏苏不解的指着冯天良问道：“他身上的那个绝煞局跟这有什么关系？”<br/>　　<br/>　　    二师爷指着头顶上的牌匾说道：“这块匾就是四合院的阵眼，王惊蛰一刀插在上面就是断了冯家的运道，刀刃压着镇宅石狮，他还能好得起来么？再加上冯天良的电话，生辰八字还有车牌号码跟你们之前去的柳河大街正好冲煞了，他不倒霉都怪了······”<br/>　　<br/>　　    易天逸恍然说道：“换个人是拔不出那把菜刀的，除非破了牌匾，牌匾碎了那就是四合院的风水也被破了，所以换个人的话就还是没用了”<br/>　　<br/>　　    大师爷叹了口气：“这刀，也不是那么容易能拿的啊！”
