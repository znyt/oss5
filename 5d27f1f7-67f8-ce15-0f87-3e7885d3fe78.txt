　　第293章
<br/>　　以赤檑为首的西颜部灰头土脸被赶出落玉寨，程大雷远远看着这一幕，就见樊梨花也走出聚义厅，气色不是很好。
<br/>　　“寨主，你便不答应，又何必说这么重的话，西颜部要是一怒之下杀过来，咱们该怎么是好！”阴万良。
<br/>　　“打过来就打过来，落玉寨易守难攻，什么时候被打破过。”樊梨花表情十分不屑。
<br/>　　“糊涂，糊涂。”
<br/>　　一个老者的声音忽然响起，樊梨花目光看过去，只见一个黑袍老人拄着拐杖过来。
<br/>　　“阴二叔。”樊梨花急忙迎过去，这老者名叫阴重，是阴万良的父亲，也是和樊梨花的父亲一起打天下的老兄弟。
<br/>　　阴重手中的拐杖敲着地：“那西颜部兵强马壮，既然他们看得上你，你嫁过去有何不好。除了戎族，帝国那肯有人要你？”
<br/>　　……
<br/>　　乱哄哄的声音，其实在程大雷也听不太真切。不过听着周围人闲聊天，程大雷倒也明白了是怎么回事。
<br/>　　西颜部的首领名叫乌茨木，他们活跃在琴川关外的草原上。而落玉寨的地理位置也算得上比较重要，如果两家能合一家，就算得上强强联手了，毕竟，樊梨花现在也是大龄未婚女青年。
<br/>　　可看樊梨花的意思，是不太看得上西颜部的。这才对嘛，毕竟，她是自己的人，要是被人撬走了那还了得。
<br/>　　不过，自己还是想办法早点脱身吧，再被关在这里，还不知事态会怎么发展。
<br/>　　当天夜里，程大雷又在撬钉子，冷不防樊梨花又一次出现在木笼外。慌得程大雷一惊，手中的马掌差点被发现。
<br/>　　“长夜漫漫，我以为只有我自己睡不着，原来樊家姑娘也睡不着……”
<br/>　　程大雷用嬉皮笑脸掩饰心中的惊慌，猛抬头，突然发现樊梨花今天看自己的眼神有些不对。神色变得颓废，和曾经趾高气扬欠揍的表情完全不同。
<br/>　　见到她手中的皮鞭，程大雷猛地一哆嗦，下意识往后退了退。
<br/>　　“嘿，狗贼，我真的好看么，还是你拿话奚落我。”
<br/>　　“姑娘羞花闭月之容，倾国倾城之貌，自然是千真万确。”程大雷以手指天。
<br/>　　“呔，狗贼，你又笑话我！”
<br/>　　樊梨花的皮鞭抽了过来。
<br/>　　“嘿，又来。”程大雷一把将鞭梢捏在手中：“我上次没有防备，还能次次被你打中。”
<br/>　　樊梨花想要将鞭子抽出来，却挣不过程大雷，程大雷笑盈盈的看着她：“姑娘花容月貌，怒不得啊？”
<br/>　　“狗贼！”
<br/>　　樊梨花啐了一口，忽地松手，口中叹了口气。
<br/>　　如此长吁短叹，倒是令程大雷怔了怔，不知这小姑娘有什么心事。
<br/>　　“我知道我生得丑，你也不用一直拿这一点嘲笑我。”
<br/>　　“嘲笑？”程大雷一愣：“呃，你是不是对自己的相貌有什么误解？”
<br/>　　“怎么，难道我不丑么？”
<br/>　　“……”
<br/>　　程大雷打量着樊梨花，好端端一个姑娘，腰细腿长声娇，妥妥的超模身材，这个丑字究竟从何说起。
<br/>　　如今帝国推崇的美人，还是纤细形，讲究瓜子脸，柳条腰，风一吹就能倒的那种，就像柳芷、苏樱。再加上这个平均身高普遍不高，樊梨花立在那里，要比很多男人高上一头，难免就容易让人升起高不可攀的感觉。
<br/>　　樊梨花今年已经26岁，并不是程大雷所理解的大龄未婚女青年，在这个时代，已直接可归类于没人要的那种。
<br/>　　虽然樊梨花是个山贼头头，但风言风语传入耳中，心里多多少少还是在意的。
<br/>　　程大雷琢磨了一会儿，才琢磨明白怎么回事，暗骂封建社会的无良，把樊梨花这样一个美女，归类到没人要的那种。
<br/>　　这……是自己的机会呐。
<br/>　　程大雷深吸一口气：“樊家姑娘，相貌是父母给的，你不用自卑，我不嫌弃你的相貌，更看重你的美好心灵，不如你放我出去，我们现在就去双宿双飞……”
<br/>　　“抱歉，我没有美好心灵。”
<br/>　　樊梨花长身而起，从木笼前离开。程大雷心情郁郁，继续撬钉子。
<br/>　　樊梨花回到房间，独自喝了杯茶，郁郁坐了一会儿，方才上床睡觉。
<br/>　　程大雷默默待在木笼里撬钉子，心情不是特别好，不过撬了这么多天，已经有了进展，快撬开一颗钉子了。等我回去了，一定好好收拾那帮小子，我被关这么多天，都想不到法子来救我。
<br/>　　五更天时分，突然一阵马蹄声疾，从山谷外，冲进来一队骑兵，正是去而复返的赤檑一伙人。
<br/>　　他们怎么又回来了……程大雷皱起眉头：来者不善呐。
<br/>　　樊梨花也听到外面动静，提着刀冲了出来，只见赤檑五十余人闯进山谷，黑压压的一片。
<br/>　　落玉寨易守难攻，山谷每日都有人把守，这些人为何能不吭不响闯进来。
<br/>　　“樊寨主，敬酒不吃吃罚酒，你非要我把你抢去嘛，不过在我们西颜部抢回去的女人一样疼的。”赤檑在马上大笑。
<br/>　　突然到来的变故，出乎樊梨花的预料，但她倒也不怯：“口气这么大，不怕闪了舌头，弟兄们，好好招待这帮贼人。”
<br/>　　樊梨花说罢，见只有几十人围了上来，而其他人则置若罔闻。
<br/>　　“寨主，你嫁到西颜部，对我们山寨也不是没有好处。”阴万良。
<br/>　　樊梨花一惊，随即明白了，是啦，如果没有内贼，戎族怎么可能轻而易举的闯入山贼。
<br/>　　“你们和戎族勾结，敢背叛我？”樊梨花秀眉倒竖。
<br/>　　“不是背叛，是我做主的。”阴重拄着拐杖出现，大声道：“从老寨主死后，咱们山寨的日子就过不下去了，西颜部肯要你，是你的福气，也是我们山寨的福气，难道因为你一个人，牺牲山寨这么多弟兄。”
<br/>　　“呵，原来有了二心。”樊梨花冷笑一声：“除掉我，这山寨岂不就是被你们父子做主。可任你们机关算尽，这落玉寨也是姓樊。”
<br/>　　樊梨花手中握刀，她有一身本事，此刻毫不畏惧。
<br/>　　正要提刀上马时，忽然腹中一痛，身体险些从马上掉下来。
<br/>　　昨夜那杯茶……有毒！
