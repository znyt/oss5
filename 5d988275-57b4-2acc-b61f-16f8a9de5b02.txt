　　而此时的东北，正处在一年中最冷的季节。
<br/>　　黑水河边，军营已经移走，黑水部的族人也再次恢复了以前的生活，不过，因为半年前的战争，族中的人少了许多。
<br/>　　不过，对于黑水部的人来说，他们并不恨那些穿着黑衣黑甲的大唐战士。
<br/>　　因为每一个靺鞨人都有杀人和被杀的觉悟，他们出去抢东西的那一刻起，便已经知道，很有可能自己会再也无法回到家乡。
<br/>　　可是没有办法，当人穷到连饭都吃不饱的时候，生命就开始变的不在重要，不得不说这就是人性的悲哀。
<br/>　　……
<br/>　　“玉儿，你好了没有，再不出来，我们可要出发了。”黑水部落中，钦陵带着十来个亲卫，等在黑水部族长家的院子外面。
<br/>　　黑水部族长在浑河边一战受了重伤，但好在命大，被黑齿常之他们给救了。
<br/>　　后来大祚荣投降，黑水部族长在伤养的差不多之后，便被送回了族中，现在正在家中静养。
<br/>　　“来了，来了，你嚷嚷什么啊，生怕别人不知道你来了是不是！”玉儿的声音从院子里传出来，尽显东北姑娘的彪悍。
<br/>　　“爹，我跟钦陵出去了啊，你在家里好好养伤，晚上我们就回来了。”对着院子外的钦陵喊过一嗓子之后，玉子又转头对自家老头子说道。
<br/>　　“去吧，路上小心些。”黑水部族长身上盖着一张毯子，那是行军毯，来自于钦陵的军营。
<br/>　　“嗯，那我走啦。”玉儿露出一个大大的笑脸，再次叮嘱了家中的仆役要记得按时给老爹煎药便出了院子。
<br/>　　“怎么样？你爹没再教训你吧？”等在外面的钦陵见玉儿出来了，迎上去小声问道。
<br/>　　“你说呢？”玉儿横了他一眼，电的钦陵全身发麻。
<br/>　　“那咱们出发吧，这一次可是大动作。”钦陵好不容易回过神，看了眼院子，然后便拉着玉儿上了战马，带着亲卫一同向远处行去。
<br/>　　……
<br/>　　黑水河上，五千黑甲军士站在已经冰封的河面上，每个人彼此间隔开一段距离。
<br/>　　玉儿跟着钦陵来到河边的时候正好看到这样的一幕，吓的她脸色都变了数变，有些紧张的对钦陵问道：“你，你竟然调动军队？”
<br/>　　“这有什么，太子殿下不是说过，要加强军地联系么。”钦陵笑着摇摇头，转对身边的亲卫说道：“发信号吧，让他们开始。”
<br/>　　“是！”亲卫目不斜视，秉承不该看的不看，不该听的不听之原则，拨转马头离开。
<br/>　　……
<br/>　　片刻之后，一道绿色的信号烟升起。
<br/>　　河面上正在列队的那些黑甲军士在看到信号烟之后便动了起来。
<br/>　　只见这帮家伙在冰面上努力的刨着，时间不大便已经刨了一个差不多有人头大小的冰坑出来，接着又每人从身上拿了一颗铁菠萝出来，放进坑里。
<br/>　　一、二、三，点火，撒丫子就跑。
<br/>　　……
<br/>　　数千颗铁菠萝同时爆炸的壮观让玉儿目眩神迷，同样也让人震惊于钦陵的笔之大。
<br/>　　但让玉儿不解的是，钦陵到底想要干什么？难道只是想让自己看看他如何把冰面炸开？
<br/>　　但是很显然，钦陵的目的并不是把冰面炸开。
<br/>　　只见那冰被震碎，露出下面的河水之后，那些大头兵有每人拿出了一个铁菠萝，点燃之后随手丢进了水里。
<br/>　　这一次的爆炸声音有些闷，除了让水面上鼓起无数的大包之外再也没有什么特别的。
<br/>　　玉儿完全被钦陵的这一套神操作弄晕了，眼睛瞪的大大的：“钦陵，你这是在干什么啊？那么多的……。”
<br/>　　“嘘……，你看！”不等玉儿把话说完，钦陵便竖起食指，做了个禁声的手势，然后又指了指河面。
<br/>　　看什么？有什么好看的。
<br/>　　顺着钦陵手指的方向，玉儿扭过头。
<br/>　　瞬间，这位靺鞨姑娘被眼前看到的一切震的一句话都说不出来，大张着小嘴，口水流出来了都没有一丝感觉。
<br/>　　只见那原本还满是碎冰的河面上，此时已经开始有一条条尺许长的大鱼浮上来，时间不大便已经覆盖住了整条河面。
<br/>　　“这，这……”玉儿已经不知道应该表示自己的震惊，好多的鱼啊，这么多鱼足够所有族人吃到明年夏天吧？
<br/>　　而就在此时，钦陵开口了，带着一丝怀念说道：“以前在长安的时候，太子殿下说东北是一个好地方，棒打狍子瓢舀鱼，当时我还不信，现在看来还是我无知了。”
<br/>　　“棒打狍子瓢舀鱼，说的真好！”玉儿呆呆看着那一群大头兵在经过短暂的愣神之后，冲向河水，用网子一网一网的往岸上捞鱼，口中喃喃说道。
<br/>　　“是啊，现在可不就是在舀鱼么。”钦陵附和道。
<br/>　　……
<br/>　　良久之后，玉儿终于从震惊中回过神来，看着钦陵的眼中满是异样的神彩：“你，你是怎么想到用炸药来炸鱼的？这样真的好厉害！以前我族都是在冰上戳一个洞，然后把网塞进去，每次捕获的鱼都不会太多，如果早知道有这样的办法……”。
<br/>　　说了一半，玉儿又顿住了，因为就算她早知道这样的办法也没用，没有炸药，没有那种铁菠萝，一切都是扯蛋。
<br/>　　不过钦陵却并没有注意到这些，他只是苦笑了一下，然后叹了口气说道：“同样是太子说的，在长安的时候我们没少聊起东北的事情，太子说了很多，只是以前我都给忘了，现在才想起来而已。”
<br/>　　“大唐太子厉害！”玉儿只是一个小姑娘，见钦陵总是‘太子说’，心中不禁也开始佩服起那个身在长安，从来没有见过面的人来。
<br/>　　“是么？”一旁听到玉儿称赞李承乾厉害的钦陵心中微微有些酸酸的反问道。
<br/>　　“当然啦！不过……”玉儿微微侧过头，俏皮的看了钦陵一眼：“不过钦陵大哥更厉害，竟然可以一次捕到这么多鱼！”
<br/>　　“呵呵……”虽然明知道玉儿说的是假的，但钦陵还是笑了起来，随后对玉儿说道：“好啦，现在鱼已经捕好了，美丽的黑水部圣女玉儿小姐应该回去展示神迹啦！”
<br/>　　“神迹？”玉儿看了看地上堆着的数不清的鱼，又看了看钦陵：“这，这些，这些都是给我的？真的？”
