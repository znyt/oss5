　　叶少阳侧身避过，起身说道：“那你听好了，待会他们来了，你完全按我说的做，只有这样才能帮他们超度，同时也能保住你儿子的性命。记住了！”
<br/>　　刘四满口答应下来。
<br/>　　叶少阳抬头看着两棵槐树，纵身爬到一棵树上，折断了一根树枝下来，来到三个婴儿的尸体面前，俯身说道：“尘归尘，土归土，你们已成亡灵，今生休矣，再世有凭。要这皮囊也没用了，没这皮囊，才好往生。”
<br/>　　说完，抓起木棍，用断口对着一具尸体的胸口，用力插了进去，有血喷出来，却不是普通的血，而是绿色的尸血。
<br/>　　“啊！”刘四媳妇发出尖叫，转过脸去，不敢看这残忍的一幕。
<br/>　　叶少阳一口气把三具尸体都串在了树枝上，然后在地上用硫磺生火，边上叉上树杈，把三具尸体架在树杈上烤了起来。
<br/>　　张道长吩咐手下两个弟子，去附近拾柴，放在火堆里。
<br/>　　火焰将三具尸体烧的皮开肉绽，绿色的尸水不断流出，散发着一股令人难以忍受的腥臭味，血被烤干的过程中，还不断冒出黑色的尸气。叶少阳让大家都站到上风口去，免得吸收尸气，对身体不好。
<br/>　　连张道长也躲开了，定睛看去，叶少阳就站在尸体身下，化着符水，对待尸气，完全没有躲避，心中更是震惊，他哪里知道，叶少阳是先天灵体，就算法力尽失，对于这些一般性邪气还是有免疫力的，根本不在乎。
<br/>　　叶少阳化好了符水，直接浇在了三具尸体的身上，加速着尸气的分化。尸气会发之后，就剩下尸油了，不断渗透出来，抵抗着火焰。
<br/>　　叶少阳知道，这些尸油根本抵挡不了太久。于是对张道长使了个眼色，让他和众弟子归位。
<br/>　　那三个在外游荡的恶鬼，感知到他们肉身被毁，肯定会前来营救，叶少阳估计他们要到了。
<br/>　　“刘老叔，大嫂子，你们远点儿站，到边上去呆着就行。刘四，你记住我吩咐的了？”
<br/>　　刘四点点头，紧张地头上都是汗。
<br/>　　叶少阳让他在自己面前做好，然后贴了一张灵符在他后脑勺上，说道：“一会照我说的做，记住了，那三个鬼魂一定会乞求你放过他们，但是你要记住，你是为他们好，一定要斩断魂之羁绊，这样他们才能安心去投胎！”
<br/>　　刘四刚在地上坐下不久，忽的一股阴风从北面吹来，叶少阳转头看去，有一团淡淡的乌云，好像一股没有散去的烟雾，从北方飘了过来。
<br/>　　来了啊。叶少阳深吸了一口气，微微一笑。
<br/>　　这是鬼魂用修为所化的一团阴云，用来遮挡阳光，好让他们在白天可以从容作法。
<br/>　　鬼魂的修为不同，用修为所化的阴云的浓度也不同，三个恶鬼一起施法，布置出这样一团阴云，已经算是不容易了。
<br/>　　如果是鬼首以上的修为，布置出的阴云，甚至可以遮盖住一小块天空，让方圆几里之内都暗如黑夜。
<br/>　　张道长也看到了那团阴云，面色立刻凝重起来，吩咐众位弟子各就各位，一起开始结印念咒。
<br/>　　阴云飘来，淡淡的一层，遮盖在了宅院的上空，好像一块薄云，三道小小的人影，从云层上空翻下来，正是三个小鬼。
<br/>　　三个小鬼眼看着自己的肉身即将被烧毁，怒上心头，什么也顾不上了，立刻附身朝叶少阳冲了过去。
<br/>　　“列阵！”张道长大喝一声，几个弟子一起作法，通过是天地三才阵的阵法之力，将灵力汇聚在一处，形成了一道结界，托住了三只小鬼。
<br/>　　“天地执法！”
<br/>　　叶少阳也喝了一声，咬破舌尖，喷了一口血在面前的阴阳镜上，阴阳镜贴在一个用朱砂在地上画成的八阵图中，正是这天地三才阵的阵眼所在。
<br/>　　舌尖血吐上去之后，叶少阳用手指蘸着，在上面写下了一个“山”字，咒语念过，四周朱砂笔在地上画的八阵图突然明亮起来，就地旋转，将灵力汇聚在阴阳镜上，形成一道光柱，正好打在了三只小鬼身上。
<br/>　　“啊……”三只小鬼一起发出痛苦的呻吟，被定在半空中，灵体左右挣扎，却无济于事。
<br/>　　天地三才阵的威力本来就不容小觑，再加上定阵的是九段光的法器阴阳镜，最重要的是，三个小鬼本是鬼尸，本尊被烧毁之后，魂体失去了灵力的源泉，这几方面因素加起来，造成了这必然的结局，三个小鬼根本就无力反驳。
<br/>　　叶少阳早就料到会是这样，口中念咒，在刘四的肩膀上拍了一下，贴在他后脑勺上的灵符燃烧起来，散发出的烟气，在刘四的头顶上空环绕旋转，刘四浑身一颤，猛然睁开眼睛，看到了被定在不远处半空中的三个小鬼。
<br/>　　在他自己跟那三个小鬼之间，有三条金色的纽带，好像是光一样，一头连接在自己胸前，另外一头连接在了三个小鬼的身上。
<br/>　　“父亲，父亲……”
<br/>　　三个小鬼也发现了他，对他伸出双手，大声哭泣着。
<br/>　　刘四当场就呆住了。这时，耳边想起叶少阳的声音：“刘四，快按我之前吩咐的做，斩断魂之羁绊！我这天地三才阵只能维持几分钟，几分钟后，他们尸身毁灭，再无留恋，必然会跑，只有这一次机会！”
<br/>　　只有这一次机会……
<br/>　　刘四深吸一口气，望着其中一道连接自己和小鬼的纽带，咬了咬牙，想象着自己手中有一把剪子，用力剪下去……
<br/>　　魂之羁绊，是一种只存在于彼此神识中的牵扯，只有运用神识的力量，才可以剪得断。
<br/>　　咔嚓！
<br/>　　刘四的心念之中仿佛听见了这样一个声音，再看前方，自己跟其中一个小鬼之间的纽带已经断裂了，还剩下两根……
<br/>　　刘四刚要运用神识去剪自己跟那个小姑娘之间的纽带，那小姑娘突然叫了一声：“爹！”
<br/>　　刘四浑身一颤，望着那小姑娘，猛然想起了她，那是自己第二个孩子，一个粉嘟嘟的小女孩。自己当时可没有什么重男轻女的思想，对这个小姑娘也是百般喜爱，可惜她也没能活过一周岁，最后不知道怎么就死了……
