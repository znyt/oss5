　　　　这条蛇精是什么实力，什么等级，涂小安现在还完全看不出来，附身在陈响的体内，他也没办法洞察扫描。
<br/>　　唯一可以肯定的就是它绝对没有三阶境界，否则也不用如此煞费苦心的附体陈响的身上。
<br/>　　要真三阶，也就没涂小安什么事情了，这个任务他是不可能完成的。
<br/>　　所以他现在要做的，就是先将蛇精给逼出来，显出原型。
<br/>　　“你不出来更好，那么就跟陈响一起灰飞烟灭吧”
<br/>　　金银蛇周身电光熠熠，愈演愈烈，整个套房内雷声大作，电闪雷鸣声不绝于耳。
<br/>　　赫然，整个套房仿佛要被雷电撕裂，异象迸发。
<br/>　　“本仙出来，非活吞了你不可”
<br/>　　如此异象下，头顶有银蛇蹿动，忽闪忽闪，蛇精心虚了，躲在人的体内，只有挨打的份。
<br/>　　猛然间，陈响的身体发光发亮起来，有一股虚影自头顶骤射而出，眨眼的功夫，陈响软绵绵的倒地不起。
<br/>　　昂...
<br/>　　霎时间，只闻一声舒展的咆哮声倏然而出，紧接着一股窒息感侵袭，整个套房跟着震动起来，就跟地震来了一般可怕。
<br/>　　在放眼望去，套房内多出了一条蛇的影子，它昂首而立，青黄赤黑，一双蛇眸灼灼发光，正死死的盯着金银蛇，浑身散发出来的蛇类气息铺天盖地，如履薄冰。
<br/>　　这个套房内的温度已经直逼零下了。
<br/>　　总算出来了。
<br/>　　涂小安松了一口气，其实他就是吓唬吓唬蛇精，他不可能真的将陈响给劈的粉身碎骨，那岂不是滥杀无辜了。
<br/>　　不过装出毫不在意人命的样子，让蛇精就范，不要以为自己有恃无恐。
<br/>　　“给我洞察”
<br/>　　“叮...觉醒异兽修蛇，等级二十七级，蛇中霸主，掌控水系能力”
<br/>　　系统扫描后，蛇精的数据资料第一时间出现在了涂小安的脑海中。
<br/>　　异兽？修蛇？
<br/>　　涂小安顿时诧异不已，表情不自觉的难看起来，说修蛇或许很多人不知道，但它还有另一个名字，叫巴蛇。
<br/>　　山海经：“西南有巴国，又有朱卷之国，有黑蛇，青首，食象。
<br/>　　说的就是巴蛇。
<br/>　　此蛇可谓大名鼎鼎，活在传说神话中的异兽，涂小安没想到现实世界中还真有这种蛇种的存在。
<br/>　　据说此蛇体长大到能够吞下一头大象，之所以叫修蛇，因为这种蛇种，天生就懂得修炼，天赋异禀，血统不凡，成长无上限。
<br/>　　涂小安不禁苦笑了起来，这下麻烦了，好像惹出了一个BOSS的存在，碰到的蛇种一条比一条要厉害。
<br/>　　区区蛇胎，还将传说中的蛇种都给引出来了，等级二十七级，拥有掌控水性能力。
<br/>　　别的都不要多说，光等级就足足高于涂小安这条金银蛇十级了。
<br/>　　现身的蛇精给涂小安一种山一般的压力，压的他有种窒息的感觉，感觉这套房都要承受不住的崩塌了。
<br/>　　涂小安对于这条蛇的身份仅仅始于传说形象，但他不知道的是，这条蛇精的来历并不比他白镇蛇王来的逊色。
<br/>　　这是一条外地蛇，在西南地区，雨水稀少，在当地也保持崇蛇风俗，有蛇仙传说，祭祀蛇仙求雨，可见倾盆大雨而下，蛇仙就乃修蛇，当地人为了它修了墓，建了庙，信徒众多。
<br/>　　看似完全是正蛇的形象，但此蛇性格乖张，行事作风全凭自己爱好。
<br/>　　有做善事，也有作恶，杀个人什么的，更是随心所欲。
<br/>　　至于它为什么来到白镇，就更不得而知了。
<br/>　　涂小安看着现身的蛇精修蛇，它全身条纹青黄赤黑，鳞片看似比钢铁还要坚硬，它的周身空气都在颤栗，畏惧在这条蛇的气息之下。
<br/>　　强，非常的强，气势澎湃。
<br/>　　“土蛇，你是不知道天有多高地有多厚，居然敢管上本仙的事情，还敢奴役本仙的孩子，大罗神仙都救不了你”
<br/>　　它发出的声音像凶猛野兽的嚎叫，让人浑身不寒而栗。
<br/>　　修蛇如今体长三米有余，但涂小安知道了此蛇的身份，清楚这绝不是它真正的体型，想必也具备大小神通。
<br/>　　黑蛇，青首，食象，可不是瞎说的。
<br/>　　“哼...谁死谁活，还未可知”
<br/>　　涂小安冷哼一声，震惊归震惊，但气势不能服软。
<br/>　　所谓强龙不压地头蛇，在他的地界上，管你什么上古异兽，来到白镇，都要给我安分守己的趴着。
<br/>　　哼的一声之后，只见金银蛇动了，电光火石间，朝着套房外的窗户射去。
<br/>　　蛇精见此，咆哮：“土蛇，想跑，本仙吃定你了”
<br/>　　它立刻随影随形的顺着窗户追了出去。
<br/>　　金银蛇在虚空上如履平地的快速飞行，随后涂小安敏锐的感觉到后面有道残暴的气息一直死死的锁定着他。
<br/>　　如黑暗中的幽灵，你无法甩开它，无论你跑到天涯海角，它都会找到你。
<br/>　　看来这蛇精也具备飞行能力，飞在漆黑的夜空之上。
<br/>　　传说中的异兽，果然名不虚传，涂小安见怪不怪了。
<br/>　　跑？不存在的。
<br/>　　只是区区不到百米的小套房怎么打架，况且还是民区，涂小安可不想引发恐慌。
<br/>　　他将蛇精往佘山的方向引去，到了那里，就不用顾忌许多了。
<br/>　　一路飞行中，涂小安想到了舒瑶的怪梦，她说总梦到一条大蛇钻入自己的被窝中，纠缠着她。
<br/>　　只见涂小安初闻，就觉得是觉醒蛇缠身。
<br/>　　现在看来这真的仅仅是一个怪梦。
<br/>　　蛇精附体陈响的身，想对她做什么都没问题。
<br/>　　她只所以在陈响不在家的时候，总是做恶梦，想必是因为条件反射，每一天跟蛇精附体的男朋友睡觉，等同跟蛇精睡觉，久而久之，身体感应下，一旦陈响不在了，她就会梦见一条大蛇钻进自己的被窝。
<br/>　　这叫梦境预言。
<br/>　　从古至今，人类的梦被视为一种警示，在很多国家的风俗文化当中都会有将梦当成是警示预言的一部分。
<br/>　　梦兆的大量应验更让人信服。
<br/>　　梦是预兆未来的，也有梦兆吉凶的卜辞。
<br/>　　舒瑶跟蛇精同睡，才照成怪梦预言，提醒她，现在每晚跟她睡觉的，其实就是一条大蛇。
<br/>　　蛇胎也由此怀上。
