　　好不容易，程处默那货把‘茄子’放下了，小唐连忙一把抓过来，瞪着他恨声骂道：“老子曰你大爷的，这东西能特么向你那么玩么，你想害死老子啊！”
<br/>　　“曰你二大爷，不就是个‘茄子’么。”程处默倒是不在乎自己的大爷有没有被日，反正他也没有大爷，不过就是看着小唐不爽，所以必须反击回去。
<br/>　　李承乾眼瞅着两人再闹下去不知道会弄出什么事儿来，连忙制止了他们，咳了一声说道：“你们两个差不多就行了，还有那么多人看着呢，也不觉得丢人！”
<br/>　　“是他先骂俺的！”程处默不依不饶的说道。
<br/>　　“行了，一个巴掌拍不响。”李承乾才没有心思给这两个纨绔断官司，翻了小程一眼，然后对唐善识说道：“这炮弹我觉得应该像弓箭那样，装上两人对平衡翼，否则你这东西打出去天知道会掉到什么地方。”
<br/>　　“不可能吧，难道你还想这东西跟箭一样准？”说到专业方面的知识，唐善识也顾不上与程处默吵了，抱着那炮弹有些不确定的说道：“这东西就像你说的，是用吊射的，就算是再准估计也控制不了方向。”
<br/>　　“我跟你说的根本就不是一个东西，我的意思是你这样的炮弹根本就没有办法击发，难道你准备靠它落地之后自己炸天？”
<br/>　　“为什么不能爆？”小唐还是没有理解李承乾的意思。
<br/>　　李承乾从地上抓起一块石头，向天上一丢，看着石头翻滚而下的样子，对小唐示意道：“因为它在天上会翻跟头，落地的时候并不一定是大头下向的，所以它前面的引信就成了废物，明白？”
<br/>　　隐约间小唐也觉查到了有些不大对头，看了看怀里抱着的炮弹，又看了看地上的石头。
<br/>　　“别看了，改吧，就是改个弹壳，容易的很。”李承乾拍了拍小唐有肩膀，安慰他道。
<br/>　　“那总得试试吧，好不容易搞也来五个。”小唐似乎有些不死心，又看了看怀里的炮弹，有些惋惜的说道。
<br/>　　“成，你是主管，你说了算，反正已经造出来了，打出试试威力也不错。”
<br/>　　“太好了。”唐善识见李承乾没有反击，立刻招来了等在一边的试炮员，将炮弹交给他：“好好打，打准点。”
<br/>　　“是。”试炮的家伙也是个愣头青，也不管这东西到底安不安全，从小唐手里接过那炮弹转身就走，末了还不忘将地上放着的四个一同拿走。
<br/>　　简单的准备工作过后，李承乾、小唐一行已经躲到了试验场的一房混凝土房子里，通过一个很小的方孔用望远镜看着外面的情况。
<br/>　　不管什么时候，长官的命都是最重的，李承乾虽然不排斥人人平等的观点，但谁还不喜欢有点特权呢，尤其是在大唐混了十多年的情况下，他已经习惯了这种安全第一的行为。
<br/>　　程处默这货跟他老子是一个德性了，虽然进了防炮的混凝土房，但嘴里却喋喋不休的埋怨着，说什么在外面看比较清楚之类，不过李承乾和小唐全当他是在排气，谁都没有理他。
<br/>　　……
<br/>　　小房间的外面，试炮手将炮弹好好的擦拭了一遍，就像对待自己的孩子一样，接着就看那家伙对着老天拜了拜，然后毫不犹豫的将炮弹塞进了炮筒。
<br/>　　“嗵”的一声，炮筒下的地面泛起一层薄薄的灰尘。
<br/>　　“什么情况？这就完了？”聚精会神盯着外面的小程木然的扭过头。
<br/>　　迫击炮的击发方式与正常的火炮不一样，发射的时候并没有那么大的声音，它的击发只是通过一部分火药的力量将炮弹送出膛，而不是直接炸开。
<br/>　　“看着！”小唐显然对程处默的无知感到了不满，斜了他一眼不满的说道。
<br/>　　不过对话似乎也就到这里为止了，因为就在远处大概五百步左右的地方，突然发出了一声惊天巨响，接着看到的便是泥土四溅飞舞的景象，显然，那炮弹打歪了，并没有砸到八百步之外的试验场。
<br/>　　但就算是这样，也让程处默张大的嘴巴，冷汗也开始在其头顶不断的冒了出来。
<br/>　　“你大爷的，现在知道刚刚你的行为有多危险了吧，如果你一个失手，咱们这些人就全都特么去见太上皇了。”唐善识借着炮手装填第二发炮弹的时间，拍了拍小程的肩膀说道。
<br/>　　“这东西，这东西真是太特么过瘾了！”程处默这个时候也顾不上和小唐拌嘴不，兴奋的抽了抽鼻子，对李承乾的说道：“我说高明，这东西啥时候装备部队？”
<br/>　　“暂时别想了，这些都是作为战略储备用的，也就是说，咱们有这东西，但暂时除了禁军，其他部队都不会装备。”李承乾的摊了摊手。
<br/>　　“啊？不装备部队？”程处默简直不知应该如何表达心中的‘震惊’。
<br/>　　是的，就是震惊，他无论如何也想不明白，为什么这么好的东西要装备给禁军那些只知道走正步的怂包，而正规部队却依旧要使用以前的老式火炮。
<br/>　　不过，李承乾却没有给他任何解释，因为就在刚才，第二发试射的炮弹竟然没响，就那掉在了地上。
<br/>　　小唐有些尴尬的看了李承乾一眼，似乎有些理解了为什么炮弹要装尾翼。
<br/>　　毕竟迫击炮炮弹的装药量与正常的炮弹是不能比的，正常炮弹因为其装药量大，所以其初速度也大，在出膛之后便不会轻易的改变自己的轨迹。
<br/>　　可是迫击炮却不行，较少的装药量便其根本无法获取更高的初速，再加上其吊射的攻击方式，注定了它无法在落地的时候一定会以弹头着地，所以加装尾翼便成了必然。
<br/>　　……
<br/>　　最终五发炮弹试射，其结果只有两发响了，另外三发不出意外的或是横着，或是倒着砸到了地上，根本没有触发前面的引信。
<br/>　　以百分之四十的成功率，赢得了回炉改造的重担。
<br/>　　程处默对于这样的结果嗤之以鼻，认为小唐刚刚就在是危言耸听，搞的唐善识好不尴尬，暗暗庆幸还好只有程处默一个人来了，若是再多来几个纨绔，估计以后这日子就不用过了。
