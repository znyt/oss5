　　昆仑神秘山谷之中，十二品青莲宝光纵横，先天灵气彭拜。
<br/>　　三清站在池塘边上，纠结着如何分配这十二品造化青莲为好。
<br/>　　突然，元始看了两人一眼，对着十二品青莲开口大声喝道：“白藕！”
<br/>　　顿时，十二品青莲之上，一白藕飞出，飘向元始，化作一三宝玉如意，落入元始手中。
<br/>　　老子通天看得目瞪口呆，灵宝还能分开用？
<br/>　　元始对着两人大声喝道：“二弟，三弟，此时不醒，更待何时！”
<br/>　　顿时，两人醒悟过来，看着玉如意，再看了看池塘之中的十二品青莲，若有所思。
<br/>　　老子上前一步，道了一声：“红花！”
<br/>　　顿时，十二品青莲之上一朵红花飘起，飞向老子，化作一拐杖，落入老子之手。
<br/>　　通天见此，心中一震，不由自主道了一声：“青莲叶！”
<br/>　　顿时，剩下的青莲叶化作一柄青色宝剑，剑气凌然，飞向通天。
<br/>　　“哈哈哈，好！既然你是青莲叶所化，自今日起，你就叫青萍剑了。”通天双手抚摸着手中宝剑，大笑道，他是剑修，最喜剑道，可惜一直没有合适的灵剑，如今有了这青萍剑正好合适。
<br/>　　“白藕红花青莲叶，三清原本为一家！”元始开口道。
<br/>　　两人见此，若有所思，似有所悟。
<br/>　　“余下的青莲，两朵九品青莲，便让与两位贤弟，三朵六品青莲，各自一朵，至于天一生水，三人均分，如何？”元始淡淡道。
<br/>　　“这……这样大兄不就吃亏了吗？”两人迟疑道。
<br/>　　“无妨，那件灵宝归我便好。”
<br/>　　元始伸手一抓，不知从何处抓来一面青色小旗，全身闪耀着青色光芒，之前三人都将注意力集中在十二品青莲之上，谁都没有注意到这面小旗。
<br/>　　“东方青莲宝光旗？！”老子瞳孔一缩，惊诧道。
<br/>　　“不错，就是先天五行旗之一的青莲宝光旗。先天五行旗为极品先天灵宝，每一件旗子都为上品先天灵宝，与九品青莲一样。”
<br/>　　老子若有所思，道：“如此甚好。”
<br/>　　通天亦称“大善”。
<br/>　　如此，三人分配好宝物后，就此出了山谷，向着三清出生的山谷而去。
<br/>　　回到山谷之中，三人宣布闭关炼化灵宝，而且这些年游历所得也需要闭关来沉淀，参悟大道，争取早日证道。
<br/>　　而元始却在五十年之后出关，留下信息之后，元始就再次出了山谷，出现在洪荒之中。
<br/>　　此次，他却是向着太古星空而去，《玄黄不灭》第五重星辰入神窍是时候该突破了。
<br/>　　若是再耽搁下去，以后星辰被妖族占去了，这对于元始来说，就有些麻烦了。当然，也仅仅只是麻烦而已。
<br/>　　元始嫌弃麻烦，但不代表着他惧怕麻烦，妖族又如何？吾元始想要干的事，你还阻止不了。
<br/>　　当然，能减少些麻烦，他也不会去做麻烦多的事。此时前往太古星空之中，仅仅只是为了日后少些麻烦而已。
<br/>　　星空之中，一片璀璨，星光烂漫，闪耀青天。更深处，则是一片幽暗深邃，仿佛一个滔天巨兽，张开大口，要将所有一切尽皆吞噬。
<br/>　　其实，星辰之中并没有什么生灵，星辰大多都是荒芜的，只有偶尔几个特殊的滔天巨兽在虚空之中行走，他们以吞噬星辰为生，体型巨大，一口之下，就能吞噬整个星球。
<br/>　　他们叫吞天兽，实力非凡，堪比准圣强者，有的甚至可以堪比混元强者。
<br/>　　不过，吞天兽十分稀少，稀少的简直快要灭绝了。而且，就算有，他们也只是生活在星空最深处，不敢出来。
<br/>　　元始踏入星空之中，看着幽暗深邃、浩瀚无垠的星辰大海，心情难免有些激动。
<br/>　　征服星辰大海，这是一个多么令人激动的目标。
<br/>　　他此次的目的是要找到星空之中最主要三百六十五颗祖星辰。
<br/>　　星辰何止亿万，但最为主要的星辰，永远是那三百六十五颗祖星辰，其中，尤其是太阳太阴两颗至尊星辰，最为重要，他们是盘古的两颗眼睛所化，照耀洪荒天地，滋养万物。
<br/>　　不敢耽搁，元始快速在这星辰大海之中飞行，一颗又一颗星辰在他脚下飞过，他在寻找那三百六十五颗祖星。
<br/>　　其实这很好找，因为这三百六十五颗星辰是这亿万星辰之中最为耀眼的几颗，其余的星辰，虽然也有特殊的，但都远没有这三百六十五颗璀璨。
<br/>　　很快，元始就找到了一颗星辰，这星辰火红无边，隔着很远，他便能感受到其中的温度，火光冲天，炙热无比。
<br/>　　这是一颗荧惑古星，也叫火星，对应着人体心之神窍。
<br/>　　一步踏出，来到火星之上，火星上火焰喷发，温度极高，就算是神铁在此处亦能被融化。但这点温度对于元始来说，并不算什么，他连太阳真火这样的神火都不怕，又何惧这样的凡火？
<br/>　　来到星辰最中央，元始盘腿坐下，感应着星辰本源，神识慢慢延伸，顺着星辰伸向星辰深处。在星辰的最深处，有一个特殊空间，穿过这层空间，里面有一个缩小无数倍的火红的小星辰，其内星辰浓郁，星光灿烂，为星辰本源。
<br/>　　元始用神识接引这些星辰本源之上的星光，使之顺着人体，进入到心之一窍之中。
<br/>　　星辰之力为极高的能量，极为霸道，若是没有极为强大的肉体，瞬间便会被星光冲破。只有强大的肉体，才能用星辰之力修炼。
<br/>　　元始的肉体自然不用多说，他极为顺利地接引到火星的星辰本源，顺着人体经脉，来到心之一窍。
<br/>　　“砰！砰！砰！”
<br/>　　星辰之力在心窍之壁上不断冲撞，三声之后，心窍之门大开，星辰本源冲入其中，充斥到整个心窍之中，心窍空间在不断地被扩展增大。
<br/>　　外面，火星之上星光瞬间变得光芒四射，本源之力被元始源源不断地接引过来，炼入到心窍之中。
<br/>　　心窍空间在缓缓增大，心窍变得通红无比，一股火红色洪流从心窍之中冲了出来，顺着人体经脉，流遍人体全身，在一点一点地淬炼着元始的肉体。
<br/>　　元始感受到无比的疼痛，但对于他来说，疼痛如今已经不能带给他任何危险了，有疼痛的感觉，斩了便是。但他并没有斩去这种感觉，而是用它来磨练自己的意志。
<br/>　　元始全身变得通红无比，星光在他身上散发，无数星辰之力涌入他的身体之中，增加着他的修为法力。
<br/>　　直至良久，元始才停了下来，星光内敛于身，火星本源之力退去，可见那神秘本源空间之内，火星本源星球变小了一些，火光也变得暗淡些许。显然，在刚才的修炼之中，火星本源损失了一些。
<br/>　　但元始并不担心这颗星球会毁灭，因为只要有足够的灵气，随着时间的流逝，这些星辰本源迟早还会回来的，没必要太过担心。
<br/>　　而灵气，这星空之中，并不缺少。
<br/>　　元始开始内视心窍，此时的心窍已经变得通红无比，就像是一颗火星在人体星空之中闪闪发光，照亮了人体内宇宙。
<br/>　　元始点了点头，极为满意。因为他感受到了自己的肉身有了一点增强。要知道，以他如今堪比半步混元的肉身，肉身竟然还能有增强，这就有些不可思议了。
<br/>　　这还是一颗星辰，要是三百六十五颗星辰全部修炼，那他的肉身又该有多强？想必已经突破混元了吧。
<br/>　　元始表示非常期待。
<br/>　　不再多言，元始向着下一颗星辰赶去。
<br/>(本章完)
