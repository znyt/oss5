　　第52章
<br/>　　“柳将军的为人我们大当家是分外佩服的，虽未曾谋面，却是仰慕已久。这次柳将军遭人冤枉，落得如此下场，听到后，我们大当家十分愤怒。”程大雷胡诌道。
<br/>　　柳芷点点头：“虽然是一个山贼，却也知道我父亲是被冤枉的，看来公道就在人心。”
<br/>　　“哼，一个山贼而已，又能做什么。”孟子云冷哼一声。
<br/>　　程大雷气得不行，山贼究竟怎么了，职业歧视要不要这么明显。
<br/>　　“大当家说了，柳将军的后人绝不能受此冤屈，所以特意派我进城打探一下情况，若有机会，便要救柳家上下的性命。”程大雷腾地站起来，说得慷慨激昂。
<br/>　　“你们打算怎么做？”柳芷被程大雷情绪感染，忍不住问出声。
<br/>　　“情况我已打探清楚，这就回山寨取钱，将其他人赎出，再来搭救柳家小姐。”
<br/>　　柳芷怔了怔，忽然泪珠滚落：“程大当家大恩大德，柳芷没齿难忘，若能救得我家老小性命，柳芷，柳芷……”
<br/>　　程大雷屏声静息，期待的看着柳芷，这就是要以身相许的节奏呐。
<br/>　　“……柳芷来生当牛做马，结草衔环以报。”
<br/>　　“……”程大雷。
<br/>　　“小姐，此人贼眉鼠眼，不可信他一面之词。”孟子云。
<br/>　　程大雷崩溃，合着你这浓眉大眼的就不会骗人是吧，在心底，程大雷又在心里狠狠记了一笔，暗中默念八个字：不是不报，时候未到。
<br/>　　“我的同伴就在楼下，我可以叫他上来，手中有了人质，你们自然不用怀疑。”
<br/>　　柳芷和孟子云对视一眼，几乎是同时点点头。
<br/>　　楼下，徐神机如坐针毡，照道理来说，程大雷不在，他岂不是如鱼得水，在身边姑娘身上大块朵颐。
<br/>　　其实不然，因为许茂一干人等死死盯着他，盯得程大雷如坐针毡，动一动都怕被大家的目光杀死。
<br/>　　而对徐神机的关注，自然因程大雷而来，有人时不时看向楼上，心中估摸着时间。
<br/>　　以前其他人进去柳芷房间，没有一盏茶就出来了，如果能多在房间内坐一会儿，便会沾沾自喜，可以吹好几天牛。
<br/>　　相比之下，程大雷进去的时间也太久了。
<br/>　　难道说，他第一次进去就得入罗帷，春风一度。
<br/>　　心中如此想着，心思难免已经往龌龊处想。
<br/>　　这个时候，程大雷出现在房间门口，冲下面喊了一嗓子。
<br/>　　“欧耶，你上来。”
<br/>　　徐神机如蒙大赦，赶紧颠颠的跑到楼上，嘴里埋怨道：“你是快活了，可是苦了我，不过不错，难得你吃肉，还让我喝点汤。”
<br/>　　“当然，我这个人向来仗义，风流快活也不忘了你，我风流够了，你也进来风流风流吧。”
<br/>　　程大雷揽住徐神机的肩膀走进房间，楼下一干人等想到房间里将会发生的事，哈喇子都快垂到地上。
<br/>　　徐神机最是激动，嘴里念叨着：没想到啊，没想到，没想到你这么仗义。
<br/>　　想入非非，心脏也火热起来，刚走进房间，迎面就是一把雪亮的钢刀，徐神机刚火热起来的心脏，就彻底凉了。
<br/>　　程大雷拍拍他的肩膀，道：“这位是柳将军手下孟子云，孟兄信不过我们，你带着他回一趟山寨，取钱过来赎人。”
<br/>　　“那你呢？”
<br/>　　“我在城中还有事，事情定下来后，你知道如何联系我。”
<br/>　　徐神机点头应承，而柳芷和孟子云却还是目光闪烁，显然是还信不过程大雷。不过事到如今，他们不信程大雷又有如何法子，程大雷已是他们唯一法子。
<br/>　　程大雷笑笑，双手一抱拳：“柳姑娘，请了，等离开这里，我等你当牛做马，结草衔环。”
<br/>　　柳芷还没反应过来，程大雷已经跳窗从后院走了，次日天将亮未亮之时，徐神机和孟子云也跳窗离开回燕楼。
<br/>　　程大雷溜出回燕楼后，就立刻去了苏家。
<br/>　　苏家的墙并不高，约莫只在两米左右。程大雷双手一攀便爬上墙头，顺着墙根轻飘飘滑下来。
<br/>　　躲在假山后观察周围的环境，夜色凄迷，倒是没有巡夜的家丁。这令程大雷松口气，如苏家这样的家族，一般都有护院家丁，如果被他们发现，凭自己这两下子恐怕会死得很干脆。
<br/>　　紧张又忐忑的心情，平白给程大雷一种刺激的心情。如西厢记中私自跳过粉皮儿墙和小姐约会的张生一般。
<br/>　　可苏樱的绣楼在哪里呢？
<br/>　　紧跟着，程大雷发现自己遇到一个问题，心中忍不住骂了一声。
<br/>　　苏家实在太大了。
<br/>　　不同于前世几十平米，几百平米的房子，也不是程大雷想象中一进两进的院子，而是一座院子套着一座院子，院子外面还有院子。
<br/>　　特么，知道你们地广人稀，也不要这么折腾吧。这么多院子，我怎么知道苏樱的闺房在哪里。
<br/>　　“呸，瞧她的样子，还把自己当大小姐呢！”
<br/>　　“老爷病得不轻，现在家里都是夫人说了算，谁还把她看在眼里……”
<br/>　　“哼，她再敢在我面前摆大小姐的样子，我就让她知道知道厉害！”
<br/>　　有两个仆役打扮的人从小路上经过，一路走一路议论着什么，表情显得很兴奋。
<br/>　　程大雷窝在假山后，悄悄听着，等二人走远，他向二人来的方向望了一眼，悄悄向那边去了。
<br/>　　苏樱就住在那个方向的某个院子中。
<br/>　　她最近的日子过得不好，或者说相当恶劣。她原先的绣楼被自己同父异母的妹妹占了，自己搬进这破败的院子。身边伺候的丫鬟仆役态度也越来越恶劣，唯有小蝶对自己还好些，可小蝶也在今天被要走了，说是二小姐身边不能没人照顾。
<br/>　　这种生活上的困难并不是不可克服，更难的事情马上就要到了。
<br/>　　她即将嫁给一个年近半百的老头子。
<br/>　　她独坐院中，对月长叹，身边是如水般稠密的黑暗，真的……挣不脱呐。
<br/>　　咚！
<br/>　　就在这个时候，一人从天而降，噗通砸在地上。苏樱吓了一跳，看到地上那人灰头土脸站起来，拍拍身上的土，强装作漫不经心的样子。
<br/>　　“漫漫长夜，我以为只有我睡不着，原来苏姑娘也睡不着啊。”
<br/>　　
<br/>(本章完)
