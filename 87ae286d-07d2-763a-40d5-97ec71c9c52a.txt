
常文书领王惊蛰去的地方在大兴郊区，一片私家园林中深藏着几栋欧式建筑，错落有致，雍容华丽，门前还停着不少的车子，刚一进来就闻到了一股纸醉金迷销金窟的味道。<br/>　　<br/>　　王惊蛰略微皱了下眉，他向来都对这种地方比较反感的，似乎看出他情绪有点变化，常文书笑着说道：“你别把自己活的太像个神仙了，有的时候也得落入凡尘看看，我带你来呢一是看个新鲜，再一个是你得认识个人，对付王子文这样的人我用点劲就能掐的他一点脾气都没有，但韩观山这人的身份有点特殊，毕竟他的部门是直接上达天庭的，我要是真豁出去了的话也能把他踩在脚底下，可恐怕得要借用家里的关系了，所以也挺麻烦的，不过我带你见的这人呢要是有他点头的话，那就容易多了”<br/>　　<br/>　　王惊蛰顿时“哦”了一声，问道：“体制内的？”<br/>　　<br/>　　“后备……”常文书意味深长的吐出了个词。<br/>　　<br/>　　车停到其中一栋建筑前，两人进到楼内，就有个穿着西装的青年过来迎接上了他俩，对方叫了声常先生后就弯腰伸手示意跟着过来，到了电梯门前进去，按着向负三层的按钮，等他们走出电梯的时候，王惊蛰顿时就感觉到一股混合着血腥和戾气的味道扑面而来，满耳都充斥着吵杂和吵闹的人声。<br/>　　<br/>　　这是个挺宽阔的地下场所，中间是个擂台，四周是一圈卡座还有几排座椅，擂台上面两个赤膊的男子正在厮杀在一起，周围的看客都脸红脖子粗的嘶吼着，看的是挺让人热血沸腾的。<br/>　　<br/>　　王惊蛰发现围观的人男女都有，大部分都是衣冠楚楚的，但脸上的表情跟他们的打扮可谓是大相径庭正好相反了，穿的是不错，可神情都是激动和狰狞的，甚至有不少人喊得嗓子都哑了，似乎潜藏在人性下的劣根，在这一刻彰显的一览无遗了。<br/>　　<br/>　　这是个地下搏斗场所，专供人赌博或者欣赏的，在国外很多地方都有，可在国内绝对很少见，因为开办这种地方风险太大，没有八面玲珑的关系，谁也不敢干。<br/>　　<br/>　　人类的发展史到现在都有几千年的历史了，但是自古以来不管社会如何的推进和进化，科技发展到了什么地步，人性中有一面都始终未曾消退过，有位哲学家就曾经斩钉截铁的说过这么一句话，赌是人类的天性，不管任何人任何的身份，他这一生中其实涉及到的赌，数都数不过来，最单纯的赌徒不过赌的是钱财而已，做生意的商人每一次买卖都是在赌博，手术室里的医生也是在赌手术会不会成功，想要升官的则是在赌自己应该站在哪个队伍。<br/>　　<br/>　　就拿最简单的放屁这件事来讲，你赌对了它就是个屁，要是赌错了那就是个悲剧了。<br/>　　<br/>　　王惊蛰和常文书来到了卡座上面的一个独立包厢，这种包厢周围一共不到十个，围墙是玻璃的，从外面看不见里面，私密性很严实。<br/>　　<br/>　　他俩进来的时候，里面沙发上坐着个大概三十岁上下的青年，上身白衬衫下身黑色西裤皮鞋，梳着三七分的发型，打扮上就是那种最普通的事业单位和公务员的装扮，十分不起眼，要说这人哪里能吸引点眼球的话，就是气势看起来有着远超同龄人的老练和沉稳。<br/>　　<br/>　　这青年旁边，还坐着个挺知性的女子，脸上化着淡妆，身穿薄款羽绒服，乍一看就属于不食人间烟火那一类。<br/>　　<br/>　　“约在这种地方见我，也亏你想的出来，要是被家里人知道我来了这，回去少不了得要被人训斥一顿了，都是书香门第以后你能不能换个文雅点的地方？”沙发上的白衬衫青年，端着茶杯直摇头的说道。<br/>　　<br/>　　常文书笑了，说道：“你也不是第一次过来，就别和我装紧了，你这人向来都喜欢剑走偏锋当我不知道呢？再说了，你现在不过就是个部委里的小科员，别搞得好像要上天了似的，及时行乐吧，再过几年你想乐都没机会了，趁着没被上纲上线呢把该找的乐子都找一遍，省的自己以后后悔”<br/>　　<br/>　　对方伸手点了点常文书，叹了口气说道：“千里之提毁于蚁穴，小事也容易发酵的”<br/>　　<br/>　　“得，下次我约你去稻田地里见面，咱俩一边耕地一边聊，你妥妥的会被扣上个楷模的帽子了”<br/>　　<br/>　　“滚蛋！”这青年笑骂了一句，然后眼神不经意的瞥了眼王惊蛰。<br/>　　<br/>　　常文书随意的坐在沙发上，冲着王惊蛰说道：“这是我范二哥，你跟着我这么叫就行了……二哥，王惊蛰，我一挺好的朋友”<br/>　　<br/>　　常文书就给两人点了下名字，其他的一概都没介绍，王惊蛰是没啥感觉，这位范二哥却知道，能在这种场合让常文书带人过来见他，这个看似平淡的青年，肯定是哪里有可圈可点之处的。<br/>　　<br/>　　随后，常文书和范二哥一直在喝茶闲聊，那知性女子一直在翻看着手机，既没和他们聊天也没看着台下的博弈，而王惊蛰坐在一旁没什么反应，眼睛就偶尔瞅着下面擂台上的厮打，他们这个包厢从上往下正好能俯瞰到台上两人交手的全貌，十分清晰。<br/>　　<br/>　　他俩聊了大概十几分钟后，台上那一对就分出了胜负然后下场了，没过多久又有两个拳手再次上台，一个主持人来到中间介绍了下双方的状况，这两位都是久经沙场的老将了，长得一身黝黑的是泰拳过来的高手，场次七十多胜只有五负，另外一个瞅着就有点寒碜了，长得干巴瘦不说身上好像还好几道伤没好，战绩是二十多胜一负。<br/>　　<br/>　　主持人在台上吧啦吧啦的介绍着，台下围观的人很多都在咬着耳朵窃窃私语。<br/>　　<br/>　　常文书忽然放下杯子，问道：“二哥，押几把？”<br/>　　<br/>　　“嗯，看完这一场就走吧”范二哥点了点头，常文书伸手按了下桌子上的铃，没过片刻就进来个经理模样的人，笑着道：“常先生，手痒了？”<br/>　　<br/>　　“唉，看了一会不下注也没啥激情啊”常文书回了一句，忽然问向王惊蛰道：“惊蛰，你看哪个人会赢啊？要不你也押一把……”
