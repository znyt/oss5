两个人相拥了一会儿，夏夕绾将他推开，“你过来，我给你处理一下手上的伤口。”
<br/>
<br/>        陆寒霆的手面被拉出了好几道血口，血口很深，夏夕绾心疼的整张小脸都拧起来，她拿澄亮的水眸气呼呼的瞪了他一眼，“陆先生，砸玻璃是不是显得你很帅，有本事下一次你砸墙啊，让我好好见识一下你能不能一拳将墙给砸出一个大窟窿！”
<br/>
<br/>        陆寒霆挑了一下英气的眉心，冷硬的俊脸上溢出了几分笑意。
<br/>
<br/>        夏夕绾将医药箱放了回去，“陆先生，你去洗澡吧，这里我来收拾。”
<br/>
<br/>        “你可以么？”
<br/>
<br/>        “当然可以，我上得厅堂，下得厨房！你单手洗澡没问题吧，刚包扎的这只手要抬起来，不要碰水。”夏夕绾交代道。
<br/>
<br/>        陆寒霆亲了一口她的脸蛋，突然覆在她耳畔低声问道，“刚才，有没有把你弄疼？”
<br/>
<br/>        夏夕绾一下子就明白他在问什么了，刚才他脱她裤子检查她是不是处，她一张小脸瞬间爆红，她扯了毛巾就丢在了他可恶的俊脸上，“陆先生，不要搞擦边，停止你的虎狼之词，我听不懂！”
<br/>
<br/>        陆寒霆伸手将毛巾扯下来，笑着进了沐浴间。
<br/>
<br/>        ……
<br/>
<br/>        夏夕绾将地上的碎玻璃给清理干净了，她进沐浴间洗手，看见他的衣服丢了一路，她迅速低腰将他的衣服给捡了起来。
<br/>
<br/>        这时她感觉有点冷，今天的气温不算低，这个沐浴间里打着明黄的灯光，更是温暖如春。
<br/>
<br/>        夏夕绾站直身感受了一下，她真的觉得冷，从骨子里感觉到的冷。
<br/>
<br/>        她小时候就浸泡各种药草，血液十分的珍贵，百毒不侵的，这是妈咪留给她最好的礼物，也是最好的爱，所以她从小就不怕冷，从来没有生病感冒过，冷的感觉对于她而言十分的陌生。
<br/>
<br/>        但是，现在的她真的感觉到了冷。
<br/>
<br/>        这微妙的身体变化让身为医者的她心生警觉，曼陀罗花毒，是世界最剧烈的毒，她仗着血液百毒不侵进行试毒，身体在那么一刻竟然也抵消不住毒性。
<br/>
<br/>        现在她竟然感觉到了冷，她不知道曼陀罗花毒是不是已经侵蚀了她珍贵的血液，让她从百毒不侵变成普通人。
<br/>
<br/>        明天她必须做一个血液检查。
<br/>
<br/>        第一次试毒失败，她必须立刻进行第二次试毒炼药，陆先生的身体等不及了，如果她体内的百毒不侵被腐蚀，那她怎么承受第二次的曼陀罗花毒？
<br/>
<br/>        夏夕绾的心一下子心乱如麻。
<br/>
<br/>        陆寒霆听到了外面的动静，知道她来了，他伸手拉开了磨砂玻璃门，只见女孩站在昏黄的灯光下，怀里抱着他脱下的衣服在出神的想事情。
<br/>
<br/>        陆寒霆勾了一下薄唇，“发什么呆呢，想知道我穿多大号的内裤可以明说。”
<br/>
<br/>        什么？
<br/>
<br/>        夏夕绾迅速回神，她垂眸看了一下，她手里一直拎着他的内裤。
<br/>
<br/>        天哪，她在干什么？
<br/>
<br/>        夏夕绾当即将他的衣服都丢进了竹篓里，这时耳畔就传来了他低沉磁性的嗓音，带着取笑，“最大号的。”
