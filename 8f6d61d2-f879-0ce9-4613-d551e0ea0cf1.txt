　　荒古之力化为巨大无比的海洋，无尽无尽的力量如同海啸般，将楚风眠的身躯一下子卷入到了其中去。
<br/>　　这荒古大道，足以是将最为简单的灵力，转变为恐怖的荒古之力。
<br/>　　荒古之力一出，所向披靡，吞噬一切，人挡杀神！
<br/>　　整个密室在这力量的撼动之下，都似乎微微开始震荡起来。
<br/>　　这三尊荒古门弟子，本来就是一方天才人物，每一尊的实力都足以抗衡生死境二重的武者。
<br/>　　如今在吞噬炼化了荒古神丹之后，他们的境界更是一跃暴涨到了生死境四重的地步。
<br/>　　如今他们三人联手，凝聚出的荒古大道，所爆发出来的力量，就算是让一百个巅峰圣者过来，都可以瞬间灭杀。
<br/>　　“完了！”
<br/>　　“这样恐怖的攻击，这小子是必死无疑！”
<br/>　　“哼，敢跟七大宗门作对，跟荒古门作对，简直就是在自寻死路，荒古门这等庞然大物，随便一个汗毛都能压死这种蝼蚁了。”
<br/>　　各种各样的灵识交织这，互相交谈，有嘲笑，有惋惜，在场所有人的觉得，楚风眠是必死无疑。
<br/>　　“为了杀了这小子，浪费了我们一人一枚荒古神丹，更是透支了我们的潜力，真是付出了极大的代价了。”
<br/>　　一尊荒古门弟子看着被淹没在荒古之力中的楚风眠，缓缓的开口道。
<br/>　　“不过杀了他，得到了他的血脉，我们的一切损失都可以弥补过来！”
<br/>　　“这样一个小人物，得到奇遇都可以拥有如此可怕的实力，在加上这颜古龙帝洞府的开启，北域中倒是都有着不小的奇遇。”
<br/>　　“杀了他，得到他的奇遇，我们回到荒古门中虽然是没有太多的收获，也可以从此一飞冲天，未来不可限量！”
<br/>　　一尊尊荒古门弟子互相交谈着，看着在荒古之力下不断轰击的楚风眠。
<br/>　　“砰！”
<br/>　　突然之间，在荒古之力中，猛然一股极为纯净的龙威，猛然响彻而起。
<br/>　　这一股龙威响起的一刻，周围的荒古之力化为的海洋，似乎却是都被震慑住了，开始裂开。
<br/>　　密室更是疯狂震荡起来，轰隆隆的无数灵力，从其中猛然爆发出来。
<br/>　　“荒古之力？荒古门的弟子，比起万年前，简直无法相提并论，就连真正荒古之力十分之一的力量都凝聚不出来，也想要来镇压楚某？”
<br/>　　一声大笑声从中响起。
<br/>　　在这一声笑声之中，令在场任何人都可以听出，其中的嘲笑之意。
<br/>　　“轰隆！”
<br/>　　荒古之力组成的海洋，突然之间一分为二！
<br/>　　楚风眠手持赤炎剑，从中猛然跃了出来。
<br/>　　“碎海！”
<br/>　　楚风眠的剑意，跟战龙决的力量，同时合二为一，全然凝聚在了这手中的剑锋之上。
<br/>　　顿时剑锋落下，这下方，荒古之力组成的海洋，在这一剑之下，节节破碎。
<br/>　　这一剑碎海，便是楚风眠前世所创的三剑之一。
<br/>　　前世的楚风眠创造剑术，也只有理论，无法真正施展，如今这一世的楚风眠，终于是有着足够的实力，施展出他所创的剑术。
<br/>　　“轰隆隆！”
<br/>　　庞大荒古之力化为的海洋，瞬间崩塌，这其中的力量，一百个，一千个巅峰圣者都可以碾碎的海洋，就这样的崩塌了。
<br/>　　“不可能！荒古之力怎么会如此轻易的被击破！这是什么剑术！就连我的没有见到过！这一剑，简直是达到了剑之巅峰！”
<br/>　　一尊荒古门弟子大叫起来，眼睁睁的看着荒古之力被破，他的脸色也是惊骇之际。
<br/>　　荒古大道凝聚出来的一刻，他们的胜利，应该已经是注定的了，可想不到，如今居然是被楚风眠这样轻易的打破。
<br/>　　“不用怕！他就算是能够化解一次荒古之力又如何？我们凝聚出了荒古大道，便是永远不败！继续出手，我就不信这个小子还能如何抵抗！”
<br/>　　令一尊荒古门弟子大声喝道，又是无穷无极的荒古之力，从荒古大道之中席卷下来，向着楚风眠轰击过来。
<br/>　　“没用的！”
<br/>　　楚风眠冷哼一声。
<br/>　　他手中剑锋再次动了，一剑之下，这荒古之力，再次破碎。
<br/>　　同时楚风眠的身形瞬息之间便是消失，在下一刻，一下子来到了那荒古大道之前。
<br/>　　“你想做什么？”
<br/>　　“对抗荒古大道，你是在找死！”
<br/>　　“我们脚踏荒古，看你如何跟我们对抗！”
<br/>　　三尊荒古门弟子，看着楚风眠冲过来，先是一惊，随后冷笑道。
<br/>　　荒古大道凝聚完成，他们踩在其中，也就意味着深处与荒古之中，一切力量都会随着时空抵消，近乎无敌。
<br/>　　“是吗？脚踏荒古？不过是在借助荒古的力量罢了！给我破！”
<br/>　　楚风眠长啸一声，猛然手中剑锋一动。
<br/>　　这一剑，猛然斩在了荒古大道之上。
<br/>　　“疯了吗？荒古大道中可是无敌的，这小子竟然打算抗衡荒古大道的力量！”
<br/>　　一道质疑声，还未说完。
<br/>　　下一刻，在这荒古大道上，其中看似古老的大道，却是已经开始，层层破碎。
<br/>　　其中的力量，似乎都被彻底剥夺，一下子失去了力量的支撑，当场破碎。
<br/>　　“荒古大道又如何？不过是借助荒古时代的力量！我今日就将这时空长河斩断！破了你这荒古大道！”
<br/>　　楚风眠大声咆哮一声，近乎一声龙吟响彻而起，他的剑锋一动，直接斩在了这荒古大道之中。
<br/>　　在这荒古大道的背后，便是一条巨大无比深邃的长河，这长河，比起任何的长河都要深邃。
<br/>　　是来自与时间的长河，这荒古大道，便是就从这长河之中，汲取着来自荒古的力量。
<br/>　　“什么时空长河，在我的面前，都要毁灭！我的剑，足以毁灭恒古！一切与我为敌的，都要灭亡！”
<br/>　　楚风眠手中长剑落下，这时空长河，都被当场打破。
<br/>　　无数人，都眼睁睁的看着楚风眠，打破了荒古大道，甚至是将其背后的时空长河，都给斩断。
<br/>　　“噗！”
<br/>　　三尊荒古门弟子，当场飞了出去，一剑之下，被震得口吐鲜血。
<br/>　　失去了荒古大道的抵挡，仅仅他们三人的力量，在楚风眠的面前，不堪一击。
