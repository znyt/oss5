        &lt;h3&gt;第三卷  第一百八十七章  各怀心思&lt;/h3&gt;
<br/>
<br/>        团团和嘟嘟并没有回到各自的房间，而是藏在他们自己的门口静静的听他们的谈话。
<br/>
<br/>        爸爸带来了爷爷奶奶，是想要领他们去张家吗？
<br/>
<br/>        不行，妈妈坚决不要答应他们，他们不想和妈妈分开。
<br/>
<br/>        可是听见了他们的谈话，俩个人似乎听不太懂，好像是他们和爸爸相认会带来危险，还有妈妈想将他们送走。
<br/>
<br/>        团团对妹妹说：“嘟嘟，刚好今天他们都在，我们要闹清楚一件事情。”
<br/>
<br/>        “对的，哥哥，我们要知道我们的敌人是谁？我们要知道妈妈有没有危险。”嘟嘟很是
<br/>
<br/>        赞同哥哥的话。
<br/>
<br/>        “妈妈。爸爸。”陈子昂将孩子喊出来了，俩个人走到陈子昂的面前。
<br/>
<br/>        “这是你的爷爷和奶奶。”陈子昂对俩个孩子介绍着。
<br/>
<br/>        “爷爷奶奶好。”俩个人异口同声的叫着。
<br/>
<br/>        张沛尧和弥青云看着两个乖巧的孩子，打心眼里高兴，但是又想到他们不能和他们亲亲的孙子相认，心里就堵得慌。
<br/>
<br/>        俩个人拉着孩子左看右看舍不得松手，团团和嘟嘟其实心里有些反感他们的，但是妈妈既然让他们喊他们就喊，只要他们不为难妈妈就好。
<br/>
<br/>        张沛尧和弥青云一人拉着一个孩子问东问西，当他们听到孩子这么小还会说多国语言时，他们惊呆了，陈子昂竟然将孩子养得这么优秀，看着俩个小大人一样的孩子，他们的心里对陈子昂多了一丝敬佩，是的，是敬佩，当张函和他们说出了孩子的事情，他们当天晚上就查找了陈子昂这几年在北京的经历，她独自一人在北京混得风生水起，固然有落家的相助，但是要是陈子昂是扶不上墙的乱泥，就算是有强大的落家怕也是无济于事。
<br/>
<br/>        想着她可能只管事业肯定是没有精力照顾好孩子，他们还担心俩个孩子会不会养歪，毕竟是单亲家庭，但是这么小的孩子知书达理，说话思路清晰，而且会多国语言，可以说小小年纪竟然上知天文下知地理，这让他们刮目相看。
<br/>
<br/>        “团团看了很多书吗？”张沛尧问团团。
<br/>
<br/>        “是的。”团团礼貌的回答。
<br/>
<br/>        “能告诉爷爷看了些什么书吗？”张沛尧想知道团团到底接触了那些书。
<br/>
<br/>        “有，算了，我书房和妈妈的书房的书我们都看完了，要不，我带爷爷和奶奶去看看？”团团说。
<br/>
<br/>        “哈哈，这么厉害，说得我很想去看看，那带爷爷奶奶去看看？”张沛尧拉着团团不舍得松手。
<br/>
<br/>        “好啊，妈妈，我能带爷爷奶奶去你书房参观一下吗？”团团征求妈妈的意见。
<br/>
<br/>        陈子昂点点头。
<br/>
<br/>        团团和嘟嘟拉着张沛尧和弥青云去参观他们的书房了。
<br/>
<br/>        张函已经见过孩子和陈子昂的书房，所以就没有和孩子一起去，俩个人坐在客厅有些尴尬，但确实他们分别这么久第一次俩个人这么近距离的坐在一起。
<br/>
<br/>        “子昂，这些年你辛苦了，我以为我能光明正大的和孩子在一起，没有想到给你又增添麻烦了。”张函内疚的很。
<br/>
<br/>        陈子昂已经从财叔哪里大致知道了张函没有找她的原因，原本心里的那一点不满也烟消云散，造化弄人，合该他们情深缘浅，再说他们都逃不过命运的纠缠，又何苦怨天尤人，苦苦相逼相互折磨呢？他们都已经很辛苦了好不好？虽然怼了张沛尧和弥青云，但是从张沛尧的分析中她也明白了张沛尧他们此次来没有坏心，她心里的那一点芥蒂也就没有了，这个世界上是需要宽恕的。
<br/>
<br/>        陈子昂看了一眼张函，他比之前更沉稳了，显得更成熟，只是眉宇之间隐隐有一股说不出的哀愁。
<br/>
<br/>        陈子昂的心里一酸：“问题来了解决问题，说那么多废话有什么用呢，再说，有些事情迟早都是要面对的。”
<br/>
<br/>        “你，过得还好吧。”陈子昂还是想知道张函这些年过得怎么样。
<br/>
<br/>        “没有你的日子世界一片灰暗，那些蚀骨的相思之疼早已将我击穿，直到偷偷的见到了你，我才从死亡的边缘梦绕魂牵，你是我的光，你是我的天，你是我活下去的动力，是我在黑暗中苟且偷生的生命之源。”曾经俩个人一起吟诗、一起赏月、一起迎着雪花飞舞赏梅斗诗，那一千多个日夜是他们这一生最美好的时光也是俩个人彼此在岁月中苦苦挣扎的念想。
<br/>
<br/>        张函的声音低沉而沙哑，含着哽咽，他每见一次陈子昂，就感觉心疼一次，可是他总是抑制不住的想要见到她。
<br/>
<br/>        陈子昂的心也撕裂般疼，明明她的函哥哥就在眼前，可是他们却隔着永远无法跨越的横沟。
<br/>
<br/>        “函哥哥。”陈子昂声音发颤，他们曾经深深相爱，张函念的诗何尝不是她的写照，这些年她也在梦中惊醒，她带着俩个孩子努力的成长，只是想要成为能够和张函比肩的人，可是无论怎么努力，都只是枉然。
<br/>
<br/>        “过去的事情都已经过去了，只要你好好的生活我也能心安。”陈子昂知道张函做过手术的，她接着说：“我会将孩子带好的，你安心的做你该做的事情吧。还有，你都已经有了老婆以后就不要再念那样的诗了。
<br/>
<br/>        无论如何，我们都回不去了，我在岁月的打磨中已经选择了原谅，原谅所有的人，包括你母亲、陆玉、还有二嫂冯睿、夏颜等。没有这些人也没有我陈子昂的今天，是痛苦是磨难也是成长的动力。所以，函哥哥，请不要难过，请不要忧愁、你原来的子昂已经成长起来了，我会面对一切，不惧风雨、不惧磨难，带着我和你的孩子会幸福的生活着，我也同样希望你幸福。谢谢你曾经爱过我，谢谢你给了我那么多美好的日子。可以说，没有你也就没有我陈子昂的今天。”
<br/>
<br/>        “子昂，请给我时间，总有一天我要跨过千山万水，跨过重重障碍回到你的身边。只要你愿意相信我，给我时间。”张函眼神灼灼的看着陈子昂，只要陈子昂相信他，还爱着他就好了，他的子昂经历了很多痛苦和磨难，在社会上打磨了这么多年，可是她的心底还是那么善良和美好。
<br/>
<br/>        她不曾有过任何改变，依然保持着她的乐观和坚强，依然是那么美丽！
<br/>
<br/>        “我相信你的心，但是我不愿意你伤害任何人，爱，不是占有，而是宽饶和祝福！”陈子昂微笑着说。
