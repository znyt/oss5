　　很快双方的第二局对决便是正式开始，这一局两方换边，白队这边来到了蓝色方，而ka战队这把则是处在了紫色方。
<br/>　　双方轮流ban人，叶帝几人ban的都是目前版本最为强势的几个英雄，而对方则是针对着上一局表现很好的叶帝三人，将鳄鱼、蜘蛛还有狐狸这三个英雄给ban掉了。
<br/>　　“我的天，这一把ka战队的ban人真的是下狠手了啊，不过这三个英雄确实是值得针对，首先就是大魔王的狐狸，他的狐狸的e技能魅惑就像是指向型技能一样，命中率实在是太高了，狐狸这个英雄最考验的就是e技能的命中率，这个技能命中的概率高，那么这个英雄表现的就越加的可怕。”看着对方的ban人米粒开口说道。
<br/>　　“蜘蛛还有鳄鱼这两个英雄也是现在版本非常强势的两个英雄，是值两个ban位的，不过这样一来的话盲僧这个英雄就要被放出来了，这一局白队这边肯定会秒选盲僧的，这样下来感觉ka战队这边更难打了啊。”林雅楠跟着开口说道。
<br/>　　而就在她开口的同时，这边香蕉已经是直接确定下了盲僧这个英雄，看起来自信的不得了。
<br/>　　刹那间场下顿时响起了巨大的欢呼声，很显然大家对于夜雪狂歌的盲僧还是非常期待的。
<br/>　　“6666666666666，居然敢放盲僧，ka战队的教练真的有点太那啥了，这一局我敢打赌，他们肯定会被打爆的。”
<br/>　　“要是我是ka战队的教练，我肯定会去ban掉盲僧的，虽然蜘蛛这个英雄在打野位置上确实要比盲僧厉害，但是不要忘了操控盲僧的那个选手是谁，那可是夜雪狂歌啊，在他手中盲僧的战斗力可是会成倍的增加的。”
<br/>　　“也许是他们以为现在夜雪的盲僧没有那么强了呢，毕竟已经很久没有看到夜雪使用盲僧这个英雄了，要知道盲僧这个英雄如果不经常练习的话，肯定会手生的。”
<br/>　　“他们既然敢抢，那肯定就是很有信心的，依我看夜雪肯定是在排位中经常使用这个英雄，这个英雄可是他的本命啊。”
<br/>　　直播间里的众多观众此刻也都在那里快速的打字议论着，在英雄联盟这个赛场上，盲僧还有锐雯这类的英雄，永远是观众们最为期待的。
<br/>　　而ka战队的比赛室，当ka战队教练看到香蕉秒锁盲僧这个英雄的时候，脸色刷的一下便是变的难看了起来。
<br/>　　他刚刚确实是在赌，赌夜雪狂歌已经有一阵子没有玩过盲僧这个英雄了，现在他对于这个英雄肯定非常的手生。
<br/>　　在这场比赛开始之前，他可是非常关注荣耀白队五人的，他们的战绩页他都看过，并没有看到过夜雪狂歌在最近的排位赛中使用过盲僧这个英雄，这也是他不ban盲僧的原因之一。
<br/>　　只不过很显然他料错了一点，那就是夜雪狂歌在玩盲僧的时候并没有使用自己的大号。
<br/>　　在世界赛来临之际，大家都知道教练会关注自己的这个大号，看自己在玩什么英雄，以此会在赛场上针对自己，所以自然是没有人去上杆子让他们发觉。
<br/>　　双方飞快的选人，很快两边的最终阵容便是确定了下来。
<br/>　　荣耀白队这边选择了：上单兰博、打野盲僧、中路圣枪游侠、下路则是小炮和布隆这两个英雄。
<br/>　　ka战队则是选择了：上单大树、打野挖掘机、中路蛇女、下路则是女警和娜美这两个英雄。
<br/>　　“果然，这一局大魔王果然选择了圣枪游侠这个英雄，之前我们就一直在猜测这个问题，看来大魔王最近一定是在苦练这个英雄。”看着叶帝的选人，米粒连忙开口说道。
<br/>　　“现在圣枪游侠这个英雄基本上每个战队的中单都会去练习一下吧，这个英雄前中期真的是太强了，尤其是在做出破败和黑切这两件装备的时候，战斗力真的是爆表。”林雅楠跟着开口。
<br/>　　“而且你发现没有，为了配合夜雪的打野，白队这边选择的都是前中期非常强力的英雄，这一局看来白队是要打前期了。”米粒继续说着。
<br/>　　片刻后双方的比赛正式开始，而在这一局的比赛之中不出意外的，夜雪狂歌的盲僧再一次展现出了他的风采，让世人知道，他的盲僧不可放。
<br/>　　4分钟，夜雪狂歌直接入侵对方野区，一套技能之后，和对方打野同时释放闪现，豪取一血。
<br/>　　接下来的时间便是彻底的沦为了夜雪狂歌的表演，尤其是当他手中的盲僧到达六级的时候，各种r闪回旋踢，简直防不胜防，甚至对方明明手中有着闪现，结果还是能够一脚被他踢回来。
<br/>　　实在是夜雪狂歌的手速太快了，在他飞过去的时候，手中的眼已经插在了对方的身后，中间几乎没有任何停顿直接飞到了对方的身后。
<br/>　　最终这一局的比赛在时间到达28分钟的时候，便是早早的结束，荣耀白队2:0，率先到达了赛点。
<br/>　　“夜雪的这个盲僧真的是太强了，我感觉他的盲僧比去年大魔王的盲僧还要强上半分。”米粒一脸震撼的说着。
<br/>　　“是的，估计这一局之后，以后再也没人敢放夜雪狂歌的盲僧了，只要这个英雄的技能不改，就算是他的伤害削弱了，只要盲僧还可以回旋踢，那么夜雪狂歌手中的盲僧就是一个bug。”林雅楠跟着开口。
<br/>　　场下此刻已经是欢呼声一片，直播间里的众多观众此刻也都是在那里快速的发送着弹幕。
<br/>　　而与这些欢乐气氛相反的是，此刻ka战队比赛室内的五名队员却是一脸的呆滞，显然他们到了这个时候还没有从比赛中回过神来。
<br/>　　或者说他们已经被白队、被夜雪狂歌那一连串的动作给打懵了。
<br/>　　“现在白队已经到达赛点了，接下来如果在胜一场的话，他们就将会是第一个进入到4强的队伍，期待双方接下来的对决。”米粒继续开口说着。
<br/>　　场上叶帝几人在取得了这场比赛的胜利后，一个个脸上也是挂上了轻松的笑意，已经打到了这个时候，他们的状态、手感都已经起来了，这个时候就是他们最强势的时候，只要他们自己不浪，那么对手绝对没有翻盘的机会的。
