　　“呼。”萧逸深呼吸一口气。
<br/>　　本想着，此次来猎妖殿总殿，或许会有别的收获。
<br/>　　当然，收获也算是有，那三枚乾坤戒内的奖励，让他的修为从圣皇境八重，突破至圣皇境巅峰。
<br/>　　对武者而言，终究修为才是基础。
<br/>　　自然的，此番收获，其实很不错。
<br/>　　只是，得了收获，却也惹下了这莫名其妙的麻烦。
<br/>　　而且是让他萧逸避之不及的麻烦。
<br/>　　若硬说这不是麻烦的话，除非他担任三殿总殿主之职时，不必摘下面具吧。
<br/>　　当然，这显然是不可能的。
<br/>　　炎殿、猎妖殿、药尊殿，三殿总殿之职，非同小可。
<br/>　　岂可能会让一个面容不明的人戴着面具上任。
<br/>　　既不可能，那便代表着，他必然会因此暴露身份；而那时，后果难料。
<br/>　　说起来，他之所以一直保留下易霄这个身份，便是用这个身份背锅的。
<br/>　　退一万步来说，自己多一个身份，行事也方便许多。
<br/>　　他凭此，可以不必有任何忌惮。
<br/>　　即便是凶名赫赫的邪君府，他照样剿杀，一路横扫，丝毫不惧所谓的报复。
<br/>　　可，若这个对象，换成是上古八殿，而且是上古八殿中的其三追杀他。
<br/>　　那么其后果，恐怕他躲到大陆的角落都逃不了。
<br/>　　上古八殿，分殿遍布大陆。
<br/>　　即便不算炎使、不算药尊殿的炼药师，单单是遍布天下的猎妖师，怕就足够他吃一壶的了。
<br/>　　“唉。”萧逸叹了口气。
<br/>　　目光，从手中的五块总殿令牌中，移向其余两块。
<br/>　　想想，修罗殿和风刹殿的令牌，也是个麻烦。
<br/>　　他想起来，当初在修罗总殿时，与总殿主的一番交谈。
<br/>　　他还记得，刚交谈完，离开总殿主的书房时，他已整个手心都是冷汗。
<br/>　　不过是短短一阵交谈，可言语间的博弈，却几乎胜过了他以往和任何一个老一辈武者交谈时的紧张。
<br/>　　他毫不怀疑，修罗总殿主，乃是位睿智无比的前辈。
<br/>　　至于风刹总殿主，就更不必多说了，掌控天下情报，世间几无事情可瞒他。
<br/>　　这两位总殿主，也不是泛泛之辈，而且也不知是否有所怀疑，或者根本已经在调查了。
<br/>　　“唉，麻烦，麻烦。”萧逸一连叹了两声麻烦。
<br/>　　他不知道的是，修罗、风刹两位总殿主，事实上早就怀疑了，不，准确来说，是已经让承风殿主与横天殿主来抓他了。
<br/>　　萧逸摇了摇头，收起了五块总殿令牌。
<br/>　　这些事，他现在多想也无用。
<br/>　　除了感到麻烦外，也改变不了什么。
<br/>　　对比这些，反倒是两月后的古帝之墓盛事，更为迫在眉睫。
<br/>　　古帝之墓，终归要去一趟。
<br/>　　可要说真正能破局，他至今还没把握。
<br/>　　而且，里头到底是何局，还暂时不知晓。
<br/>　　只是，当初听副院长的意思，竟然是他，还有莫悠，再算上五大学宫首席，联手下，方有一丝机会破局。
<br/>　　整整7个当世妖孽联手，才仅仅是一丝机会？
<br/>　　说白了，古帝之墓一行，必然是一次难于登天的挑战。
<br/>　　不过，对萧逸而言，这才更加有意思。
<br/>　　另外，古帝之墓的名额，他虽然已经有了。
<br/>　　但，猎妖殿总殿主的话，却提醒了他。
<br/>　　据他所知，各大势力的名额之争，还未开始。
<br/>　　古帝之墓，共10个名额，中域各大学院占5个，中域各大宗门、家族等势力，也占5个，合共10个。
<br/>　　而上古八殿，则由于地位超然，可不必争夺便各有一个名额，合共8个。
<br/>　　上古八殿的名额，无需争夺，便不必多说了。
<br/>　　百院之争，早便结束了，各大学院的5个名额，也已定下。
<br/>　　可各大势力的那5个名额，可还没定下。
<br/>　　这些天，萧逸在各大地域行走，倒是知晓这一消息。
<br/>　　各大势力的名额争夺，恰好便在半月之后开始，地点，正是极东之地。
<br/>　　萧逸不在乎那个名额，但如同百院之争的第一名，有极其丰厚的奖励，各大势力间的争夺，胜出者，同样奖励诱人。
<br/>　　想也知道，中域各大家族、宗门等等各大势力联手参与的盛事，连中域十八府、北隐宗等霸主势力都参与其中，奖励，怎么可能会差。
<br/>　　嗖…萧逸的身影，瞬间动了。
<br/>　　紫炎火翼振动下，速度飙升到极点。
<br/>　　反正这两月时间，也要捉紧继续提升实力，尽可能实力更强地去闯古帝之墓。
<br/>　　去极东之地那边见识一番，顺便参加比拼，夺些奖励也好。
<br/>　　嗖…嗖…嗖…
<br/>　　紫炎火翼的振动下，每扇一次，便是瞬间横跨千里之外。
<br/>　　远远看去，与其说是一道飞跃的火焰，倒不如说是一团在空间中跃行的流光。
<br/>　　一边飞行，萧逸也一边参悟着脑海中的武道知识。
<br/>　　之前在猎妖殿总殿，两位总殿主对他的炼药本事和掌控天地武道的本事无比惊骇。
<br/>　　外人只看到了的他的绝世天赋，却未看到他的刻苦和磨砺罢了。
<br/>　　炼药一道，这么些年来，萧逸自身所需的庞大修炼资源，皆是靠自己所得，庞大的丹药，也皆是他自己炼制而成。
<br/>　　每次炼制，便是丹药堆积如小山。
<br/>　　这么些年来，他都不记得自己到底炼了多少丹药了，数以万计？不，数以十万计？不，应该是更多。
<br/>　　任何一粒丹药，都未曾假手他人，再高级的丹药，都需得自身参悟丹方，随后亲自尝试炼制。
<br/>　　千锤百炼下，早已让他的炼药本事无比娴熟。
<br/>　　加上他本就天赋过人，才有了这一身连药尊殿总殿主都为之惊讶的炼药本事。
<br/>　　而掌控天地武道方面。
<br/>　　他这些年闯荡、历练，还有通过各种盛事等等，早已身怀多种传承。
<br/>　　其中更是不乏古老之殿的传承、剑帝碑内的武道传承等等。
<br/>　　他的每次赶路，或者有限的闲暇时间，乃至睡觉时的打坐时间等等，几乎都用在了参悟之上。
<br/>　　日复一日，年复一年，加上他本身的天赋，早便将这些传承内的武道知识，大量参悟掌控。
<br/>　　方才做到了每每突破时的降下天地武道力量，他瞬间就能吸摄入小世界内，且达到随意掌控的地步。
<br/>　　修炼之路，萧逸从未耽误半分、懈怠半分，方才有了今日的绝世实力。
<br/>　　……
<br/>　　中域东面，极东之地。
<br/>　　天际之上，一道青色流光，忽然疾驰而来。
<br/>　　流光之内，点点白芒，萦绕其中。
<br/>　　地面，一群老者，早已等候在此。
<br/>　　“好可怕的气息，那是…”一老者，直视远方，面露惊骇之色。
<br/>　　“是幻月天鸾，传说中最为凶猛的传奇妖兽。”一个老者，瞬间反应了过来。
<br/>　　青色流光，转瞬便至。
<br/>　　一头巨大的青色鸾鸟之上，一女子，缓缓而下。
<br/>　　女子身后，数个老者，紧随其后。
<br/>　　“参见圣女。”
<br/>　　等候在此的老者，纷纷行礼。
