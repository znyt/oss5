　　高空中，两道身影追赶不休。
<br/>　　一道，如火焰流光，疾速飞行。
<br/>　　一道，黑气如雾，追赶间如黑云疾驰。
<br/>　　嘶…嘶…嘶…
<br/>　　空气中，十数条蛛丝破空而出。
<br/>　　蛛丝，细微得肉眼难视；在空气中的激跃，只如一道白光，一闪而过。
<br/>　　蛛丝，瞬间缠绕了萧逸的双臂。
<br/>　　毒袍，与萧逸的距离，始终在一个不远不近的距离。
<br/>　　这个距离，毒袍追不上。
<br/>　　萧逸，也离不出毒袍的视线。
<br/>　　既是追不上，毒袍也只能试图用他的驭鬼丝拦下萧逸。
<br/>　　只要拦下一顿时间，他就有把握彻底让萧逸无路可逃。
<br/>　　只是，萧逸甚至没有任何动作，缠绕而上的蛛丝，已瞬间断裂。
<br/>　　双手之上，那黝黑而狰狞的鳞甲，坚硬得可怕，锋利得可怕。
<br/>　　嘶…嘶…嘶…
<br/>　　断裂而回的蛛丝，让得毒袍吃了一惊。
<br/>　　“这小子的武魂，到底是何种兽武魂。”
<br/>　　他毒袍，可不是籍籍无名之辈，而是邪君府护法，凶名赫赫。
<br/>　　偌大个中域，谁人不知？
<br/>　　以他的高度，他的见识，竟然丝毫认不得这是何种兽武魂？
<br/>　　那浑身漆黑的鳞甲，狰狞的细刺，坚硬得惊人，锋利得惊人。
<br/>　　他的蛛丝，连圣器都能轻易束缚乃至割裂。
<br/>　　可是在碰到那黝黑鳞甲之上，却是一瞬间便自动断裂。
<br/>　　还有那些鳞甲之间的点点红芒。
<br/>　　点点红芒，看似寻常，但明显能从中感知到异常骇人的火焰气息。
<br/>　　如此微弱的一点红芒之内，竟蕴含着爆发性的火焰力量。
<br/>　　仿佛整只手臂，都是从火焰中诞生的至强之手。
<br/>　　这样的妖兽，他自问不曾见过，也不曾听闻过。
<br/>　　而面前这个追杀着的妖孽，竟觉醒出如此恐怖的武魂，难怪紫炎易霄之名，能震惊中域。
<br/>　　轰…
<br/>　　这时，萧逸一道火焰打出。
<br/>　　前方，一道紫炎游龙，狂猛而至。
<br/>　　毒袍大手一挥，数根蛛丝，轻松便将紫炎游龙割断打散。
<br/>　　紫晶灵炎，竟是奈何不了他分毫。
<br/>　　嘭…
<br/>　　“身映万毒。”毒袍再次冷喝一声。
<br/>　　身影，化作无数毒虫，速度大增。
<br/>　　前方萧逸，手中的五色圆环，同样一变，化作六色火焰。
<br/>　　轰…萧逸的速度，同样大增。
<br/>　　“该死。”身后，毒袍的脸色，愈发阴蛰，愈发难看。
<br/>　　嗖…无数毒虫，再度化作毒袍的本体。
<br/>　　毒袍的速度，顷刻降下。
<br/>　　前方，萧逸的速度，同样骤降。
<br/>　　“哼。”萧逸瞥了眼身后的毒袍，冷笑一声。
<br/>　　他知道，毒袍追不上他了。
<br/>　　不过，他想要彻底摆脱，也做不到。
<br/>　　二人，只能这样互相拉锯着。
<br/>　　当然，他也没打算彻底摆脱或者逃遁。
<br/>　　他只是在拖延些时间罢了。
<br/>　　如今，他的控火一道实力，早已远远超过了剑道实力。
<br/>　　不过，要说和毒袍这等人物比，还是差远了。
<br/>　　毒袍，这等人物，起码掌控着9998道以上，几乎是最强的绝世强者了。
<br/>　　距离当世传奇，也不过仅差一筹。
<br/>　　他的世间强悍火焰，也丝毫奈何不得。
<br/>　　不，准确来说，不是奈何不得。
<br/>　　世间强悍火焰，乃是世间最强的火焰。
<br/>　　所谓奈何不得，只是他现在修为实力太弱；特别是与毒袍这等人物，差距太大了。
<br/>　　说到底，他不过是圣皇境七重的修为。
<br/>　　圣皇境七重，再过两重，才是圣皇境九重；之后是圣皇境巅峰、武道大能，随后绝世强者。
<br/>　　绝世强者间，更有着10个层次的差距；每个层次，相差一条完整武道，却是云泥之别。
<br/>　　这般算起来，便知萧逸与毒袍的差距，到底大到了什么地步了。
<br/>　　圣皇境九重、圣皇境巅峰、武道大能、绝世强者；之间，几乎是秒杀的差距。
<br/>　　绝世强者之内，9990道，至9999道，同样是几乎秒杀的差距。
<br/>　　萧逸，不过是凭借着世间强悍火焰，才有了一敌绝世强者的实力。
<br/>　　这中间，已经拉平了极大的差距。
<br/>　　可再之后，差距愈发大，世间强悍火焰，便无法企及了。
<br/>　　直至借助九阳轮回诀，将各种强悍火焰的功效融合，方才有了几乎秒杀9993道绝世强者的实力。
<br/>　　至于现在，更多的，则是借助这些兽手、兽脚。
<br/>　　这些兽手、兽脚，也才是让萧逸最为感到惊骇的存在。
<br/>　　它们的每一只，几乎都有着完全拉平绝世层次中一条天地武道的力量。
<br/>　　两只兽手，一只兽脚，整整拉平了3个层次。
<br/>　　在五色火焰圆环之下，便有了几乎秒杀9993道绝世的实力。
<br/>　　再拉平3个层次。
<br/>　　也就是说，在这两只兽手、一只兽脚的增幅下，萧逸直接有了几乎秒杀9996道绝世强者的实力。
<br/>　　算起来，这般状态下，应该便有着等同9997道绝世强者的实力了。
<br/>　　可，这仅仅是五色圆环。
<br/>　　若是达到六色呢？
<br/>　　六色火焰圆环，萧逸暂时无法随意使用。
<br/>　　六种火焰融合下，九阳圆环自成轮回，连带着他体内的庞大元力也自成轮回。
<br/>　　那般庞大的元力强行快速扭曲，他的身体根本承受不住。
<br/>　　明显可以看到，即便他之前对战毒袍时，或许现今与毒袍拉锯追赶之时，他的六色圆环，都是一瞬间凝聚，又一瞬间散去。
<br/>　　虽说只是一瞬，但那一瞬间强行扭曲下的疼痛，还是让人极其难受。
<br/>　　不过，他对这种痛苦的忍耐力足够，故眉宇间并没有表现出来，毒袍也没有发现。
<br/>　　如今，能稳下体内元力扭曲的办法，只有一种。
<br/>　　便是用冰鸾剑武魂强行压制元力轮回。
<br/>　　让得元力的扭曲异常慢，让得身体能够承受。
<br/>　　只是，上次在山洞中修炼，这种办法，足足让他耗费了一天时间，才彻底让得体内元力成就轮回之势。
<br/>　　但，生死大战中，如何能有一天的时间去准备？
<br/>　　他能做的，便是不断磨砺，让得身体能习惯且接受这种轮回之势带来的扭曲。
<br/>　　他原本想着，能在之后的历练和战斗中，慢慢磨砺。
<br/>　　可才刚出那山洞不久，便遇到了天藏学宫一行人，最后直接遇到了这毒袍。
<br/>　　现今，他也只能不断拖延时间；且体内元力，以极可能承受的速度，尽快完成轮回之势。
<br/>　　他强行承受下，以冰鸾剑武魂压制，再加快元力的流转速度，顶多半天，足以完成。
<br/>　　时间，渐渐过去。
<br/>　　二人的追赶，始终没有出现什么变化。
<br/>　　数个时辰后，萧逸内视一番。
<br/>　　体内元力的流转，已然速度不慢；轮回之势，完成大半。
<br/>　　萧逸满意地笑了笑。
<br/>　　又是数个时辰。
<br/>　　这时，半天时间已过。
<br/>　　萧逸内视一番，眼眸一喜，“元力流转轮回，已成。”
