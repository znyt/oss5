　　因为之前两场比赛结束的很快，所以荣耀蓝白两支战队的比赛准时在3点开始。
<br/>　　叶帝他们进入游戏调试好设备后，便是在那里打着自定义模式，找着手感。
<br/>　　“好的，荣耀蓝白两支战队的比赛，现在正式开始。”伴随着‘砰’的一声金属撞击声，解说席上的米粒激动的大声喊道。
<br/>　　与此同时现场十万人也是轰然爆发出一阵巨大的欢呼声，许多人还举着自己手中的画板在那里使劲的摇晃着，俨然一副世界决赛的气氛。
<br/>　　“这一局蓝队在蓝色方，白队在紫色方，这边蓝队上来直接ban掉了夜雪的盲僧，看来他们对于夜雪的盲僧很是忌惮啊。”看着蓝队的ban人，林雅楠连忙开口说道。
<br/>　　“自从男枪被削弱之后，虽然还是可以登场，但是用他的人却是越来越少了，那么自然而然的夜雪第二擅长的盲僧直接取代了男枪的位置，蓝队这边准备很深啊。”米粒跟着开口说道。
<br/>　　看着蓝队的ban人，叶帝这边犹豫了一下后，开口对着身边的香蕉说道：“ban掉锤石。”
<br/>　　锤石这个英雄也是接连被砍了几刀，如今在赛场上已经不是很活跃了，不过考虑到对方的辅助是秦霄，叶帝还是给予了足够的尊重。
<br/>　　只要锤石那几个技能还在，那么无论他的伤害怎么削弱，这个英雄放在秦霄手里，那肯定还是会爆发出属于世界第一锤石的实力来。
<br/>　　“白队这边也是不出所料的针对了一手霄神，双方针尖对麦芒，谁也不让谁。”
<br/>　　就在林雅楠话音刚落，蓝队这边已经是快速的ban掉了第二个英雄，不是别人正是香蕉的鳄鱼。
<br/>　　在最近香蕉几场的比赛中，鳄鱼俨然已经成为了他最拿手的一个英雄，这个英雄在前中期可是非常强势的，尤其是在香蕉手里，加上鳄鱼还有e技能的位移，想要抓他还是有些困难的，所以这边李景禹还是选择了这样一手ban位。
<br/>　　“把林萧的圣枪游侠ban了吧。”这一把叶帝ban人还是主要针对对方的下路，和他们相比，自家的萝卜和茄子两人却是是有所不如，所以他这边才决定将对方最擅长的下路英雄ban掉。
<br/>　　双方轮流ban人，很快各自便是将英雄ban完。
<br/>　　最终蓝队这边ban掉了：盲僧、鳄鱼和卡萨丁这三个英雄。
<br/>　　而白队这边则是ban掉了：锤石、圣枪游侠和泰坦这几个英雄。
<br/>　　双方飞快的选人，片刻后最终阵容便是确定了下来。
<br/>　　蓝队这边：上单大树、打野人马、中路机械先驱、下路则是女警和娜美这两个英雄。
<br/>　　而白队这边则是选择了：上单巨魔、打野蜘蛛、中路妖姬、下路则是ez和扇子妈这两个英雄。
<br/>　　典型的依旧是下路抗压组合，在面对蓝队这种超一流下路组合，萝卜和茄子两人还是果断的选择了怂。
<br/>　　“没想到这一局大魔王居然拿出了妖姬这个英雄，这下子可有看头了。”米粒笑着说道。
<br/>　　“如果单论阵容的话，到了后期还是白队这边更强一些的，无论是大树还是人吗亦或者是机械先驱，这些英雄到了后期的能力可都是要强过白队这边的同位置英雄的。”林雅楠跟着开口说道。
<br/>　　而就在两人议论着的时候，场上双方已经是进入到了游戏之中。
<br/>　　“准备一级抓一波。”看着自家这边的阵容，进入游戏的第一时间叶帝便是开口指挥道。
<br/>　　既然阵容已经占据劣势了，那么他自然是想要在前期拿到一点优势，这样的话也容易滚起雪球来。
<br/>　　只不过仿佛早就已经料到他们这边会这样行动是的，在叶帝五人如今对方红buff野区的时候，秦霄五人也是来到了叶帝他们这边的红buff区域，在这里布置视野。
<br/>　　“双方这开局的博弈很精彩啊，不过还是蓝队这边技高一筹，已经料到白队会来抓他们了，所以出门直接奔着对方的红buff而来。”米粒大声的说着。
<br/>　　而场上在看到对方红buff野区没有人后，叶帝几人已经是第一时间猜到了对方的动作，布置了一番视野后，叶帝一行人便是向后退去。
<br/>　　还是回到了自家的蓝buff区域，他们这边倒是也不担心对方会偷自家的红buff，如果真的那样的话，大不了自家这边拿完蓝，在直接去对方的红buff就好了，反正蜘蛛这个英雄打野速度还是相当快的。
<br/>　　1分40秒双方正常对线，叶帝这边虽然想要利用w技能消耗一下对方，不过他这个时候也并不确定蒋念一级点的到底是什么技能，所以他也没敢有所动作。
<br/>　　如果蒋念一级升级的是q技能的话，那么自己这一波过去换血的话，那绝对是亏的。
<br/>　　对方的q技能又护盾、有伤害，到时候再加上他的天赋雷霆在，真要是换血，自己绝对是换不过他的，更何况自己w技能真的要是踩中他的话，可是还要承受着对方小兵的攻击的。
<br/>　　所以一级的时候叶帝这边倒是选择稳了一手。
<br/>　　“现在只能抓二级这一波试着消耗、压制一下蒋念了，不过估计这小子也不会给我这个机会。”叶帝心里默默的嘀咕着。
<br/>　　这一局从选人上他们这边就已经占据了劣势，所以叶帝心中还是有些急的。
<br/>　　如果换做是其他战队还好，他们完全可以在中期就结束对手，不过对方是蓝队，这一场比赛肯定会被拖到后期的，那个时候自家是很难打的。
<br/>　　叶帝这边在盯着小兵，另一边蒋念同样也在关注着这里，二级妖姬的爆发有多高，他自然是非常清楚的，所以他肯定是不想被叶帝平白打上一套的。
<br/>　　当叶帝这边清理第七个小兵的时候，另一边蒋念已经是先行后撤，知道自己抢不到二级之后，蒋念果断的选择了怂，前期丢几个兵倒是没什么问题，只要自己起来了，到时候一个e技能一波兵，自己完全可以利用三只手这个英雄牵制住对手。
<br/>　　到时候对方无法游走，那么妖姬这个英雄也就失去了一半的作用过来。
