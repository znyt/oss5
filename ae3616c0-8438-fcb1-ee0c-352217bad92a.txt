        &lt;h3&gt;第三章第四十七章  闫欢和陈子悦的事业&lt;/h3&gt;
<br/>
<br/>        闫欢忙着和陈子悦开超市，干她们的事业。
<br/>
<br/>        她似乎是忘记了那一段不堪回首的经历。
<br/>
<br/>        偶尔，她会在夜深人静的时候惊醒，还以为她还在甸村那个小山村里，周围是男人粗野的笑声。往事不堪回首，如果，她不认识张函，不认识陆玉或者说不去上那个重点的学校，也许，她的人生会不一样。
<br/>
<br/>        她是不是已经是一个地方的重要的人物？亦或者是一个教授？有一个美满幸福的家庭？有活泼可爱的孩子？
<br/>
<br/>        这个世界上没有如果，当她到达那个黑暗的地带，她求生不得求死不能的时候，她被恐惧折磨着。
<br/>
<br/>        她逃不出那个地方，她走不出那个村口。一次一次的失败，一次一次的受尽屈辱，最后她麻木了。她心里充满了怨恨，如果她能逃出去，如果有来生，她一定要让那个人受到最恶劣的报复。
<br/>
<br/>        她在绝望中过了一年又一年，她以为她再也没有了将来，谷强的突然出现，给了她希望，也给了她尊重，谷强在的那一段时间，她认为是她此生最快乐的时光，可是不久，那希望消失了破灭了，她又堕落回到了那黑暗的角落。
<br/>
<br/>        后来，张函从天而降，像天神一样将她带出来了，她充满了感激。
<br/>
<br/>        张函将她放在家里，她知道，张函是用她来气陆玉的，而她也愿意，在张家的眼皮子底下，陆玉也不能将她怎么样，她故意的当着陆玉的面和张函亲亲蜜蜜的说话。
<br/>
<br/>        她也知道，张函喜欢的人是陈子昂，不知道陆玉是用了什么手段将陈子昂逼走，唯一幸运的是，陈子昂逃过了一劫。
<br/>
<br/>        也许，陈子昂那几年就是为了躲避陆玉的迫害？可是陈子昂为什么后来又成了一个业界大佬？她有着怎样的经历？
<br/>
<br/>        闫欢对陈子昂充满了好奇，所以，陈子悦要去见陈子昂她也要跟着一起去，按道理，人家兄妹聚会，她是不应该去的，但，她就是想去见一见陈子昂。
<br/>
<br/>        “想啥呢？”
<br/>
<br/>        “想张函。”
<br/>
<br/>        “你一直单身就是因为张函？”
<br/>
<br/>        “不是。”
<br/>
<br/>        “张函给我说，子昂是他的爱人，我很好奇，为什么他们没有在一起。”
<br/>
<br/>        “门不当户不对。”陈子悦后来才知道，陈子昂和张函的事情，陈子昂去北京也一直都没有和她联系，也就是最近这几年才开始联系。
<br/>
<br/>        她很后悔当初没有阻止张函和妹妹来往，她也很后悔轻信了张函的承诺。
<br/>
<br/>        只是后来崔建来找过她，询问陈子昂的下落，告诉她，张函到处在找陈子昂，她对张函的哪一点芥蒂才慢慢消失。
<br/>
<br/>        “怕不是门不当户不对吧？张函其实是很爱子昂的。”
<br/>
<br/>        “你怎么知道？”
<br/>
<br/>        “我曾经在张函家里呆过一段时间，张函的书桌上压着子昂的照片，还有张函找过子昂。”
<br/>
<br/>        “找过又能怎么样？如今还不是各奔东西？当初看着多好的一对，在现实面前还不是得乖乖低头。”
<br/>
<br/>        闫欢看了一眼陈子悦：“不过，你们兄妹都很厉害，个个都是人才。”
<br/>
<br/>        “我们是农村的，现在不是流行一句话吗，没有伞的孩子必须得奔跑。我们没有任何依靠，只能靠我们自己努力的奋斗，虽然我们比不上人家先天家世好的人，最起码，我们努力了就有收获。我想，我家子昂也是这样想的吧。她应该比我更有抱负，是她改变了我。”
<br/>
<br/>        “你们家兄弟姐妹真多，可以相互帮衬着。”
<br/>
<br/>        “各家都有各家的事情，再加上我们年纪小的时候，家里都穷都是自顾不暇，哪能帮着别人呢，倒是这几年，大家日子过得好了，来往才多起来。”陈子悦感慨着，那个时候不是兄妹们不齐心，而是都无能为力。
<br/>
<br/>        “也对呢。”
<br/>
<br/>        “你们兄妹八个啊，现在都挺好的吧？”
<br/>
<br/>        “都不错，就是我六弟一直杳无音信。”
<br/>
<br/>        “你六弟叫什么名字呢？”
<br/>
<br/>        “陈子寒。和子昂长得特别像。”
<br/>
<br/>        闫欢心里一动，谷强！
<br/>
<br/>        “你们一直都在找他吗？”
<br/>
<br/>        “当然啊，自己的弟弟，只是这么多年，生不见人死不见尸，连一点点音讯都没有。”
<br/>
<br/>        “子悦，没有音讯就是最好的消息，也许子寒在某个地方呢。”
<br/>
<br/>        “谢谢你，闫欢，我们四个小的关系是最好的，子寒只比我小两岁，有时候还梦见小时候的事情呢。”
<br/>
<br/>        闫欢是见过谷强的，谷强和陈子昂有八分像，不过一个是女孩子一个是男孩子罢了。
<br/>
<br/>        谷强的身份她也是知道的，她曾经在村口偷偷给蒋靖送过东西。
<br/>
<br/>        后来谷强受伤昏迷了一段时间，也是她照顾的，不过谷强走后，她就再也没有见过谷强，也不知道谷强怎么样了，再后来张函就将她救回来了。
<br/>
<br/>        “你和我说什么客气话呢，我和你们有缘分呢。”闫欢说的是真话，她和他们一家人真的有缘分呢。
<br/>
<br/>        认识谷强，后来又被张函送到陈子悦这里。
<br/>
<br/>        陈子悦是个好人，心地善良，努力上进，给了她很大的发展空间，信任她，她们俩个女人在一起合作的得心应手。
<br/>
<br/>        “欢欢，你得找个人嫁了。”陈子悦催嫁。
<br/>
<br/>        “你嫌弃我了？”
<br/>
<br/>        “你嫁人又不妨碍我们在一起。”
<br/>
<br/>        “我不想嫁人，这样过得也很好，你看我们现在的事业正是如日中天的时候，我要是再嫁人，就没有精力管这一大摊的事情了。”闫欢的眼前闪过谷强的面容。
<br/>
<br/>        陈子悦并不了解闫欢的过去，只是觉得闫欢年龄大了，应该嫁个人了。
<br/>
<br/>        “这样一说，倒是我拖累了你。”
<br/>
<br/>        “嫁人多麻烦了，要生儿育女，要伺候别人的妈妈，你看看我多轻松的。好了，等我以后有合适的再说。”
<br/>
<br/>        俩个人平时忙忙碌碌的，倒是难得有今天这样的聊天机会，这样一路聊着天，路上有些堵车，竟然也不感觉到着急，很快就到了指定的地方。
<br/>
<br/>        天气很好，西城是四面环山，夕阳西下，撒下的余辉笼罩着这个城市，泛着金色的光芒。
<br/>
<br/>        天边的云彩如火般熊熊燃烧，那一坨圆圆的火红在楼宇间、在山边偶尔被云彩遮挡，将云彩燃成火红色，然后悄悄的西下。
<br/>
<br/>        陈子悦等闫欢停好车，俩个人亲密的拉着手走向饭店。
<br/>
<br/>        女人之间一旦经历过同甘共苦而又不离不弃，那么这样的情感在此后的人生是不会有任何改变的。
<br/>
<br/>        陈子悦和闫欢是同一类型的人，初始懦弱，后来坚强，再后来无坚不摧。
