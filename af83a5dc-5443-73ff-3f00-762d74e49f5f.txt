陪我走这一趟？”<br/>　　<br/>　　众人听到这话，顿时全都大急。<br/>　　<br/>　　“万万不可！”<br/>　　<br/>　　“叶少将，你这是在拿林司令的性命开玩笑！”<br/>　　<br/>　　“简直胆大包天！”<br/>　　<br/>　　……<br/>　　<br/>　　林震南却再次摆了摆手，沉吟道：<br/>　　<br/>　　“其实我个人的性命倒也不算什么，只是如果真的有什么万一的话，南海战事，一触即啊！”<br/>　　<br/>　　说到这里，林震南直视着叶尘的眼睛，缓缓道：<br/>　　<br/>　　“我可以按照你所说的去做，但不知道你有几成把握？”<br/>　　<br/>　　叶尘傲然道：<br/>　　<br/>　　“十成！而且我以我的项上人头做担保，定能确保林司令安然无事！”<br/>　　<br/>　　林震南望着叶尘身上，那股强大的自信气场，眼眸中不由得流露出一抹深深的赞赏，当即豪迈一笑，道：<br/>　　<br/>　　“好！反正我这条命，也是叶少将你救回来的，我就陪你走这一遭！”<br/>　　<br/>　　欧阳寒等人，见林震南居然跟着叶尘一起疯，顿时全都懵了，<br/>　　<br/>　　“林司令不可呀！”<br/>　　<br/>　　“这太危险了！”<br/>　　<br/>　　……<br/>　　<br/>　　然而，林震南似乎心意已决，<br/>　　<br/>　　“诸位不必再劝，就这么说定了！”<br/>　　<br/>　　说实话，这个结果，连叶尘也颇感意外，他本以为林震南身居高位，肯定不会这么轻易以身犯险，没想到他居然答应的如此爽快，心中不由得油然升起一股敬意。<br/>　　<br/>　　“林司令，我需要五天时间准备，至于这五天之内，如何将这个信息传递出去，让那些杀手组织相信，这就就要看你的安排了！”<br/>　　<br/>　　林震南欣然应允。<br/>　　<br/>　　……<br/>　　<br/>　　夜半时分，一座无比荒凉的小岛，<br/>　　<br/>　　叶尘盘坐在一块硕大的岩石上，缓缓从怀里取出了两样东西，<br/>　　<br/>　　一个黑不溜秋的小钟，和一把小巧精致的赤红色飞剑！<br/>　　<br/>　　那黑色小钟，赫然是当初在云间小镇拍卖而来的半灵器，而且经过叶尘这一段时间的元神蕴养，已经诞生出一丝灵智，逐渐进阶成为了真正的防御灵器。<br/>　　<br/>　　至于那把赤红色飞剑，则是楚飞烟的家传灵器飞剑，飞剑上的邪恶之气，也已经被他基本驱除干净。<br/>　　<br/>　　“我虽然自信，这世上无人会是我的对手，但事关林司令的性命，为了保险起见，我必须要在五天之内，将这两件灵器炼化，到那个时候，才是真正的万无一失！”<br/>　　<br/>　　第二更，<br/>　　<br/>　　(本章完)<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/>
