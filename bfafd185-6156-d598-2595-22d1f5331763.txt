“绾绾，林婶我带走了，我在帝都城等你。”
<br/>
<br/>        说完，苏希转身离开。
<br/>
<br/>        他在帝都城等她？
<br/>
<br/>        他可是海城四大豪门之一，他竟然在帝都城等她？
<br/>
<br/>        又是帝都城。
<br/>
<br/>        似乎很多人都在帝都城等着她，那里好像有什么东西总在召唤她。
<br/>
<br/>        一开始夏夕绾就有这个感觉，现在苏希的打来让这种感觉更加明显强烈了些。
<br/>
<br/>        夏夕绾侧身，看着苏希离开的背影，雨点噼里啪啦的下，一身黑衣的他显得格外俊逸清寒，雨滴从伞面掉下来，竟然丝毫溅不到他的身上，平添出几分神秘的色彩。
<br/>
<br/>        这个苏希究竟是什么人？
<br/>
<br/>        他是完全听命于她的妈咪的？
<br/>
<br/>        夏夕绾知道妈咪很喜欢苏希，那时候妈咪总是一手牵着她，一手牵着苏希，那几年里苏希是妈咪亲手带出来的。
<br/>
<br/>        她好像忽略了些什么。
<br/>
<br/>        夏夕绾在冥思苦想，她没有注意到墓园的前方静静的停着一辆加长版的商务豪车，陆寒霆来了。
<br/>
<br/>        陆寒霆坐在后座上，他已经出院了，今天他穿着白色衬衫打领带，外面商务马甲，最外面还罩了一件黑色大衣，刚从公司的高层会议上下来，整个人英俊矜贵，举手投足都弥漫着成熟优雅。
<br/>
<br/>        现在他深邃的狭眸透过落地窗落在女孩的身上，见女孩盯着苏希的背影看，他蹙了一下英气的剑眉，眉心里染上了一层阴霾的寒霜。
<br/>
<br/>        驾驶座上的严毅已经感觉到了男人身上骤冷的气场，最近总裁没什么情绪起伏，但太过面无表情，总透着一层薄冷疏离，整个公司都是压抑的，大气不敢出。
<br/>
<br/>        这时外面又来了一道俊美颀长的身影，是陆子羡打着黑伞来了。
<br/>
<br/>        严毅觉得头皮有点发麻，他透过后视镜看了一下后座上的男人，“总裁，这…这太太还真抢手，这才刚离婚，这些人就按耐不住了呵。”
<br/>
<br/>        陆寒霆很早就来了，一直在这里看着她，他知道她很难过，但是他没有办法走近她，没有办法安慰她。
<br/>
<br/>        后来他看到苏希来了，现在陆子羡也来了…
<br/>
<br/>        陆寒霆抬头，锋利如鹰隼般的眸子往严毅脸上扫了一眼。
<br/>
<br/>        严毅那声“呵”就卡在了喉咙里，出不来了，他觉得脖间凉飕飕的，总裁吃醋的眼神真可怕。
<br/>
<br/>        “回去吧。”这时陆寒霆掀了掀薄唇开腔道，他没有办法再留在这里，他怕控制不住自己。
<br/>
<br/>        “是，总裁。”
<br/>
<br/>        加长版的商务豪车疾驰而去。
<br/>
<br/>        ……
<br/>
<br/>        墓碑前，陆子羡将手里的花献到了墓碑前，然后看向夏夕绾，“夕绾，你接下来有什么打算，两天后我要离开这里回帝都了。”
<br/>
<br/>        “你要走了吗？”
<br/>
<br/>        “恩，我只有一年的时间，现在时间已经到了，我不能离开太久，而且我妈一直在催我回去，夕绾，你跟我一起走吧。”
<br/>
<br/>        陆寒霆曾经说过，他这一年的时间就是来海城寻找她的，不过夏夕绾垂下纤长的羽捷，无法给予他回应，“我把这里的东西整理一下，很快就会去帝都。”
