　　第570章
<br/>　　抬头就见火光冲天，万卷藏书在火海中飘扬，历代先人智慧，前辈心血都在慢慢化为灰烬。
<br/>　　夫子一张脸煞白，冷汗直冒，结结巴巴指着程大雷。
<br/>　　“你……你……”
<br/>　　程大雷仰天大笑，十分嚣张欠揍。
<br/>　　“我且看看你，是杀人还是救火？”
<br/>　　夫子此刻心如刀绞，他当然选择救火，楼中的藏书很多都是孤本，千辛万苦收集起来废了多少力气，他容易么他。但是，他也想杀了程大雷。今日一番乱战，已见程大雷的悍勇，今日不杀，怕以后便没有机会再杀。
<br/>　　望着程大雷猖狂的笑容，他很想上去给程大雷两个大嘴巴子。但现在救火要紧，也没时间和程大雷扯淡。
<br/>　　“姓程的，你莫要太得意猖狂，我问你还想不想要那女娃子性命？”
<br/>　　“什么女娃子？”程大雷稀里糊涂。
<br/>　　“哼，一个道姑，一个小姑娘，是姓李是么？”
<br/>　　程大雷一怔，脸色变得难看起来。夫子既然能说得这么详细，自然不是空穴来风，可李婉儿与李莫愁如何落到他们手中，程大雷还想不明白。
<br/>　　“想要她们活着，三日后到白沙岛来，我到时候再亲自会你。”
<br/>　　夫子带着人就走，他知道只要一撤走人，剩下几个根本拦不住程大雷，无非是给程大雷多造杀孽的机会而已，所以他干脆全部带走，一个也不留。
<br/>　　“喂，太过分了吧，江湖上的事，为何牵扯女人进来。”
<br/>　　“哼，对你这样恶贯满盈的恶贼，何必讲江湖道德。”
<br/>　　夫子连头都没回，带着人着急走了。现在他一门心思想的就是救火，从火场中多抢救几本书来。很多书烧毁了，可就再也没有了。
<br/>　　程大雷呆呆立在那里，良久没回过神来。眼前空空荡荡，一个人影也无，刚才想要杀他的人，此刻都不见了。
<br/>　　“大当家，咱们走吧？”刘发财的声音响起。
<br/>　　“走。”
<br/>　　程大雷拽着黑牛往山下走，却是无人敢拦。倒也没人去拦，大多数人都忙着救火，现在程大雷明显成了无关紧要的存在。
<br/>　　下山路上，还真被程大雷碰见一人。一个丑汉抱头鼠窜，被程大雷逮到后，双方都是一楞。
<br/>　　“咦，怎么是你？”
<br/>　　二人异口同声，被程大雷抓住这人，正是华正则。
<br/>　　“你不在山上救火，跑到山下做什么？”程大雷奇怪道。
<br/>　　“救火和我有什么关系。”华正则理所应当道：“我是个大夫啊。”
<br/>　　山上着了大火，说不得会引燃其他建筑。如华正则这种人，自然想着逃命要紧。对于书楼中收藏的医书，他当然也是很心疼的，可让他身赴火场救火，他却没有这个觉悟。
<br/>　　只是没料到，跑下山的时候，正好和程大雷碰个正着。
<br/>　　“哎呀，太好了。”程大雷眼前一亮：“正好碰上，你与我一起走吧。”
<br/>　　“那个……去哪儿？”
<br/>　　“上次你答应帮我去救人，难道不作数了？”
<br/>　　“当然作数，华某言出必践，最是一诺千金。”
<br/>　　“那就好，走啦，走啦。”
<br/>　　程大雷扯上华正则，队伍又增加一位成员，三人一牛结伴下山。
<br/>　　三人走的是另外一条路，所以没碰见山门前大战的燕不归。而此时此刻，燕不归方面的战斗也同样接近结束。
<br/>　　本来，会以另外一种方式结束，那就是燕不归被活活耗死。可山上那场大火却在烧烤着所有人的心。尤其是宋寒之，他心若刀割，心里琢磨着：哪个恶人胆敢放火烧楼。
<br/>　　他当然想不到楚云生身上，不过确实觉得这件事可能和程大雷有些关系。如今已经是无心恋战，想着还是上山救火要紧。
<br/>　　指挥者的松懈，令整个剑阵出现破绽。而对燕不归这样的高手来说，任何破绽都是不应该出现的。
<br/>　　打到现在，燕不归已渐渐清醒过来，以血浇仇，今天杀的人已足够多，鲜红的血令他冷静下来。
<br/>　　张六迟迟不肯出来，或许，他当真不在书院。自己杀再多人，杀不掉张六这个元凶有什么用。
<br/>　　留得青山在，不愁没柴烧。
<br/>　　这个念头冒出来，燕不归已经想溜。双方的距离越拉越远，在找到空隙后，燕不归麻溜快的往外逃。
<br/>　　他的确已接近油尽灯枯，战斗哪怕再持续一刻钟，他恐怕连逃跑的力气也无了。
<br/>　　程大雷三人离开万卷山后，找了个废弃的破庙藏身。今夜的琅琊城乱糟糟一片，估计书院一时半刻也分不出人手捉拿他们。当然，如果不能将程大雷陷入绝地，想要拿住程大雷也并不容易。
<br/>　　今夜，程大雷已经证明了这一点。
<br/>　　华正则先给刘发财和程大雷身上的伤口止血，简单的包扎后，二人都躺在枯草堆上。
<br/>　　这华正则不亏有神医之名，妙手回春，药到病除。更重要的是，他没有职业歧视，的确，连戎族他都不在意，何况程大雷只是个山贼。
<br/>　　“那个程当家，你要带我去哪儿？”
<br/>　　“凉州。”程大雷窝在草堆中。
<br/>　　“呃……那么远么，要不要我先回家打个招呼，收拾几件行李。”
<br/>　　程大雷瞪了他一眼，不是很愿意搭理他。华正则察觉到程大雷情绪不是很好，所以也没再说什么。
<br/>　　“大当家，咱们什么时候动身？”
<br/>　　“等你的伤养好，你们随时可以动身。”
<br/>　　“你不和我们一起去？”
<br/>　　“嗯呐，我还有些事要做。”
<br/>　　华正则激灵，刘发财却是比他更加激灵。他当然知道程大雷为何情绪不好，也知道对方要做什么。但以他对书院的了解，程大雷在意的人，现在是活着死了还说不好。或许，对方只是钓鱼的饵，而那白沙岛，怕是好去不好回。
<br/>　　“大当家，不若你再想想，书院的话未必是真的。你也说了，读书人什么的，说话最是信不过。”
<br/>　　“嗯，我也就是上去看看。”
<br/>　　程大雷轻描淡写道，但刘发财明白，他越是平静，越是心中主意已定，再难更改。
<br/>　　他轻轻叹口气，却也没有再说什么。
<br/>　　此夜此城，在琅琊城的街道上，一个年轻人一瘸一拐的向前走着。
<br/>(本章完)
