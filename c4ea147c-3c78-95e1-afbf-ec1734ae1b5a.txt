        &lt;h3&gt;第三卷第七十四章  为乔小东庆生&lt;/h3&gt;
<br/>
<br/>        国庆节放假了。
<br/>
<br/>        公司里除了安保人员和了钱喵喵值班外，其余都放假了。
<br/>
<br/>        钱喵喵是策划部经理，从搬到新的办公地点之后，他们公司就决定每年都会安排高层和安保人员一起值班。
<br/>
<br/>        陈子昂想回西城，自从上次回去看了父母亲之后，她总是想回家去看看，可是俩个孩子怎么办？
<br/>
<br/>        七天假如果她不带走孩子心里又不甘，带着孩子又如何和家里人交代？
<br/>
<br/>        想了一想，她征求了一下春姨和财叔的意见。
<br/>
<br/>        “春姨，财叔，我想回西城去看看我爸妈，想把孩子也带上。”
<br/>
<br/>        “带上就带上吧，让孩子出气转转也好。”
<br/>
<br/>        陈子昂就是带着俩个孩子出过国之外，国内基本没有去过。
<br/>
<br/>        “可是.....”
<br/>
<br/>        “我们俩个人也没有去西城呢，我们一起去吧，你回家了我就带着团团和嘟嘟逛一逛。”春姨说
<br/>
<br/>        “啊，那太好了。”其实他们出国的时候财叔和春姨也会一起。
<br/>
<br/>        春姨和财叔了解陈子昂的为难，便主动提出来和陈子昂一起到西城。
<br/>
<br/>        俩个孩子能和妈妈一起，便也是开心得就像是两只小兔子一样蹦蹦跳跳的，小学的孩子对这个世界还是充满了好奇，他们知道妈妈是西城人，所以经常会看一些西城的历史，早就对西城向往不已。
<br/>
<br/>        陈子昂买好机票一行五个人就回到了西城。
<br/>
<br/>        母子三人在飞机上就已经开始讨论西城的历史了，团团摇头晃脑的说：“五千年华夏谱新篇，十三朝古都我来看，唐诗宋词阿房宫，秦砖汉瓦又古陵。”
<br/>
<br/>        陈子昂听见儿子最后的那几个字，像被雷穿过心墙般疼楚，她和张函有着美好回忆的地方，第一次见张函的地方。
<br/>
<br/>        眼泪差一点就会夺框而出，为什么最近总是时不时的想起张函？
<br/>
<br/>        陈子昂并没有给陈子霖说她回西城了，她要先带着孩子在西城转两天，然后再回去看父母亲。
<br/>
<br/>        陈子昂包了那次给她留手机号的那个出租车司机的车，那个司机叫从哲，而那个出租车司机因为带着寒伯安转了两天，现在已经是西城物流部的领导了，他自己买了一辆商务车，陈子昂叫过来刚刚好。
<br/>
<br/>        西城分公司的车被陈子霖开回去了一辆，还有一辆在公司，陈子昂不喜欢开车，人多车多，开车很累的不如包一辆车。
<br/>
<br/>        五个人，四天时间基本逛得差不多了，五号，陈子昂打算回去看看父母亲，财叔和春姨带着孩子去陵县。
<br/>
<br/>        六号陈子霖和妹妹一起赶回西城，在俩个人聊起西城后期的项目进度的时候，聊到了乔小东，陈子霖说：“子昂，我记得乔夫子的生日好像是六号呢，不知道他在不在西城？如果在的话我们给他过个生日？”
<br/>
<br/>        “好啊，那我打电话联系一下？”
<br/>
<br/>        “行，呼延寰宇和李凌菲应该都在家里，他们都没有出去。另外，江俞轩好像也在西城，江家有个大项目，他一时半会儿那是不能离开西城。”
<br/>
<br/>        “好，我先联系一下，看看他们都在不在，如果在西城，晚上一起聚聚。”
<br/>
<br/>        陈子昂先给乔小东打电话：
<br/>
<br/>        “嗨，子昂。”
<br/>
<br/>        “夫子？出去玩了吗？”
<br/>
<br/>        “没有，在家呢。人太多，不如在家里舒坦。”其实乔小东原本想去青海逛逛的，只是孩子不巧有点咳嗽。
<br/>
<br/>        “太好了，我到西城了，晚上喝两盅？”
<br/>
<br/>        “你回来了？怎么不早说？”
<br/>
<br/>        “回去看了我妈妈，今天到西城了。”
<br/>
<br/>        “行，那晚上见。我来定地方？”
<br/>
<br/>        “你别定地方了，我已经定好了，等着我一会将地址发给你。”陈子昂知道了乔小东没有出去，就在离乔小东近的地方定了个吃饭的地方，某珍轩，西城数一数二的大酒楼。
<br/>
<br/>        然后给江俞轩打电话，江俞轩因为要开发某江国际社区附近的楼盘，陪着姐姐江俞琪加班加点，原来准备十一期间要结婚也就推后了。
<br/>
<br/>        “子昂，有事吗？”江俞轩一手拿着鼠标，眼睛紧紧的盯着电脑屏幕，害怕错过一个字，一手接陈子昂的电话。
<br/>
<br/>        “晚上和俞琪姐姐一起吃饭吧？”
<br/>
<br/>        “你到西城了？我问一下我姐姐有没有时间？”江俞轩直接对着话筒说：“姐，陈子昂让我们晚上一起去吃饭，你去不去？”
<br/>
<br/>        “去吧。”江俞琪本来说不去，他们这几天都没有放假，实在是太忙了，但是想到陈子昂上一次帮了他们，觉得还是去一下。
<br/>
<br/>        “二哥，你在西城吗？如果在西城的话，晚上一起吃饭。”陈子昂看见江俞琪说晚上一起吃饭就又给二哥陈子建打了个电话。
<br/>
<br/>        “子昂，行，在哪里吃？”
<br/>
<br/>        “我等会把地址发你。”
<br/>
<br/>        陈子昂又依次给呼延寰宇和李凌菲打了电话，告诉他们晚上一起吃饭。
<br/>
<br/>        约好了众人，兄妹俩又赶着去买了一个十英寸的双层蛋糕，他们又赶到酒店帮着服务员布置了一下包间。
<br/>
<br/>        一切准备妥当，陈子昂和陈子霖在饭店等着众人的到来。
<br/>
<br/>        最先到的是呼延寰宇，下来是李凌菲，再下来是江俞琪姐弟，二哥陈子建和二嫂冯睿刚到一会，乔小东就到了。
<br/>
<br/>        “你们都比我早啊，我最近来得最晚，真是不好意思。”乔小东有些不好意思。
<br/>
<br/>        “正因为我们远所以要提前到啊。”陈子霖笑着说。
<br/>
<br/>        其实离指定的时间还有二十分钟，大家都是提前来的，乔小东也不算来得迟。
<br/>
<br/>        “来，我给大家介绍一下。”陈子昂来介绍大家认识。
<br/>
<br/>        “乔小东，乔总。”
<br/>
<br/>        “这个是某江集团的两位老总，江俞琪、江俞轩。”
<br/>
<br/>        “乔总好！”
<br/>
<br/>        “俩位陈总好。”乔小东和江俞琪、江俞轩分别握手。
<br/>
<br/>        “这个是我二哥陈子建，这个是我二嫂冯睿。”乔小东又和陈子建夫妇分别握手。
<br/>
<br/>        “其余的我就不用介绍了吧，乔总都认识的。”
<br/>
<br/>        “哈哈，认识，认识的。”
<br/>
<br/>        “那就就座吧，我们边吃边聊。”
<br/>
<br/>        “来，乔总，你坐这。”陈子霖安排座位。
<br/>
<br/>        “不。陈二哥年龄大，陈二哥坐那边。”乔小东推脱。
<br/>
<br/>        “哈哈，今天这个位置非你莫属了。”陈子霖将乔小东推到首位坐下。
<br/>
<br/>        江俞琪坐在乔小东的左手位，陈子建坐在乔小东的右手位，江俞轩挨着江俞琪坐，冯睿挨着陈子建坐。
<br/>
<br/>        陈子昂挨着江俞轩坐，呼延寰宇挨着冯睿坐，李凌菲挨着陈子昂坐，陈子霖坐在呼延寰宇和李凌菲的中间，下首位。
<br/>
<br/>        陈子霖对于活跃酒桌上的氛围那叫一个杠杠的。大家都场面上过来的人，不一会就都已经融入到一块。
<br/>
<br/>        准备上菜之前，突然包间断电了。
<br/>
<br/>        “怎么会没电了呢？”乔小东感觉很奇怪，像这种档次的酒店一般情况下是不会停电的。
<br/>
<br/>        门被推来，陈子昂亲自推着装着生日蛋糕的推车，唱着生日歌。
<br/>
<br/>        “乔大哥，生日快乐！”
<br/>
<br/>        大家都站起来拍着手唱着生日歌为乔小东庆生，二嫂冯睿才明白为什么没有让陈子建坐上首位，心里的哪一点不快也就退去了。
