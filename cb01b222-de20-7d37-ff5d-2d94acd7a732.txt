　　　　虚家，便是属于北邙学院附属的一些家族之一，在斗玄国度，称王称霸的存在。
<br/>　　每次考核，虚家也会得到许多考核名额，凭借这些，足以让虚家强盛。
<br/>　　楚风眠听到拓跋鼎的话，开口询问道。
<br/>　　“这虚家中，有没有什么强者？”
<br/>　　“虚家之中，有着一尊生死境二重的老祖宗，力量指数达到三十，算是一尊小有名气的强者。”
<br/>　　拓跋鼎如实禀告到。
<br/>　　“生死境二重……”
<br/>　　楚风眠目光闪烁，似乎是在思考了什么。
<br/>　　随后他脸色便是一变，看向那虚堇开口道。
<br/>　　“那好，若是虚家愿意将考核名额交给我们，我们也不会白白索要的，可以给予你们一部分财富。”
<br/>　　楚风眠开口的同时，手一挥，在楚风眠的手心中，却是突然出现了一道灵药长河，其中便是无穷无尽的九鼎丹，足足有着百亿九鼎丹之多。
<br/>　　“恩？”
<br/>　　虚堇的表情，听到楚风眠的话，本来还是不以为然。
<br/>　　毕竟在她看来，楚风眠的这一行人，也不过就是在一些破落小国度中出来的人物，能有什么财富。
<br/>　　可是当看到这百亿九鼎丹的时候，虚堇的脸色却是一下子变了，眼神中尽是贪欲。
<br/>　　百亿九鼎丹，对她来说可是一笔巨大的财富了，她虽然是来自与虚家，但是虚家中，也是有着无数杰出弟子。
<br/>　　除非是他们虚家的家主，不然谁都不可能一次性拿出如此多的财富来。
<br/>　　虚堇的脸色微微变色后，很快便是恢复了平静，脸上尽是笑意道。
<br/>　　“诸位救了我，等诸位跟随我回到家族之后，这考核资格的名额，自然会给各位的。”
<br/>　　“还请各位，跟我来吧。”
<br/>　　虚堇说了一句后，便是带着楚风眠，直接继续飞了过去。
<br/>　　虚家所在的斗玄国度，便是就在北邙学院附近，所以路线一样，楚风眠一行人先是来到了宣州古城，便是在利用传送法阵，跨越了无数国度。
<br/>　　终于是在七日之后，来到了一处巨大无比的城池之前。
<br/>　　这一方城池，便是斗玄国度的都城，斗玄城。
<br/>　　虚堇背后的虚家，便是位于此地。
<br/>　　这一座城池，比起一个武胜国度都要庞大，一眼望去，在城池上方，隐隐约约有着无数波纹在波动。
<br/>　　一股股灵力波动，从中传了出来，这斗玄城中，有着许许多多的禁制，阵法，来守护。
<br/>　　跟那泉城国度相比，都相差不多。
<br/>　　楚风眠一行人，直接降落在了城池之中，
<br/>　　两尊斗玄城的看守，看到楚风眠一行人落下，几人都是迎了过来。
<br/>　　这两尊斗玄城的看守，都是有着御风境的实力，不过资质一般，终生只怕都无法踏入神力境成就圣者。
<br/>　　“来者何人！擅闯城门！站住！”
<br/>　　这两尊看守，大声厉喝道。
<br/>　　“两个废物，就连本小姐都认不出来了？”
<br/>　　虚堇表情一寒，口中冷冷的说道。
<br/>　　“都给我滚开！”
<br/>　　“是堇小姐。”
<br/>　　“堇小姐回来了。”
<br/>　　那两尊看守，看清楚虚堇的样子，脸色突然大变，一个个脸色苍白，极为紧张起来。
<br/>　　“我们有眼无珠，没有认出堇小姐来，还请堇小姐赎罪。”
<br/>　　看起来这两尊看守，也是极为惧怕这虚堇，直接跪倒在地，祈求了起来。
<br/>　　“哼，今日我带着贵客上门，就不跟你们计较了。”
<br/>　　虚堇挥了挥袖子，便是直接带着楚风眠一行人走入了斗玄城中去，没有在理会那两尊看守。
<br/>　　进入了这斗玄城，楚风眠一眼望去，便是数不清的人。
<br/>　　在这斗玄城中，贸易却是并没有太过繁盛，起码远远不如泉城古城。
<br/>　　不过在其中，却是有着不少武者，起码楚风眠一眼望去，便是看出了几人，都是有着可以冲击神力境，有着踏入圣者资质的年轻武者。
<br/>　　只不过就算是有着冲击神力境的资质，但是真正可以冲击神力境，成就圣者的武者，却是寥寥无几。
<br/>　　整个斗玄城，仿佛都是练舞之人，人人练就武道，一片斗志昂扬。
<br/>　　“倒是要看看，这斗玄城中，到底有着什么强者。”
<br/>　　楚风眠不动声色之间，一股灵识便是爆发出来，他两世为人，灵识倒是比起寻常武者要强的多。
<br/>　　一瞬间，楚风眠便是将这斗玄城中大半部分，都给扫视了一遍。
<br/>　　这斗玄城中，一共便是只有这一尊生死境强者，应该就是那拓跋鼎所说的，虚家的老祖宗了。
<br/>　　就算是这诺大的斗玄城中，也就只有一尊生死境强者，看来这生死境，却是并没有那么容易踏入。
<br/>　　北邙学院中虽然生死境强者众多，但是那可也是千万年来的累计，在加上整个北域的天才，才培养出来的。
<br/>　　真正一尊生死境武者，可都是百万人中脱颖而出的，百万武者都不一定可以培养的出一尊生死境强者。
<br/>　　“楚公子感觉，这斗玄城如何？”
<br/>　　一边在城中走动，一边虚堇便是向着楚风眠随意询问道。
<br/>　　她的话虽然看似闲聊，不过其中，却都隐藏了几分试探。
<br/>　　以楚风眠的心性，自然一眼看得出来，他脸上故意装出一副连连惊讶的神色，如同刚刚见到市面的土包子一样开口道。
<br/>　　“太繁华了，比起我的家乡要繁华无数倍！”
<br/>　　“果然是偏远之地来的土包子！”
<br/>　　听到楚风眠惊讶的话，那虚堇心中冷笑连连，表情却是极为平静开口道。
<br/>　　“北邙学院所在的北邙城，哪里才是真正的繁华，比我们斗玄城要繁华数十倍，数百倍，未来楚公子进入到了北邙学院中，自然会见的。”
<br/>　　“哦？比起这斗玄城还要繁华数十倍？数百倍？”
<br/>　　楚风眠故意装出一副吃惊的样子来说道。
<br/>　　“不错。”
<br/>　　虚堇点了点头，看向楚风眠的眼神中，更是鄙夷。
<br/>　　如此偏远国度出身，楚风眠一行人还有这如此实力，更是有着这样一大笔财富，一看就是得到了什么奇遇宝藏的暴发户。
<br/>　　她自然便是极为不屑。
<br/>　　不过虚堇的表情，始终都是露出几分淡淡微笑，并没有任何的变化。
<br/>　　“快走吧，我已经通知了父亲，父亲已经在府中等候了。”
<br/>　　说着，一行人便是加快了脚步，向着虚家所在的虚府前去。
