　　钱鑫心中狂喜，拿着手机的手都有些微微颤抖，说道：“三爷，三爷，您能再给我一粒吗？一粒，就给我一粒！您是不知道，我这几天是越来越想要女人，我他妈就是路上看到那些四十多岁的站街女，我都想直接冲上去把她给上了，可、可是我下面它不行啊，怎么刺激都没反应……”
<br/>　　“宝贝儿，再快点。”
<br/>　　三儿把手机开从自己耳边拿开，拍了拍陆岑的头，然后把手机开了免提放在桌子上，对电话那头的钱鑫说道：“小钱啊，不是我小气不给你，但是这东西价格可不便宜，我也是给人打工的，不能平白无故的就这么给你，这坏了规矩，我也不好做不是？”
<br/>　　“三、三爷，您说，需要我做什么？”钱鑫现在是抓住了救命稻草，死死地拽着说什么也不会放手了，说道：“您只要给我一粒，我这条命就是您的，你要我放火我绝对不去杀人，您要我往东我绝对不往西！”
<br/>　　“什么杀人啊，放火啊，嘶……你个坏东西。”
<br/>　　三儿点了点陆岑的额头，沉声对钱鑫说道：“我们是遵纪守法的人，怎么能去做杀人放火的事情吗？想要我手里的好东西，那就拿情报来换。”
<br/>　　“我、我一直都在盯着佟医生那边呢，可是她没什么动静啊。”钱鑫都快要哭了，这不是他不愿意干活，是根本没机会啊。
<br/>　　“既然这样，那我也没办法。”三儿咬紧牙关，强压下缴械的冲动，对钱鑫说道：“你要的东西，我这里有，但是没有我想要的东西，你想也不要想！”
<br/>　　“三爷，三爷，您就给我一粒吧，求求您了。”钱鑫在电话那头对着三儿哀求。
<br/>　　可是三儿这个时候已经到了爆发的边缘，哪里还顾得上钱鑫的哀求，怒声说道：“我话已经说得很清楚了，想要那天晚上的东西，你就拿东西来换，否则想也不要想！”
<br/>　　说完，三儿就伸手挂断了放在桌子上的手机，然后按住陆岑的头，不让她动弹，在她嘴里喷涌而出。
<br/>　　三儿舒爽的舒了口气，松开陆岑的头，靠在椅子上，任由陆岑拿着纸巾给自己进行清理。
<br/>　　陆岑把嘴里的东西吐在纸巾上，扔进垃圾桶，拿起桌子上的矿泉水漱了口，顺势坐在三儿大腿上，腻声问道：“三爷，您什么时候看上那个什么佟医生了？是想玩制服诱惑？”
<br/>　　“吃醋了？”三儿捏了捏陆岑的下巴，说道：“这是力哥交代的事情，具体是什么事情你也别问了，我知道的不多，就算是知道了，我也不能说。层面太高，不是我们这种人能接触到的，稍有不慎就会粉身碎骨！”
<br/>　　“能让你三爷都说层面高的事情，我还真是有点好奇呢。”陆岑抱着三儿的脖子，说道：“您就给我说说，我保证不说出去。”
<br/>　　三儿取出一支烟，让陆岑点燃，吸了两口，想了想，说道：“其实我知道的也不多，只知道力哥能有现在的发展是因为他上面有人，而且是国外的大势力，跟国外的大势力相比，我们这种最多只能算萤火之光。对上面的势力，力哥很忌讳，说得并不多，我记得有一次力哥提起的时候，称她‘毒皇后’！”
<br/>　　“‘毒皇后’？”陆岑皱了皱眉，说道：“是个女的吗？”
<br/>　　“不知道，力哥也很少提及。”三儿摇了摇头，回答道：“上次我让你给钱鑫那小子用的‘水晶珠’，就是力哥从上面得来的，能研制出‘水晶珠’这种东西的势力，有多强大，不用我多说，你也应该能明白。”
<br/>　　“说起这‘水晶珠’真是出乎我的意料，刚才钱鑫在电话里的反应你也听到了，现在这个时候，如果我拿出一粒‘水晶珠’来，让他帮我去杀人，他也会毫不犹豫的去做。”三儿的手伸进陆岑的衣服中乱摸，想到最近场子里“水晶珠”的销售，忍不住感慨道：“目前我严格控制了场子里‘水晶珠’的销售，并且把每颗的价格提升到了六千块，依然是遭到抢购。本来我所说的拍卖会还不至于这么快就推出，但现在我觉得时间差不多了。说实话，我看到钱鑫用完‘水晶珠’后的效果，我差点就没忍住，这东西对男人的诱惑实在太大了，不过还好我没用，不然我就跟钱鑫一样变成废人了。”
<br/>　　“唉哟~”
<br/>　　陆岑腻在三儿身上，说道：“三爷您这么厉害，哪里还需要用什么‘水晶珠’啊，您要是用了‘水晶珠’，让我们这些女人可怎么办，还不得被您给折腾死？”
<br/>　　“你个小妖精，刚把你喂饱，这就又想了？”三儿手中用力捏了两下，说道。
<br/>　　“那是上面，还有下面呢。”
<br/>　　“你自己来。”
<br/>　　很快，房间里就传来一阵阵男人沉重的喘气声，还有女人拼命压抑后的呻吟声……
<br/>　　三儿跟陆岑在房间里快活的时候，钱鑫却还是在苦苦煎熬着，对于钱鑫来说，他现在简直就是度日如年，钱鑫觉得自己如果再不找个女人发泄，自己就要爆炸了！
<br/>　　“你这次什么时候回部队去？”
<br/>　　吃过晚饭，慕容飞陪着佟千雪出去散步，从门卫处经过的时候，佟千雪询问起慕容飞何时返回部队。
<br/>　　“这次应该能多留几天。”慕容飞想了想说道：“你下个月去T国还是要多当心一点，毕竟你现在可不是一个人。”
<br/>　　“行了，我知道了。”佟千雪无奈的翻了个白眼，说道：“我发现你现在怎么比我妈还要唠叨呢，我妈知道我要去T国后，不仅把我姐训了一顿，还对着我进行唠叨轰炸，现在她消停了，你又来了是吧？”
<br/>　　坐在门卫处窗口的钱鑫正想着要怎么再从三儿手里弄一粒宝贝来的时候，听到佟千雪和慕容飞的对话，眼前顿时一亮，真是瞌睡了就有人送枕头啊。
<br/>　　“佟医生，又出去散步啊？”钱鑫站起身来对佟千雪问道。
<br/>　　“是啊。”佟千雪微笑着回答，发现钱鑫脸色很不好，问道：“小钱，你是生病了吗，脸色这么难看。”
