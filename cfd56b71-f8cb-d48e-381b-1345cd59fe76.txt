　　维尔纳特虽然因为断腕失血过多，加上受到情绪的影响，一时间气急攻心晕了过去，但很快就被人救醒。
<br/>　　醒来之后的维尔纳特并没有立刻去医院接受手术，将断掉的手掌续接回去，而是选择了加入到追击队伍中，对战术中队展开追击。
<br/>　　这次行动是“黑箭”小队的滑铁卢，也是维尔纳特军旅生涯中的耻辱，唯一可以洗脱这个耻辱的就是把战术中队的那些人重新再抓回来，拿到他们手中的P病毒。
<br/>　　如果可以把P病毒控制到自己手中，维尔纳特一定可以免于被强制退役然后寻找薪水低微的快递工作，说不定他还能被当做英雄，即便是不在作战部队服役，他也能在军队中获得一个光线的职位，而到了那个时候，他的断手就成了他最好的勋章！
<br/>　　这是维尔纳特摆脱force-recon队员退役或者阵亡后，得不到政府承认抚恤的唯一办法，所以维尔纳特必须要赌这一把，他别无选择！
<br/>　　维尔纳特将一部单兵PDA放在自己大腿上，在吗啡镇痛效果下，暂时压制住断手的剧痛，手指在单兵PDA上滑动着，想要找出战术中队的逃脱路线。
<br/>　　现在已经明确罗昊、齐凡他们会选择利用那架“支奴干”运输机逃脱。而要用这种办法逃脱，就必须要给“支奴干”运输机提供一个降落的位置，太过狭窄的街道肯定是不行的，因为一旦运输机碰撞到周围的建筑就是机毁人亡的结果，所以必须是宽敞的路面。
<br/>　　宽敞的路面，那到底是在什么地方？维尔纳特让自己的汽车紧咬着前方的两辆防弹突击车，在单兵PDA上寻找着，只要能找到战术中队打算在什么地方撤离，他就能让人提前绕路过去堵住他们的撤离路线。
<br/>　　“圣地安列斯大桥！”
<br/>　　维尔纳特在看到圣地安列斯大桥的时候，眼睛顿时一亮！这座大桥长5.5公里，八车道，足够宽敞让“支奴干”直升机进行降落，让那两辆防弹突击车冲上飞机。
<br/>　　而他们现在所行驶的方向，正是往圣地安列斯大桥所去，正更加确认了维尔纳特的猜测。
<br/>　　“立刻让一个小队绕路去圣地安列斯大桥！”维尔纳特大声对着坐在副驾驶上的force-recon队员吩咐道：“他们打算在圣地安列斯大桥上汇合，一定要赶在他们抵达圣安地列斯大桥之前，拦截住他们！”
<br/>　　维尔纳特尽管断了一只手，而且状态也不是很好，但是他的决策能力还在。在force-recon中没有人会小看维尔纳特在战场上的决策能力，也正是因为维尔纳特出色的决策能力才让得“黑箭”小队有着不败的战绩，打下了赫赫威名！
<br/>　　坐在副驾驶上的force-recon队员立刻通过无线电吩咐两辆车绕路去圣地安列斯大桥，打算拦截住战术中队的撤离路线！
<br/>　　天蛇、玄龟猛踩着油门，只是在转弯的时候才轻轻松开油门，可即便是松开了油门，可是在高速下产生的强大离心力还是把汽车里的人狠狠地往车门上挤压。
<br/>　　两辆防弹突击车引擎发出沉重的声响，这是引擎工作到极限才会有的声音，如果继续保持着现在这个状态，让引擎超负荷的运转，很有可能出现爆缸，使汽车彻底失去动力的情况，可是到了现在不管是天蛇还是玄龟都顾不上这些了，依旧死死地踩着油门，逼迫着突击车继续向前冲刺，并且在心中暗暗祈祷突击车在平时没有缺乏保养，能再坚持一会儿，坚持到他们冲上圣地安列斯大桥！
<br/>　　Force-recon两辆突击车因为绕路的关系，竟然抢在了天蛇和玄龟之前赶到了圣地安列斯大桥，数名force-recon队员拿着自动步枪对着笔直冲来的防弹突击车进行扫射；可是子弹打在突击车上，除了在防弹玻璃上留下几个不痛不痒的白色印痕，落在车身上发出“铿铿铿”的闷响外，并没有对两辆突击车造成致命的威胁。
<br/>　　“把大桥升起来，阻止他们过桥！”
<br/>　　一名force-recon队员看到天空中的“支奴干”运输机又是开始下降高度，立刻吩咐升起大桥，打算以此来阻止战术中队过桥；而这的确是最有效的办法，一旦大桥被升起，就算是有重火力把升起的大桥轰开，他们也没办法成功登上“支奴干”运输机！
<br/>　　圣地安列斯大桥从中间位置断开，两侧大桥被钢索缓缓拉起。
<br/>　　张子航一把转过车载机枪的枪口，对着站在控制大桥升降闸前的force-recon队员扫出一串子弹，直接将他打成了筛子，可子弹穿过那名force-recon的身体也对升降闸造成了破坏，直接造成了升降闸无法停止工作，大桥继续保持着升起的状态。
<br/>　　“妈的！”天蛇嘴里咒骂了一声，继续加速冲向已经呈四十五度角升起的大桥，只要速度够快，借助道桥升起的斜坡，是完全可以从大桥上飞过去的，只要能从大桥上飞过去，那些force-recon队员就只能望洋兴叹了，再也拿他们没有任何办法！
<br/>　　而此时突击车的转速表和时速表指针全都已经进入了红区，而且汽车引擎沉重的喘气声也变得越发明显，也许就是在下一秒，引擎就有可能发生爆缸，彻底失去引擎。
<br/>　　在高速情况下发生爆缸，汽车就会失控，情况是十分危险的。
<br/>　　玄龟驾驶的防弹突击车冲在天蛇那辆防弹突击车前面，一马当先直接飞过了升起的大桥，加速冲上了“支奴干”运输机放下的尾部舱门，冲入到机舱中。
<br/>　　“快点，再给我快点！”
<br/>　　天蛇这个时候已经喊出了声音，看着越升越高的大桥，心中焦急无比。可就是在这个紧要的时刻，这辆防弹突击车却是不打算要给他这个面子，防弹突击车的引擎发生了爆缸！
<br/>　　瞬间突击车转速表和时速表的指针迅速回落，在告诉情况下突然失去动力的防弹突击车也是在大桥上走起了蛇形步。
<br/>　　吱——！
<br/>　　在天蛇拼尽全力的控制下，防弹突击车的轮胎跟桥面发生剧烈摩擦。
