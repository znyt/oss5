　　花费了一些时间，三人回到义庄，九叔正在停尸间里，不知道鼓捣些什么。
<br/>　　秋生带着两人走了上去，拿出已经灭掉的香递给九叔，文才将现场的情况如实交代了一遍。
<br/>　　“有鬼有尸，大凶之地。”九叔握着三炷香，皱起了眉头，在迁坟的时候，他就感觉有些不太对劲了，如此看来，那块陵园恐怕不是什么善地。
<br/>　　扭头看了一眼从山上取回来的棺材，九叔继续说道：“人最怕三长两短，香最忌两短一长，偏偏就烧成这个样子，家中出此香，必定有人丧！”
<br/>　　他有些担心，任家今晚或许会出什么事情。
<br/>　　“是不是任老爷家里？”文才有些疑惑的开口，他不懂香阵，但对于师父的话，他是绝对信服的。
<br/>　　狠狠瞪了文才一眼，九叔有些不满的开口：“难不成是这？”
<br/>　　讪笑了一下，文才也没有将此事放在心上，反正这件事和他没什么关系，只要义庄不会出事就好了。
<br/>　　“那任老爷的女儿，会不会有事啊？”看着九叔，秋生有些担忧的说道。
<br/>　　最近几天，他和任婷婷相处的不错，关系也在逐渐升温，他也不想任婷婷出了什么事情。
<br/>　　“哎呀，总之姓任的马上就要有难了。”文才打断了秋生的话语，随之一愣，像是想到了什么。
<br/>　　瞥了两人一眼，刘哔没有理会两人，径直走到了九叔身边，看着九叔站在棺材旁边，不知道在想些什么。
<br/>　　“师父，有什么办法能让任家避免这次意外发生？”犹豫了一下，刘哔开口，这件事情，可是关系到他的任务。
<br/>　　最起码在今晚之前，他不会让任家发生任何意外，等任务结束，任家人的死活，就和他没什么关系了。
<br/>　　“香烧成这样，是必死的局，基本很难破解。”九叔双手抱胸，回答道：“但也不是完全没有办法，要不我也不会让人把棺材抬回来。”
<br/>　　文才和秋生两人，此时也停止了争吵，纷纷来到九叔身边，一副慌慌张张的样子。
<br/>　　“师父，是不是这幅棺材有问题？”指了指身前的棺材，文才大概清楚了自己师父的想法。
<br/>　　点了点头，九叔伸手，点了点棺木的边缘，道：“棺材没问题，是死人有问题。”
<br/>　　说罢，径直从棺木旁离开，向祭品台走去，他要趁着现在，去准备一些东西。
<br/>　　三人站在棺材旁边，看着棺木，心中微微有些发寒，为了证实心中的想法，刘哔伸手，缓缓拉开了棺盖，陡然心中一惊。
<br/>　　此时棺材中躺着的老人，已经不是白天时的那种样子，不知是不是错觉，整具尸体都要胖了一圈，更是有尖锐的獠牙，从其口中露出来了一截。
<br/>　　更让人触目惊心的是，这具尸体的双手，长满了乌黑的指甲，看起来无比锋利，在烛光下反映出一抹不正常的光亮。
<br/>　　“师父，你快过来看看，尸体好像发福了！”文才往后退了一步，惊慌的向九叔喊道。
<br/>　　一旁正在摆弄蜡烛的九叔，听见文才的呼叫，连忙迎了上来，看见棺材中尸体的变化，脸色一变，伸手拉住了棺材盖。
<br/>　　“准备纸笔墨刀剑。”面色有些凝重，九叔如临大敌的盯着棺材，他怕有什么意外发生。
<br/>　　听见九叔的吩咐，三人不由得一愣，皆是有些懵逼的开口：“什么？”
<br/>　　“黄纸，红笔，黑墨，菜刀，木剑！”无奈的重新重复一遍，九叔说罢，长叹了一口气。
<br/>　　他也是挺服气这几个徒弟的，刘哔刚加入不知道也就算了，文才和秋生跟了自己这么长时间，居然连这些基础的东西都没记住。
<br/>　　三人看师父有些生气，也没敢在此时触什么霉头，纷纷走出了义庄，找齐了九叔需要的东西。
<br/>　　片刻的时间，三人回到停尸间，看到九叔抓了一只公鸡，站在祭台旁边。
<br/>　　这只鸡原本是养在九叔房间里的，和别的鸡不同，九叔很早就将它的声带摘除，故此刘哔一直都不知道，家里还有养了这么一直公鸡。
<br/>　　按照九叔的说法，摘除掉声带，是为了防止鸡体内的阳气外泄，这样的话，用来做法事效果会更好一些。
<br/>　　故此，在房间里养几只鸡，这是九叔一直以来的习惯，这样的话，需要的时候，也不用出去找鸡了。
<br/>　　看到九叔手中的攻击。秋生也是一愣，不过也没多说什么，而是手中握着菜刀，伸手在公鸡的脖颈处，划开了一道口子。
<br/>　　因为声带被摘除，公鸡发不出一丝声音，只能剧烈的在九叔手中扑腾着，看上去，让人莫名感觉有些阴森。
<br/>　　将鸡血滴在一个瓷碗中，九叔伸手，粘起一粒糯米，将手指放在了蜡烛上，随后，他的整个手指，都诡异的燃烧了起来。
<br/>　　但九叔像是感受不到疼痛一般，对自己的手指不管不顾，而是将点燃的糯米，扔进了盛放着鸡血的瓷碗中，又打开墨斗，倒出了一些黑墨，搅拌均匀后，才将自己手指上的火苗熄灭。
<br/>　　之后，九叔用一个木质八卦，盖在瓷碗上面，拿在手里掐了一个决后，将里面的液体，缓缓倒入了一个特质线盒之中。
<br/>　　“这是什么东西啊？”文才杵了杵秋生，一脸好奇的问道，一旁的刘哔也竖起了耳朵，他也不知道这到底是什么。
<br/>　　瞥了一眼文才，秋生随口说道：“这叫墨斗，师父经常用它来对付僵尸，很有用的。”
<br/>　　恍然大悟的点了点头，文才继续盯着场中，此时，九叔的法事已经快要结束了。
<br/>　　只见九叔用红笔在黄纸上画了一个符咒，随后用木剑挑起，放在蜡烛上点燃，将整个火团都扔进了墨斗里，待完全熄灭之后，拿着交给了秋生。
<br/>　　“去，弹在棺材上面。”看到三人接过墨斗，九叔站在旁边看了看，嘱咐道：“整福棺材都要弹上线，千万不要有什么遗漏的地方。”
<br/>　　三人应了一声，拉着染上墨斗的白线，在棺材上面弹出一个个规则无比的正方形。
