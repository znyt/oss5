        傅氏集团旗下的高管公寓。
<br/>
<br/>        裴金澈正慢条斯理地系上皮带时，他身边的大床上，一条雪白的藕臂将被子掀开，手里还拿着手机，随之那诱人的身躯也跟着坐了起来。
<br/>
<br/>        夏玄灵完全不顾自己的一丝不挂，而是双手捧着手机，看着屏幕上的信息，难以置信地瞪大了眼睛：“童筱颖真的死了！”
<br/>
<br/>        裴金澈拿起床上的衬衫淡定地穿上，面无表情地应和了一声：“恭喜你啊！这下你可以继续去追傅夜沉！”
<br/>
<br/>        “裴金澈，你积点口德！”夏玄灵没好气地白了裴金澈一眼。
<br/>
<br/>        裴金澈这才言归正传地在夏玄灵的床边坐下：“你的养父夏阳枭，和你的干哥哥傅夜沉，现在正在明争暗斗，你支持谁？”
<br/>
<br/>        “他们两在斗什么？”夏玄灵诧异。
<br/>
<br/>        自从她为了自己的咖啡馆，同意当他裴金澈的情人以后，裴金澈对她的态度一点都没转变。
<br/>
<br/>        一如既往的毒舌，三句话不到，就能把她说得够呛。
<br/>
<br/>        “现在的局势，不大好。”裴金澈凝了凝眉头，接着说道，“我是给傅夜沉做事的，而你现在又是我的秘书。如果，你帮你养父。我得将你从我身边踢走。”
<br/>
<br/>        “看样子，你是不打算跟我玩阴的，而是打算直来直往喽？”夏玄灵咧嘴一笑。
<br/>
<br/>        裴金澈一脸严肃地看着夏玄灵：“因为，我不想防着自己的女人。”
<br/>
<br/>        “我只帮你。”夏玄灵放下手机，从被子里出来，双手勾着裴金澈的脖子，接着说道，“这个回答，你满不满意？我是认真的。”
<br/>
<br/>        “当然。”裴金澈面无表情地回答。
<br/>
<br/>        夏玄灵仰起头来，吻了吻裴金澈那微凉的薄唇，关问道：“你最近到底做什么去了？脸色很差。你是……给谁献血了吗？”
<br/>
<br/>        “我是去献血了，所以，你再给我补补血。”裴金澈勾了勾唇，俯身将夏玄灵压倒在床。
<br/>
<br/>        不同于昨晚的疯狂，此刻的他，吻落在她细致的肌肤上细碎而缠绵。
<br/>
<br/>        随着裴金澈的唇逐一滑过的地方，夏玄灵的身体不由自主地轻颤，一颗心也跟着逐渐沉沦。
<br/>
<br/>        只是在他覆在她身上，她不经意间地偏头，真真切切地看到他手臂上的针孔时，理智猛然被拉回。
<br/>
<br/>        “你真去献血呢？给谁献血？”夏玄灵惊愕，又有些心疼。
<br/>
<br/>        裴金澈俯下身去，俊脸埋在了夏玄灵的颈窝里，呢喃细语：“给真正的‘夏玄灵’献血。”
<br/>
<br/>        “真正的‘夏玄灵’？你是说，我爸爸他找到那个……”夏玄灵身心一怔。
<br/>
<br/>        裴金澈抬起手来，伸出食指，抵在了夏玄灵的唇上。
<br/>
<br/>        “我只不过是个实验品，那个真正的‘夏玄灵’也一样。我们都是实验品。你父亲真狠！拿自己的亲生女儿做实验！呵！”
<br/>
<br/>        “阿澈，你可不可以……带我见一见，那个真正的‘夏玄灵’？我顶着她的身份，当了这么多年的夏家少小姐，也是时候，该还给她了。”夏玄灵黯然神伤道。
<br/>
<br/>        裴金澈不以为然地回答：“等时间到了，你养父自然会让她认祖归宗。”
<br/>
<br/>        “等等！”话题到这里，夏玄灵忽然间意识到一个严重的问题，“你不是有快速自愈的能力吗？为什么，你手臂上还留有针孔？”
<br/>
<br/>        “……”裴金澈此刻沉默了。
<br/>
<br/>        夏玄灵神经紧绷起来：“你……不对！快告诉我，这段时间，在你身上到底发生了什么事情？”
<br/>
<br/>        “如果，真正的‘夏玄灵’能醒过来，就证明，他们研究了这么多年的实验便成功了。如果不能醒过来，就代表他们的实验失败了。”裴金澈苦笑着翻了个身，将夏玄灵拥入怀中，“所以，这就是我和她存在这个世上的意义。”
<br/>
<br/>        “那从今以后，我会成为你存在在这个世上唯一的意义！”夏玄灵下意识地往裴金澈的怀里钻了钻。
<br/>
<br/>        她心知肚明，哪怕他对她态度不好，却是她咎由自取。
<br/>
<br/>        十五岁时，她对自己的干哥哥傅夜沉情窦初开，懵懵懂懂以为那就是爱，在十八岁那年，她还想向傅夜沉表白，但是因为醉酒误事，错上了裴金澈的床。
<br/>
<br/>        夏玄灵一直以为自己喜欢的人是傅夜沉，其实不然，直到后来，她才发现，自己一直喜欢的是这个和哥哥夏凌焕一起接她送下学，却从来不肯主动跟她搭话的裴金澈。
<br/>
<br/>        不然，那年冷夏两家的团圆宴，他和傅夜沉一起留宿她家，她明知道对面才是傅夜沉住的房间，却偏偏跑到裴金澈的屋里去献身？
