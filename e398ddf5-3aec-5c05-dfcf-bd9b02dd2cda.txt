　　第21章
<br/>　　其实在知晓黄三元经历后，程大雷已经决定收留对方。
<br/>　　韩虎踞生前说不好有多少田地，又有多少货物往来，各方面关系要打理，这些可都是靠黄三元管理。放到程大雷前世，黄三元起码等于某五百强企业的总经理。这是个人才呐，现在人家离职来到自己这个初创的小公司，自己还有什么资格挑三拣四。
<br/>　　而蛤蟆寨目前还真需要黄三元这号人，若拿一个公司来比，秦蛮和林少羽属于技术工种（打劫），徐灵儿属于后勤（喂马），程大雷这个空降的董事长半点经验没有，正需要一个职业经理人。至于徐神机……好吧，他属于蛤蟆寨的吉祥物。
<br/>　　却是没想到，随便摆摆架子，就真炸出宝物来了。
<br/>　　“什么藏宝图？”程大雷兴奋道。
<br/>　　“是城主生前收藏的，我从黑石城逃出来时拿到的。”说着黄三元从怀里掏出一份羊皮地图，递给程大雷。
<br/>　　“三十年前，戎族入侵大武，在帝国境内烧杀抢夺，后来他们被赶跑时，掳走许多财物，这些地方财物一时带不走，便就地埋了起来，这份图上就是他们的藏宝地点。”
<br/>　　程大雷入手羊皮地图，就感觉心情激动，没想到自己还能过过挖宝的瘾，而且是羊皮卷，多么有质感。
<br/>　　“这份地图上的地点是哪里？”程大雷将地图摊开。
<br/>　　“不知道。”
<br/>　　“什么，你不知道？”
<br/>　　“当然不知道，要是有人知道，不早就挖出来了。”黄三元一本正经道。
<br/>　　摊开地图的时候，程大雷才知道麻烦在哪里。地图上是山脉和河流的走势图，中间有一圆圈。可问题在于，上面一个字都没有，至于这山，这河在哪里……恐怕鬼都不知道。
<br/>　　这个世界的地图精度本来就差，更何况是连文字都没有的戎族，程大雷也不能说拿着这份藏宝图四处比对山脉河流的特点。目前唯一的线索就是知道这是戎族撤退时的藏宝地，简单来说，可能藏宝的地点根本就不在帝国境内，也可能就在青牛山方圆，随便挖一铁锨就挖到了，可是这概率，不比中彩票大多少。
<br/>　　激动的心情冷下来，程大雷把藏宝图放到一边，道：“这份图……真的差点意思，黄管家，你要迅速证明自己的价值。对了，眼下山寨的确有件麻烦时，算是给你的考核，目前你就算山寨的临时工，如果你能完成这件事，我就给你转正。”
<br/>　　黄三元听不懂什么叫临时工，他是真没想到落草当山贼也这么难，可现在自己也没有别的地方去，只好问：“什么事？”
<br/>　　程大雷将那批马的事情说了，黄三元听罢之后几乎就是不假思索。
<br/>　　“以山寨的长远发展来看，这批马的价值很大，可对于目前的山寨，反而是累赘。依我看，越早出手越好。不知大当家在落叶城有没有说的上话的商户，如果没有的话，只能卖给附近别的山寨。”
<br/>　　“军师，你听见了没有？”程大雷冷喝一声，心道看看人家，这主意几乎信手拈来，要你有什么用。
<br/>　　“我一直在听。”徐神机点点头：“这个法子与我不谋而合，我正准备说呢，反而是被他抢了先，这正是，啧啧……英雄所见略同。”
<br/>　　程大雷郁闷得吐血，还真是从未见过如此厚颜无耻之人。之后，大家又一起商量了具体实施的方法。
<br/>　　有办法后程大雷心情痛快许多，他拍拍黄三元的肩膀：“你做的不错，我感觉你很快就能转正，继续努力喔。”
<br/>　　“嗯。”
<br/>　　黄三元兴奋的点点头，忽然一楞：咦，自己这种莫名其妙激动的感觉是怎么会事。
<br/>　　……
<br/>　　青牛山，飞虎寨。
<br/>　　“我的马喔！”高飞豹盯着自己面前的独眼马发出一声长叹。
<br/>　　“老二，你的那匹马已经瞎了，以后也不能上阵，只能给山寨拉磨用。你也别太难过，以后有机会，我帮你找一匹更好的。”
<br/>　　“怎么，大哥你要把你的马让给我，兄弟情深呐，真的是让我太感动了。”
<br/>　　“我只能说你想多了。”高飞虎。
<br/>　　在任何一个山寨，马匹都是稀缺资源。以飞虎寨在青牛山的实力，整个山寨也不过有五匹马而已。每人把马看得比自己媳妇都要金贵。高飞豹的马被秦蛮一箭射瞎，这几天他如同丧偶一般，整个人都憔悴许多。
<br/>　　铛铛！
<br/>　　“好消息，好消息！”
<br/>　　这声音突然从飞虎寨外面传来，高飞虎和高飞豹都是一激灵。
<br/>　　“大哥，是不是有人来偷袭我们山寨！”
<br/>　　“走，出去看看！”
<br/>　　寨门打开，飞虎寨一伙人听声音走出山寨，只见山林中空空荡荡，一个人影都没有。而周围几棵大树的树干上贴着一张张白纸。
<br/>　　“咦，这是什么？”高飞虎皱起眉头，盯着纸上开头两个大字：“广……告……”
<br/>　　不远处的大树后，程大雷手里拎着铜锣，林少羽拎着浆糊，两个人鬼鬼祟祟凑在一起。
<br/>　　“你那边贴完了嘛？”
<br/>　　“贴完了，你呢？”
<br/>　　“走走，去下一个山寨。”
<br/>　　高飞虎一伙人凑在大树下，盯着上面的白纸发愣。
<br/>　　“上面写的是什么呀，大当家快给我们念念。”
<br/>　　“对呀，告诉我们写的是什么。”
<br/>　　原谅一群大老粗，整个飞虎寨认字的人也不多，高飞虎本人是认字的。此时他盯着面前的『广告』，脸上的表情颇堪玩味。
<br/>　　『好消息，好消息！
<br/>　　你还在为没有座骑而困扰吗，你还在因为别的同伴骑在马上，而你只能步行而自卑嘛！
<br/>　　在这个江湖，没有马会有那个妹子会多看你！
<br/>　　心动不如行动，比你快的人已经载着你中意的妹子奔驰在原野上，你还在等什么！
<br/>　　好消息，本山寨有一批名贵战马限量出售，数量有限，先到先得。
<br/>　　注：持此宣传单页可享受八折优惠。
<br/>　　另注：活动期间更可以享受折上折喔』
<br/>　　歪七扭八的笔迹，稀奇古怪的话，可是自己这种蠢蠢欲动的心情是怎么会事。高飞虎陷入一种深深的错位之中。
<br/>　　“老二，你在干什么，你为什么要把它揭下来？”高飞虎。
<br/>　　高飞豹这个时候已经动手揭树上的宣传单，他一边动手一边回过头来，眨着眼睛道：“大哥，折上折喔。”
