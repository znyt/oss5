　　在三秒钟时间内，使用急速射解决掉五个全副武装的影子部队队员，对于罗昊、李毅峰、李然、慕容飞这四个完全就是用子弹喂出来的玩枪高手来说，这并不存在什么困难。
<br/>　　用赵天宇的话说，费了几吨的子弹，就是条狗都成神枪手了，你们要是达不到我的要求，就连一条狗都不如！
<br/>　　四个人分成两组，分左右两侧快速移动向前推进，在距离别墅大门还有差不多一百米的时候，停了下来。罗昊将G36C自动步枪的快慢机拨到半自动模式，将全息瞄准镜中的红色小圆点移动到距离他最近的那个影子部队队员脑袋上，对着空气导管耳麦说道：“听我口令，三，二，一，放！”
<br/>　　“噗！”
<br/>　　“噗！”
<br/>　　“噗！”
<br/>　　“噗！”
<br/>　　在罗昊话音落下的瞬间，四个人同时扣动扳机开枪射击，四颗子弹在夜色下划出四道明亮的轨迹，击中了别墅门口四名影子部队队员。
<br/>　　剩下第五名影子部队队员意识到危机，抓起挂在身上的M4A1自动步枪，看向子弹飞来的方向；但是还没有等到他把枪举起，又是两颗子弹从不同的方向飞来，分别击中了他的右前额和左前胸。
<br/>　　钨芯穿甲弹穿透了那名影子部队队员的身体，击中了他身后的围墙，将贴在围墙上的面砖都崩碎开来。
<br/>　　罗昊放下G36C自动步枪，看了眼李然的方向，刚才那两颗击毙第五个影子部队队员的子弹就来自他们两个人手中的G36C自动步枪。
<br/>　　“上！”
<br/>　　罗昊一挥手，四个人快速朝着别墅大门推进过去。
<br/>　　别墅大门门口，五个影子部队队员已经彻底死去，在他们身底下都有着一滩暗红色的血迹。五个影子部队院即便已经死去，但是他们的眼睛依旧睁得滚圆，灰白色的眼睛中早已经失去了生命的光彩。
<br/>　　“狐狸！”
<br/>　　罗昊用手指了指别墅大门，李毅峰会意的点了点头，走上前去，从背包中拿出两块塑胶炸药用力按压在别墅大门上，然后安装上****，连上导线，后撤到门口那四辆改装过的SUV边，拉开罗昊驾驶的那辆改装SUV的后车门，钻进SUV，站在M134转管机枪背后，打开了机枪的保险，右手握着机枪握把，左手拿着起爆器。
<br/>　　“妖瞳，准备好了吗？”罗昊通过耳麦对驾驶着旁边那辆改装SUV的李然问道。
<br/>　　“就等你下命令了。”李然对着罗昊竖起一个大拇指。
<br/>　　罗昊发动了汽车，两道明亮的汽车大灯灯光打在别墅大门上，对着后排的李毅峰低声吼道：“炸！”
<br/>　　李毅峰按下起爆器，引爆了安装在别墅大门上塑胶炸药。
<br/>　　“轰”的一声巨响声中，别墅金属大门被炸飞，罗昊踩下离合器，一拉排档，踩下油门，引擎低沉的咆哮声中，改装SUV冲进了大门背后了院子。
<br/>　　突如其来的爆炸声和炸飞的别墅大门让在院子中巡逻的影子部队队员吃了一惊，全都大惊失色的看向别墅大门的方向，紧接着他们就看到两辆改装SUV冲进了院子，两辆改装SUV的M134转管机枪背后都是站着一个人！而更令院子中这些影子部队队员感觉到恐惧的是，在改装SUV冲进院子的同时，两挺M134转管机枪也是指向了自己这边。
<br/>　　看到六根漆黑的枪管指向自己这边，院子内的影子部队队员只觉得自己手脚冰凉，身体都不听使唤起来。
<br/>　　在M134转管机枪变态的火力面前，不论他们的单兵作战能力怎么样，全都弱小的跟一只蚂蚁一样。
<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　李毅峰和慕容飞几乎同时按下了M134转管机枪的发射键，外部电源驱动机枪枪管转动，并且完成供弹、击发、抽壳等一系列动作。子弹扫向院子中的影子部队队员，那些影子部队队员的身体在M134转管机枪的扫射下疯狂的颤抖着，身体爆出一团团血雾，然后疲软的倒在地上。
<br/>　　面对M134转管机枪这堪称“变态”的火力，人的生命是那么的脆弱不堪，仅仅只是几秒钟的时间，院子中的二十几个影子部队队员就全都成了死尸。
<br/>　　两栋别墅中，又有几名影子部队队员端着武器试图冲出别墅前来增援，可以还没有等他们踏出别墅大门，李毅峰和慕容飞控制着M134转管机枪就指向了别墅。
<br/>　　两道钢铁洪流扫过，别墅墙壁上出现一排弹孔，别墅的落地窗户、窗玻璃全都是被子弹打得粉碎，玻璃碎片掉落了一地。
<br/>　　别墅中的影子部队队员被死死地压制着别墅中动弹不得，根本不敢有任何动作。
<br/>　　罗昊和李然两个人驾驶着改装SUV缓缓前进，改装SUV前进的过程中，李毅峰和慕容飞两个人暂时松开了转管机枪的发射键，让机枪枪管进行冷却。
<br/>　　枪声刚停下，被压制在别墅中的影子部队队员就不安分起来，用自动步枪对着别墅外的改装SUV进行射击；这一次这些影子部队队员身体体会到了什么叫做搬起石头砸自己的脚。

<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　“哒哒哒哒哒哒哒哒哒哒……”
<br/>　　密集的弹雨再次铺天盖地的朝着别墅倾斜而来，别墅的墙壁、大门、地板被子弹打得破败不堪，飞入别墅的子弹打碎了客厅天花板上的水晶吊灯，玻璃碎片掉落了一地；打碎了放在柜子上的醒酒器，装在醒酒器中价值不菲的红酒就这么流到了地板上……
