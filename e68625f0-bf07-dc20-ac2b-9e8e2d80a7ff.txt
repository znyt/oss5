
<br/>　　<br/>　　    三间屋子，两个卧室一间客厅，外加厨房，房子不算大，但对两个人来说刚好够用。<br/>　　<br/>　　    只是小草姑娘明显有点意见想要发表：“孤男寡女共处一室合适么？要不，我给你买个帐篷搭在院子里？”<br/>　　<br/>　　    “你这么说合适么？我受伤了”王惊蛰可怜兮兮的仰着小脸说道。<br/>　　<br/>　　    “好吧，不过有几件事我得事先和你摆明车马了·····”茅小草义正言辞，一本正经的徐徐道来：“我的卧室你不能进，我用洗手间的时候你不能催，我的衣服，个人用品你不能碰，晚上睡觉的时候不能弄出动静吵到我，卫生必须要保持，特别是卫生间里的，关于买菜做饭这点你可以不插手，因为你做的不一定合我的口味，我做的不管好不好吃，做完了你都得要吃下去，嗯，暂时就这么几条，剩下的我随时想到随时添加，哦，对了，最后声明一点，你所有的反抗都是无效的，解释权归我”<br/>　　<br/>　　    王惊蛰确实想要反抗来的，但后来想了想口袋里的钱和他此时的处境，就果断的把反抗压了回去，然后屈辱的点了点头，认了。<br/>　　<br/>　　    王惊蛰含着泪，委屈的说道：“大恩大德，无以为报”<br/>　　<br/>　　    茅小草笑眯眯的点着头说道：“嗯，乖了”<br/>　　<br/>　　    下午的时候，茅小草一直忙里忙外的操持着这三间屋子，王惊蛰想动手帮忙来的，但对方以她的地盘她做主给推掉了，因为茅小草说家里的东西必须由她亲自来放才可以，别人放过之后她容易找不到，比如放厕纸的角度和地方，必须是她伸手就能拿到才行，只有她自己亲自放的东西，她才会自带记忆功能。<br/>　　<br/>　　    后来，小草姑娘在忙活的时候，王惊蛰就坐在自己房间的床上发呆，手边放着王冬至从小昭寺为他借来的摩诃瑜经文，本来王惊蛰打算仔细看看这卷经文对自己到底有什么帮助的时候，但却发现一个字都看不下去，隔壁的屋子总是经常传来“咣当，咣当”的动静，会经常打算他的思路，他索性就干脆坐着发呆了，想等夜深人静的时候再说。<br/>　　<br/>　　    “咚，咚，咚”王惊蛰发呆的时候，茅小草斜靠着门框子，冷笑着说道：“我在外面忙的要死，累的满头大汗，你坐在床上跟个死人似的，你觉得这样好么？你就不能动动手，干点什么？寄人篱下像你这样的，也挺不容易了哈”<br/>　　<br/>　　    王惊蛰连忙从床上下来，点头哈腰的说道：“我想帮忙来的，但你不是说我容易帮倒忙么？屋里的东西必须由你亲自放才可以，我不能插手的么”<br/>　　<br/>　　    “······”茅小草顿时一阵语塞，想了想好像确实是自己这么要求的，可女人从来就不是个讲道理的生物，她们会把一切不在理的地方，全都合理化了。<br/>　　<br/>　　    茅小草说道：“总之，我忙的时候，你闲着就是不行，我看着憋气”<br/>　　<br/>　　    “我擦，这是什么道理？”王惊蛰当时就懵了，憋了半天才不知所措的问道：“不然呢？”<br/>　　<br/>　　    茅小草想了想，歪着脑袋说道：“我干活的时候，你在旁边看着，但你不能挡到我的路，我要是看你看烦了，你马上就得闪开，不能碍眼，还有不能发出动静，我容易烦躁”<br/>　　<br/>　　    王惊蛰懵逼的搓了搓脸蛋子，屈辱的点头说道：“好吧，你说什么你都有理，我穷我寄人篱下，我认了”<br/>　　<br/>　　    茅小草笑眯眯的说道：“嗯，乖了”<br/>　　<br/>　　    一直忙到天黑，茅小草才准备晚饭，两个菜都比较素，只有一碗鱼汤算是荤的，对方的解释是她要保持身材体型，太油腻的东西不能吃，这碗鱼汤就当是给王惊蛰补身子用的了。<br/>　　<br/>　　    对于吃，王惊蛰倒是无所谓了，这几年风餐露宿的时候，他吃的也就比狗粮强点而已。<br/>　　<br/>　　    吃饭的时候，开始两人之间都比较沉默，只有咀嚼的动静，后来几次欲言又止之后，茅小草才按捺不住，主动发问了。<br/>　　<br/>　　    “他们之前说的，是真的么？”茅小草夹着一根青菜，忽然没头没尾的问了一句。<br/>　　<br/>　　    王惊蛰愣了愣，点头说道：“连山和归藏我确实知道一些，不过归藏我差不多读懂了七八成，而连山只有两三成左右，我姐姐知道的倒是会多一些，不过她······”<br/>　　<br/>　　    茅小草打断他，皱眉说道：“我问的不是这个”<br/>　　<br/>　　    “哦，昨天接我们的那个女人，我其实和她不熟的”<br/>　　<br/>　　    “我问的也不是这个”<br/>　　<br/>　　    王惊蛰手里的筷子顿住了，半晌后他放下碗筷，低着脑袋看着桌子说道：“我是有可能活不过三十岁的，之前我爸告诉我赊刀人到了每逢三代的时候必然会受天谴，就是在我三十岁那年，因为赊刀人泄露天机太多，上天要责罚我们，开始我很难接受比较沉闷，后来姐姐告诉我，人生下来就难逃命运的轮回，不管是王侯将相还是贩夫走卒都是命运轮回下的结果，我只是运气不好轮回到了王家的这一代，怨天尤人也没有，这一切都是命啊，就像天要下雨娘要嫁人一样，谁也改变不了”<br/>　　<br/>　　    “没有解么？”<br/>　　<br/>　　    “会有，只是会很难”<br/>　　<br/>　　    茅小草不知为何，幽幽的松了口气，然后说道：“明天我去买只鸡鸭炖了吧，或者你喜欢吃什么也可以告诉我”<br/>　　<br/>　　    “你不是不太吃荤的么？”王惊蛰不解的问道。<br/>　　<br/>　　    茅小草磨了磨牙，说道：“我是怕你活不到三十岁就夭折了，然后赖到我的身上······”<br/>　　<br/>　　    “哪怕我活不到三十岁，生命里也总归会有些值得回忆的东西，比如认识了你”<br/>　　<br/>　　    茅小草拿起碗筷，烦躁的说道：“吃饭，吃饭”<br/>　　<br/>　　    王惊蛰狐疑了，自己刚才那句话说的不够煽情么？<br/>　　<br/>　　    茅小草脑袋里无语的冒出一个念头：“妈蛋的，心真大，都没几年好活的人了，还有心思泡小姑娘呢······”
