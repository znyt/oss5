        耳畔的尖叫声再次响起。
<br/>
<br/>        连星儿觉得自己的耳朵都快要被身旁这群女生的尖叫声给震聋了。
<br/>
<br/>        就在她想抬起双手捂住耳朵的时候，手中的本子和笔被人夺走了。
<br/>
<br/>        当她定睛一看时，只见“C”当着所有女粉丝的面，埋头给她签名。

<br/>        明司寒签的是“余寒犹厉”，字迹行云流水，很漂亮。
<br/>
<br/>        他写完后，便将本子和笔递向连星儿。
<br/>
<br/>        连星儿怔愣地看着面前这个戴着口罩和鸭舌帽的“C”，有些受宠若惊地接过自己的本子和笔。
<br/>
<br/>        同时，也在这一个瞬间，她的心跳蓦然加速了。
<br/>
<br/>        他在赛场上的表现除了优秀以外，而且还很霸气，攻得对方措手不及。

<br/>        而现在，他在众多女粉丝里，只接她的本子和笔签名时，那种独宠的待遇，让她心花怒放。
<br/>
<br/>        明司寒见连星儿拿稳本子和笔后，眼中含笑，转身便走。
<br/>
<br/>        其他女生再次疯狂地尖叫起来，更多是向连星儿投来了艳羡的目光。
<br/>
<br/>        连星儿回过神来后，垂眸看着本子上的签名，兴奋地忍不住亲了一口。
<br/>
<br/>        她的师父给她签名了！
<br/>
<br/>        对了，她应该跟师父要合照留影的。
<br/>
<br/>        想到这里，连星儿心里想着要不要上线约师父来个线下见面。
<br/>
<br/>        虽然她未看清楚师父的真容，但至少师父长得不会丑。
<br/>
<br/>        就算是一张大众脸，她也不在乎，只要能跟师父合张影就好。
<br/>
<br/>        连星儿随即从人群里钻了出来，抱着师父给自己的签名本，高兴地回到了酒店房间。
<br/>
<br/>        她打开电脑，登录游戏上线后，给师父“余寒犹厉”留了言。
<br/>
<br/>        师父，你还记得今天你走过来给签名的那个女生吗？就是我噢！我还想跟你合照留影，你方不方便，出来跟我见个面？就算你戴口罩也没关系。我只想跟你合个影。——Star大小姐。
<br/>
<br/>        留完言后，连星儿的心里莫名其妙地打起了小鼓。
<br/>
<br/>        只可惜，她等了老半天，师父也未上线。
<br/>
<br/>        连星儿也不知道自己是什么精神支持着她，她就趴在茶几上，盯着笔记本电脑荧幕上“余寒犹厉”的游戏头像。
<br/>
<br/>        现实生活中的偶像，她确实比较喜欢洛羽，网络世界里的话，当然要数她的师父了。
<br/>
<br/>        如果师父也活在了她的现实生活圈里，跟洛羽比起来的话，连星儿觉得自己会更加偏向于师父。
<br/>
<br/>        “师父师父，你快上线啊！”连星儿歪着脑袋，无聊地自言自语起来。
<br/>
<br/>        直到她一直盯着的师父的头像突然亮起，她立马就像一只活蹦乱跳的兔子般，精神满满地继续打了一行字过去。
<br/>
<br/>        师父师父，你终于上线了！——Star大小姐。
<br/>
<br/>        明丽五星级酒店，501号房，我现在有空，你可以过来。——余寒犹厉。
<br/>
<br/>        那你等我！我马上过来！——Star大小姐。

<br/>        连星儿回复完后，连电脑都懒得关了，直接拿起手机塞进包包里，抽了房卡出了门。

<br/>        如果不是明丽五星级酒店客房已满，不然，她也住那家酒店里去了。

<br/>        MS战队（黑客部）已经入围了，他们接下来要准备的是半决赛。

<br/>        明司寒和自己的队友们此刻正坐在电视机前，观看其他战队的入围赛。

<br/>        俗话说，知己知彼，才能百战百胜。
