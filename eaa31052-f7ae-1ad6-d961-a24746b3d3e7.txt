        一定是苏漫雪跟他母亲说了什么，才惹得他母亲对他极其不满。
<br/>
<br/>        也对，自从他带着童筱颖离开临海城的傅家大庄园后，他就再也没有回去过了。
<br/>
<br/>        之前，冷太太还当着童筱颖的面，训斥过童筱颖不会抓住丈夫的心，说傅昼景这么久不回傅家大庄园并对她童筱颖不闻不问，一定是她童筱颖这当妻子的哪里做得不够好。
<br/>
<br/>        现在可好，冷太太“被打脸”了，别说那个童筱颖了，就连她的准儿媳妇苏漫雪，也遭受到她儿子傅夜沉和傅昼景对待童筱颖那一样的态度。
<br/>
<br/>        这口气苏漫雪能咽，她这当娘的可不能噎！
<br/>
<br/>        傅夜沉直接拒绝，冷太太便放狠话说，他不亲自来接，就不让苏漫雪来滨江城。
<br/>
<br/>        为了能让筱颖在自己的私宅里住得安心，傅夜沉不得不答应了母亲的这个要求。
<br/>
<br/>        只不过是回临海城一趟，亲自去傅家大庄园里接苏漫雪，没什么大不了的。
<br/>
<br/>        事后，冷太太还安慰了苏漫雪，说她儿子心里还是有她的。
<br/>
<br/>        苏漫雪得知傅夜沉会回来接她，心里顿时乐开了花，同时也为自己精心打扮了一番。
<br/>
<br/>        午餐的时候，童筱颖坐在了傅夜沉的对面，自己低着头，默默地吃饭。
<br/>
<br/>        而傅夜沉，则时不时，目光总会在不经意间地落在童筱颖的身上。
<br/>
<br/>        这是当一个人喜欢上另一个人时才会有的表现。
<br/>
<br/>        就连刘婶都看得出，她家这位大少爷，是喜欢这位大小姐的。
<br/>
<br/>        吃过饭后，傅夜沉让童筱颖下午留在私宅里，明天再和他一起去公司里报到。
<br/>
<br/>        童筱颖答应了，傅夜沉跟刘婶交代了几句，便出了门。
<br/>
<br/>        以后，她便不再是一个人吃饭、一个人待在家里发呆了。
<br/>
<br/>        童筱颖还给傅昼景发了短信，告知傅昼景她已经搬到大哥私宅里来住了，并且，大嫂苏漫雪也会搬过来和她一起住。
<br/>
<br/>        傅昼景只回了一个“好”字，就没下文了。
<br/>
<br/>        童筱颖不知道自己为什么会变成现在这个样子，明明知道昼景不会跟她回过多的话，但仍旧对他抱有期望。
<br/>
<br/>        希望他多多关心她，希望他多多怜惜她，还希望他能给她一丁点的爱。
<br/>
<br/>        但终究，她的希望，还是落空了。
<br/>
<br/>        傅昼景白天和季思妍一起面对商业大佬们谈合作，晚上和季思妍一起在酒店的大床上水乳交融。
<br/>
<br/>        傅夜沉回到临海城的傅家大庄园里是，被冷太太拉着一起吃晚餐，还说是苏漫雪亲自下的厨。
<br/>
<br/>        看着这一桌子色香味俱全的菜，傅夜沉看得出，苏漫雪是下了功夫的。
<br/>
<br/>        但是她的心意，他并不领情。
<br/>
<br/>        在他眼里，苏漫雪只不过是个虚情假意的拜金女。
<br/>
<br/>        一个从骨子里就坏透了的女人，表面再怎么光鲜亮丽，伪装成人畜无害的样子，那也改变不了她在他心里早已种下的认知度。
<br/>
<br/>        傅夜沉突然觉得，他确实不能再让苏漫雪待在爷爷和母亲身边了，万一哪天，这女人又耍起了其它什么花招或是给爷爷和母亲灌了什么迷魂药，让爷爷和母亲真的逼他娶她进他们傅家的门，那岂不是更糟糕？！
<br/>
<br/>        不过，比起上次见苏漫雪，她那满脸浓妆艳抹的精致妆容，现在的她看上去要清淡了许多，而且素颜的样子，若不仔细看，还真就跟筱颖长得一模一样。
<br/>
<br/>        苏漫雪的一颦一笑，就和筱颖犹如一个模子里刻出来的一般。
<br/>
<br/>        “夜沉，你尝尝这道菜。”苏漫雪坐在傅夜沉的身旁，拿起筷子，举止投足之间就像一位受过家教礼仪的千金大小姐般，优雅地替他夹菜。
<br/>
<br/>        就连她的声音……
<br/>
<br/>        也变得跟筱颖很像了……
<br/>
<br/>        或许，成天和苏漫雪待在一起的爷爷和母亲没有听出来，但是他傅夜沉倒是听出了她刻意模仿筱颖的声音的端倪。
<br/>
<br/>        “今晚就留下来过夜，明日再回滨江城也不迟。”
<br/>
<br/>        这冷太太一开口，傅老爷子也跟着点了点头。
<br/>
<br/>        冷太太是希望自己的儿子多多陪陪自己这准儿媳妇。
<br/>
<br/>        而傅老爷子却是希望自己这长孙能多多陪陪他这老头子才是。
<br/>
<br/>        傅夜沉点头同意了，但没有如自己母亲所愿，而是从了自己的爷爷。
<br/>
<br/>        晚饭过后，傅夜沉陪着傅老爷子在后花园里散步。
<br/>
<br/>        傅老爷子又谈起了傅昼景的事情，说傅昼景最近不太安分。
<br/>
<br/>        傅夜沉听着，并不吭声。
<br/>
<br/>        傅老爷子唉声叹气道：“你啊！就是过惯了军旅生活，重情重义，却不知商场不同于战场，这自己人与敌人不易区分，人心险恶啊！”
<br/>
<br/>        “爷爷，阿景是我的弟弟，也是您的孙子。就算阿景有自己的想法，或者是他自己想单干，这些都说明了阿景有上进心，难道不好吗？”傅夜沉帮着傅昼景说话。
<br/>
<br/>        傅老爷子却不以为然地反驳：“他若是只是单纯地有上进心，我就不用这么犯愁了。我担心的，是他日后会把手伸向傅氏集团。”
<br/>
<br/>        “爷爷，就算阿景想要傅氏集团，我分阿景一半，又有何不可？他是父亲的儿子，爷爷的孙子，我的亲兄弟。傅氏集团将来有一半是阿景的，合情合理也合法！”傅夜沉振振有词地说。
<br/>
<br/>        他和阿景，兄弟情深，爷爷为何总要挑拨离间？
<br/>
<br/>        傅老爷子听了傅夜沉的这番话后，那苍老得犹如树皮的手，杵着拐杖，气得狠狠地往地砖上一戳。
<br/>
<br/>        “咚”地一声，傅夜沉瞬间闭嘴了。
<br/>
<br/>        “你爷爷我，为什么只有你父亲这一个儿子？那是因为，你爷爷我不想看到兄弟相残！傅氏集团传承了百年又余，爷爷曾经作为长子，明明已经是傅氏集团内定的继承人了，但是还得和自己那两个不安分的弟弟明争暗斗。事到如今，小景是个什么样的人，爷爷难道会看不清？你看看你堂叔冷岩峰，他的父亲至今都在记恨着你爷爷，还唆使你堂叔冷岩峰对傅氏集团虎视眈眈。傅氏集团从祖辈至今，向来都只有一个继承人，从不被瓜分。就算你父亲在世，他也只会赞同爷爷的做法，把傅氏集团交到你的手里。这点，小沉，你要给爷爷记清楚了！爷爷不允许你对小景心软，傅氏集团的未来只能是你的，未来的未来只能是你长子的。傅氏集团只传嫡长子的制度，永远都不会改变！”傅老爷子吹胡子瞪眼，说得条条是道。
<br/>
<br/>        傅夜沉不得不保持沉默，静静地聆听爷爷的教训。
<br/>
<br/>        原来爷爷对阿景的忌惮，源于爷爷年轻时，与两个弟弟之间的权财纷争。
<br/>
<br/>        傅老爷子冷静下来后，心平气和，又有一丝忧伤地说道：“小沉，这是爷爷最后的希冀。我不希望小沉忤逆爷爷的意愿。”
<br/>
<br/>        傅夜沉低下头去，终于信誓旦旦地应了声：“是！孙儿明白了！孙儿，一定守住傅氏集团，绝不让它被人瓜分。”
<br/>
<br/>        这话，可以说得非常的漂亮，但傅夜沉的心里是矛盾的。
<br/>
<br/>        他不是旧时代的人，不在乎旧时代的规矩。
<br/>
<br/>        但爷爷的话，他又不得不遵循。
<br/>
<br/>        傅氏集团继承人之位只传嫡长子的事情，阿景心知肚明。
<br/>
<br/>        那么，阿景……
<br/>
<br/>        他真的有野心吗？
<br/>
<br/>        将来，他真的会想要跟他这个大哥一起瓜分傅氏集团吗？
<br/>
<br/>        如果，他来真的了，到时候，他该怎么办？
<br/>
<br/>        是坚持爷爷的意愿，还是兄弟合作，一起成为傅氏集团的继承人？
<br/>
<br/>        如果到那时候……
<br/>
<br/>        爷爷已经百年了，只要他和阿景仍旧兄弟情深，一起携手共同执掌傅氏集团，又有何不可？
<br/>
<br/>        在傅夜沉的内心深处，他终究还是选择了要违背爷爷的意愿。
