
<br/>　　<br/>　　    王惊蛰是中午时回来的，回到家里就看见菜刀文躺在沙发上眼巴巴的望着他。<br/>　　<br/>　　    “我以为，你这个点回来了，会把午餐和早餐一起带上，但你两手空空的就回来了，宝宝很桑心知道么”菜刀文随即很慎重的，指着他说道：“还有一件事你得记清楚了，千万不要这么无声无息的就消失这么长时间，你姐姐把你交给我了，我就得为你的安全负责，但我总不能把你拴在裤腰带上啊？所以，哥们下次再出门，咱守时点可以么，长长心吧”<br/>　　<br/>　　    “下次不会了”王惊蛰皱眉说道。<br/>　　<br/>　　    “哎，但愿不会有下次了”<br/>　　<br/>　　    丁武没埋怨他，就问了一句他去哪了，王惊蛰说道：“我觉得这个地方很熟悉，可能以前来过，只是有点想不起来了”<br/>　　<br/>　　    “没事，在川中我们的时间还长着呢，以后我经常带你出去转转······”丁武安慰着说道：“川中其实也没多大，有几天的时间就能转遍了，多走走多看看就行，可能说不上有什么地方就能触动到你了，行了，我叫点外卖咱们吃饭吧”<br/>　　<br/>　　    王惊蛰勉强的笑了笑：“好，不急”<br/>　　<br/>　　    时间一晃过去三天，这天下午四点钟左右的时候，齐泉提前给丁武打了个电话，让他们在楼下等着，说是今天晚上要带他们三个去见为陈先生，就是齐泉上面的大老板。<br/>　　<br/>　　    四点半，齐泉开着奔驰接上三人后，就往川中郊区附近的一片别墅区走，路上的时候跟丁武说道：“我这几天跟老板一直在忙，最近我们谈了个大生意，天天都在外面跑，今天算是才闲下来，老板在郊外的别墅办了个小型的聚会，我正好带你们过去看看，跟他见一面打个招呼，然后我找点事给你做”<br/>　　<br/>　　    丁武呲牙笑道：“泉哥，你老板，那不就是我的大哥大了么？”<br/>　　<br/>　　    齐泉说道：“你和我是兄弟，我老板以后就是你的老板了，咱俩齐头并进往前冲，以后全是大把的好日子等我们呢”<br/>　　<br/>　　    丁武好奇的打听道：“咱老板到底是个啥人，是个多大的人物啊？”<br/>　　<br/>　　    齐泉想了想，说道：“以前川中的刘x听过么？”<br/>　　<br/>　　    “那肯定听过啊，多火的一个人，我在监狱里就总听人提起过他，说他在川地是这个”丁武竖起拇指说道。<br/>　　<br/>　　    齐泉一撇嘴，说道：“他这么牛逼那不也倒了？照样让人收拾了啊，他跟咱老板比就是个弟弟，跟老板不管是喝茶还是喝酒，杯子从来都得矮半截，那还得看老板愿不愿意搭理他呢，小武啊我这么跟你说吧，在川中听过老板名头的人不多，你一提他名字寻常人压根都不知道是谁，像刘x那种人是纯粹被推到台上来当白手套敛财的，我们老板不显山不显水的，那才是高人”<br/>　　<br/>　　    丁武讶异的说道：“老板这么厉害·····那你不也照样蹲了四年大狱么？”<br/>　　<br/>　　    “你给我下套呢，是吧？”齐泉笑骂着拍了他一巴掌，然后正色说道：“你记住，千万不要把自己或者任何人看的太高了，因为你不管处于什么位置，总有人站的比你还高，现在和你说这个还有点早，以后跟我混时间长了，你慢慢的就知道了”<br/>　　<br/>　　    不到六点的时候，齐泉开车进了郊区附近的一个偏僻别墅区，山水环绕风景不错，郁郁葱葱的给人的感觉挺沁人心脾的。<br/>　　<br/>　　    王惊蛰在车里，一路望过来后，忽然说了一句：“这地方，风水不错”<br/>　　<br/>　　    齐泉转过脑袋，调侃着说道：“惊蛰老弟还懂风水呢？我跟你说，这里当初开盘的时候特意从港岛请了个风水大师来看的，具体什么名头我也说不明白，但据说住在这里的人肯定都会顺风顺水的”<br/>　　<br/>　　    王惊蛰也不知道自己怎么就会脱口而出一句这里风水不错的话，他就是下意识的有了这种感觉。<br/>　　<br/>　　    车停到一栋别墅前的时候，旁边已经停了好几辆车，有两台车的牌子还都是个位数的，从落地窗往别墅大厅里看，能看到几道人影坐在沙发上闲聊，旁边的院子摆着几个长条的餐桌，上面已经摆好了酒水和冷餐。<br/>　　<br/>　　    齐泉下了车，领着三人就进了别墅，沙发上坐着几个年纪都四五十左右的中年，茶几上放着几个茶杯，坐在中间的中年梳着短发，鬓角有些发白，右手随意的夹着根烟，看见齐泉他们进来后，就抬手招呼了一声。<br/>　　<br/>　　    “回来了，小泉？”<br/>　　<br/>　　    “嗯，刚回来老板”齐泉笑着回应了一声，然后跟旁边另外几人挨个打了声招呼，就指着丁武说道：“老板，这就是我跟你说的丁武，跟我特别对脾气，这是王惊蛰和小文，他们前两天刚过来的”<br/>　　<br/>　　    “好，能让小泉做为朋友的人都错不了，来自己家里了就别客气，随便转转·······”陈重点头说道。<br/>　　<br/>　　    齐泉的老板见到三人后眼神逐一扫了一圈点了点头，然后就跟旁边的人继续聊了起来，齐泉又把他们给领了出去。<br/>　　<br/>　　    丁武问道：“泉哥，这就算完了？”<br/>　　<br/>　　    “不完了，你还打算坐在那和他们高谈阔论啊？呵呵，我都没资格坐在屋里呢，那些人都是川地这一片的大爷，还有你别以为老板跟你们打了个招呼看起来挺随便的，他看你们一眼就已经记在脑子里了，下次再碰见了他还能叫出你姓啥来，行了，今天就是过来混个脸熟的，过会人多了我再带你认识认识吧”<br/>　　<br/>　　    过了没多久后，别墅院里的人就渐渐多了起来，王惊蛰和菜刀文都不太喜欢这种热闹的场合，一个拎了瓶酒找了个地方自饮自酌去了，王惊蛰就随意的在里面逛了起来，最后来到一处水池边坐了下去。<br/>　　<br/>　　    他对于里面那些什么大人物，其实一点兴趣都没有，因为他发现有那么几个瞬间，他差不多一眼就把很多人都给看透了，面由心生罢了。<br/>　　<br/>　　    面对着可能对自己来讲没有什么秘密的人，他真的是很难提起什么兴致，索性干脆就自己静静的呆会算了。
