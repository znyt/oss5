　　第665章
<br/>　　窗外虫鸣不休，桌前点着一盏油灯，到四更天时分，程大雷才有些微困意，轻轻打了个哈欠。
<br/>　　毕竟兵力悬殊，还需多做准备，这样才能多几分胜算。虽然需要海纳百川，听取众家的意见，但同样自己心里也要有个主意。
<br/>　　直到现在，脑子里还是一团浆糊，在这场即将到来的战斗中，自己并没有多少胜算。程大雷忽然觉得自己有些异想天开，战场上瞬息完毕，而自己却想在战争未开始前，便决定最后的胜局。
<br/>　　这的确是一件不容易办到的事情。
<br/>　　程大雷熬了半夜，还要继续绞尽脑汁的想，为的就是在这件不容易做到的事情中找到些破局的希望。
<br/>　　这时候，柳芷端着茶水从外面进来，将桌上凉掉的茶水换掉，摆上点心，灯芯挑亮，然后将窗户开了半扇，引得凉风进来。
<br/>　　程大雷在书房内绞尽脑汁，柳芷在外间屋里也不能闲着。她要给程大雷烧水，温汤，程大雷饿的时候，她还要准备各种宵夜。
<br/>　　本是夏夜，终究有几分闷热，所以茶炉都要摆在外面。
<br/>　　程大雷看得入神，虽然知道柳芷进来，但却没太多反应。忽然，一阵凉风从身侧吹来，程大雷这才扭过头，见柳芷立在一旁，轻轻摇扇。
<br/>　　她已经有些困了，眼神中露着疲惫，因为炎热的缘故，发丝粘在脸上，下巴垂着汗珠。
<br/>　　本也是十指不沾阳春水的人物，到现在这种端茶递水伺候人的活计已做得十分习惯。
<br/>　　“你要是困了就下去睡吧，不用管我。”
<br/>　　“我不困，只是别打扰大当家做事。”
<br/>　　程大雷本来不太将柳芷放在心上，只是一味做自己的事。可此刻柳芷就立在自己身边，距离近得能嗅到她发丝上的香气。
<br/>　　程大雷的确有些分神，心慌意乱的，面前的材料看不进去。
<br/>　　柳芷心中还是蛮紧张的，她不知道程大雷在做什么事，的确也有些不懂，但心里觉得应该是大事，这看程大雷的紧张就能看出来。她自然不敢去打扰，也知道自己帮不上什么忙，于是就小心体贴的伺候着，算是自己唯一能做的事。
<br/>　　程大雷的注意力已无法集中，总是往柳芷身上瞟。心中又想起前些天发生的事，忽然咯噔一下，如此孤男寡女的，她该不会又要旧事重提么。
<br/>　　这黑天半夜的，自己想躲似乎也没有地方可以躲。
<br/>　　“哈。”程大雷突然打了个哈欠：“这苏樱怎地回事，想走就走了，现在这个局势，她若是有个闪失怎地是好。不行明天得差人将她们接回来。”
<br/>　　柳芷吓了一跳，还以为程大雷要说什么事。她很是想要附和程大雷，可想想自己不该说夫人的坏话。
<br/>　　“外面的确有些乱，还是蛤蟆城安全些，我帮大当家将事记上，明天提醒大当家。”
<br/>　　“嗯。”
<br/>　　程大雷虚伪的点点头，心里也不知道自己是如何想法。若苏樱一直待在府中，他觉得有些麻烦，可苏樱一走，他心里也是空落落的。原也知道，只让人家做金丝雀，是自己的不应该。
<br/>　　不过男女之事么，本就是一团浆糊，程大雷也没能力办的妥妥当当。
<br/>　　麻烦总是有麻烦的，这不苏樱刚走，柳芷这麻烦便显了出来。她立在身侧，看似在伺候自己，却令自己有种如坐针毡的感觉。
<br/>　　可话又说过来，取舍取舍，自己是既不知该如何取，心里也是舍不下。
<br/>　　“罢了罢了，不看了，睡觉。”程大雷将面前的材料合上。
<br/>　　“那个我伺候大当家歇息。”
<br/>　　程大雷心中一紧，想这柳芷该不会是图谋自己什么吧。而柳芷口中的话，很明显和程大雷想的不是一个意思。
<br/>　　她说的伺候是铺床叠被，将程大雷糟蹋的狼藉收拾干净。
<br/>　　“大当家，我外面温着鸡汤，您要不要尝尝？”
<br/>　　熬了半夜，柳芷不说则罢，如此一说，程大雷当真觉得肚子有些饿。
<br/>　　他立在一旁，免得打扰柳芷收拾屋子，口中道：“也好。”
<br/>　　“再有一炷香肉就熬烂了，马上就好。”
<br/>　　柳芷说着话，用扫帚扫下炕上的灰尘，程大雷无事可做，目光不由自主的落在柳芷身上。
<br/>　　她躬着身，绿袍下显现出完美的身体曲线。这时代，男女穿着十分有讲究，如苏樱可以穿红，樊梨花可以穿紫，柳芷这小丫鬟就只能穿些粉粉绿绿的。普通的丫鬟仆人大多是灰黑的麻布袍子。帝国尚黑，包括天子龙袍都是黑色的，程大雷的衣服也是以黑色为主。
<br/>　　眼神从她身上的曲线落到她的脚踝处，又蔓延到她的脖颈。程大雷心中忽然一阵感慨：当真是个大姑娘了。
<br/>　　依稀初见时还是个少不更事，故作成熟的女孩，这几年身体长开了，一举一动都流露出女人的妩媚。倒也不是柳芷故意如此，只是到这个年纪便是理所当然的事。
<br/>　　程大雷脑海里七想八想，或许和珅说得话不错，自己平时做人做事还是太拘着了。如此倒也不是没有好处，上有所好下必笑焉，自己一直这么拘着，手下人在这方面倒也做得很好。
<br/>　　否则的话，一帮大老粗在这种事上肯定早就无法无天了。
<br/>　　或许，就应该给别人做个好榜样。
<br/>　　程大雷忽然觉得有些苦恼起来，明明蠢蠢欲动，却又爱自寻烦恼。
<br/>　　“大当家，大当家……”
<br/>　　柳芷回过身来，是要告诉程大雷，床铺已经收拾好，她这就端汤进来。可连唤了两声，程大雷才回过身来，只见他脸皮发烧，神色慌张，似乎做什么坏事被抓个现行。
<br/>　　“啊。”
<br/>　　二人目光撞在一起，不用说话，程大雷就感觉自己的想法已被看穿。说也奇怪，刚才守了半夜，二人谁也不曾动一点心思，但就是这一刹那的注视，心里那点蠢蠢欲动的小火苗就一下被点燃起来。
<br/>　　程大雷目光有些发直，柳芷的呼吸也凝重起来，她忙垂下头去，迈开步子向外走。
<br/>　　“大当家，汤应该熬好了，我去给大当家端进来。”
<br/>(本章完)
