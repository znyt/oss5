　　上路的香蕉拿下了一血塔的经济，自然是美滋滋，现在他已经两个人头在手，在加上这个一血塔的650块钱，现在无论是装备还是等级，他都已经爆炸好了。
<br/>　　在防御塔被打倒残血的时候，夜雪狂歌就已经后撤了，将这个防御塔的经济让给了香蕉一个人独吃。
<br/>　　而另一边的苹果，在看到香蕉一个人吃下这个上路一血塔的经济之后，这下子是真的绝望了，以鳄鱼现在的这种状态，就算是他和蜘蛛两人来攻击对方，恐怕都要被对方给反杀。
<br/>　　除非楚白衣的沙皇也跟着过来gank，不过现在看起来楚白衣都已经自身难保了，所以让他来上路gank这恐怕十分的困难。
<br/>　　就算是真的来上路gank了，而且gank成功了，恐怕他们这边的中路防御塔也会被偷掉。
<br/>　　而且说不定到时候叶帝还会操控着手中的妖姬去下路gank一波，要知道妖姬这个英雄可是非常灵活的，而且现在同样也是两个人头在手，现在的伤害已经是非常的爆炸了。
<br/>　　所以这一局他们这边崩盘已经是定局了，现在就是要看看他们到底能够坚持多久了，在苹果几人的心里，他们现在能够多坚持一分钟都是一种莫大的胜利了。
<br/>　　“让香蕉拿下一个一血塔的经济后，这下子泰坦更加的难打了，现在就算是蜘蛛过来，也肯定是要被这个等级和装备都要领先的鳄鱼反杀的。”看着场上的局势，米粒开口说道。
<br/>　　“这一局白队的gank思路太明确了，中路靠大魔王打开局面，而夜雪狂歌则是死抓上路，二级抓上，紧接着大魔王抓上，现在夜雪的盲僧六级又抓上，就算是matin过来打这个上单，估计也会绝望的。”林雅楠跟着开口说道。
<br/>　　看着不到十分钟的时间，白队这边就已经领先了2000多的经济，无数的职业选手这个时候也都是暗暗咂舌。
<br/>　　“这可是皇族啊，去年可是打入过世界8强的战队，居然被打的这么惨，这还不到十分钟呢。”
<br/>　　“666666666666，无敌的白队，无敌的荣耀蓝白，今年的比赛可真的是精彩了。”
<br/>　　“如果可以我今年真的想退役休息一年啊，今年我们注定都将是配角，最终决战的只会剩下三支队伍，荣耀蓝白加上一个ST战队。”
<br/>　　“ST战队现在的势头也非常的猛，夏季赛开赛之后，他们也是一路连胜，而且每场比赛的结束时间较之春季赛的时候平均要提前了5分钟，这可是非常厉害的一个数据了。”
<br/>　　“不管怎么说还是咱们华夏赛区的比赛最好看了，如果没猜错的话，夏季赛肯定又是荣耀蓝白两支战队的对决了，能够在赛区比赛场观看到世界决赛级别的比赛，还有什么比这更赚的呢。”
<br/>　　无数的人在那里议论着，到了这一刻在看到荣耀白队的实力也是这样的强劲之后，已经不再有人认为会有什么其他的战队能够阻挡他们的脚步了。
<br/>　　中路叶帝在将兵线推到对方塔下后便是选择了回城，有着蜘蛛在，他就是想推对方的这个中路塔，也是没有机会了。
<br/>　　不过这一波倒是赚了一个上路塔的经济，加上刚刚拿的那个人头，自己都是可以回家好好补一波装备了，等到回到线上在拿一个蓝buff，白衣这小子的沙皇可就更加没有生存的空间了。
<br/>　　而在这一波之后，皇族这边也是更加的猥琐了下来，现在他们只是在塔下清兵，苹果和香蕉两人现在都已经不敢出塔了。
<br/>　　不过他们这样做的直接结果就是视野被无限压制，甚至很多时候当叶帝还有香蕉两人消失的时候，皇族这边都不能捕捉到两人的具体动向，弄的下路的司马贼和白鸽两人也都是战战兢兢的，生怕鳄鱼和妖姬突然来到了下路。
<br/>　　虽然楚白衣已经尽量的在快速的清理着兵线，并没有让叶帝这边的小兵推掉防御塔多少血量，不过他的血量这个时候倒是有些少的可怜了。
<br/>　　原来就在叶帝拿下蓝buff之后，面对着在塔下清兵的楚白衣，叶帝毫无顾忌的用技能在那里消耗着对方。
<br/>　　利用w技能位移靠近，然后qre三个技能甩出，即使e技能不能命中到对方，以qr两个技能的伤害还是能够打掉对方不少的血量。
<br/>　　这样两波下来，楚白衣的血量已经是掉到了半血一下，到了这一刻楚白衣已经是不敢继续在塔下待着了，因为以她现在的血量，只要叶帝再次过来，一套qre连招打中的话，那么他绝对是会阵亡的。
<br/>　　楚白衣可没有那个信心每次都能够躲过对方的e技能，而且这个时候还要时刻防备着夜雪狂歌的盲僧过来gank，以自己的血量，恐怕只要盲僧踢中自己，一套技能就可以直接将自己给秒杀了。
<br/>　　将一波兵线清掉后，楚白衣没敢在线上继续逗留下去，连忙选择了回城。
<br/>　　趁着下一波兵线还有一段时间过来，这个时候回城是最好的选择。
<br/>　　而叶帝这边在看到对方回城后，倒是也没有继续在中路待下去，看了一眼下之后，他便是操控着手中的妖姬绕了一个大圈，直奔这里下路而来。
<br/>　　另一边在楚白衣看到中路两波小兵相遇之后，并没有出现妖姬的身影之后，在看着下路石头人草丛旁那一片漆黑的视野，以及正在那里快速的推着线的女警和扇子妈两人，楚白衣心下顿时一颤。
<br/>　　“妖姬去下路了，赶紧撤！”在这种情况下一状态十分好的妖姬没有在中路清兵，在看看对方下路的动作，楚白衣哪里还会不明白，连忙对着两人提醒道。
<br/>　　其实这个时候不用楚白衣提醒，在看到对方的下路组合这样一反常态的样子后，司马贼和白鸽两人便是已经猜出了大概。
<br/>　　而就在他们两人想要后撤的时候，叶帝已经是操控着手中的妖姬大摇大摆的出现在了他们的后方。
<br/>　　此刻场上的时间已经是来到了15分钟，叶帝手中的闪现和引燃都已经转好了，现在就算是蜘蛛和沙皇从旁边包过来，他也无所畏惧。
