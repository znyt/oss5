        “你如果告知了你妈妈这件事情，只会让你妈妈和你爸爸闹矛盾。因为，你爸爸不把童筱颖当人看。”裴金澈冷不丁地泼冷水。
<br/>
<br/>        夏玄灵顿时懈了气，只好放弃了这个念头。
<br/>
<br/>        比起童筱颖的遭遇，她忽然觉得自己要幸运得多。
<br/>
<br/>        另一边，童筱颖拿着裴金澈的手机，并未给连华生打电话。而是提着药箱，去了一家私人典当行。
<br/>
<br/>        她将手机和药箱当了赎期，并叮嘱当铺的掌柜，药箱得放在冰箱的冷藏室里。
<br/>
<br/>        因为当的钱不多，掌柜便不在意她那药箱里是什么药，索性答应帮她保管一段时间。
<br/>
<br/>        离开典当行的大门后，童筱颖看着眼前来来往往的车辆、稀稀疏疏的人群，以及放眼望去城市高楼大厦的霓虹夜景，心里忽然一阵不知所措。
<br/>
<br/>        她不知道自己该何去何从，也不知道自己接下来该去做什么，眼神有些迷离。
<br/>
<br/>        好像除了那个实验室，这世上就真的没有她的容身之地了。
<br/>
<br/>        童筱颖一个人在大街上走走停停，走过一条街后，又接着走下一条街，直到一条街的人比一条街的少，最后路灯下，就只剩她形单影只。
<br/>
<br/>        “咱们待会去酒吧‘捡死鱼’！放完炮就走。”前面，三个吊儿郎当的男人，一人一手一只啤酒瓶，勾肩搭背，笑得格外轻浮。
<br/>
<br/>        “前面那妹子长得不错。”其中一个男人的目光落在了正在向他们迎面走过来的童筱颖身上。
<br/>
<br/>        “你看看她，连个包都没背，手机也没拿手上。十有八九是‘离家出走’，咱们去搭讪搭讪。”另一个男人提议道。
<br/>
<br/>        于是，这三个男人朝童筱颖围了上去。
<br/>
<br/>        步伐被这三个男人逼停，童筱颖十分淡定地看向这三人丑陋的嘴脸。
<br/>
<br/>        “美女，这么晚了，一个人在外多不安全。不如让我们三送你回家？”
<br/>
<br/>        “是啊！不如先一起去吃个夜宵。”
<br/>
<br/>        “我们请客，就当交个朋友。”
<br/>
<br/>        三个男人一人一句，不停地调侃童筱颖。
<br/>
<br/>        童筱颖淡定地看着这三个男人，刚想说什么的时候，这三个男人顿时惊恐万状，连忙推搡着彼此，不再二话地绕过童筱颖，仓皇离去。
<br/>
<br/>        童筱颖很是无语地侧过头去，在看向那三个仓皇而逃的男人时，一个健硕的胸膛也落入了她的视野中。
<br/>
<br/>        顺着这胸膛的衣襟，视线缓缓上移。
<br/>
<br/>        这个男人的脸……
<br/>
<br/>        薄唇完美，鼻梁挺直，深不见底的眼眸浓烈而深邃，恍若能隐住夜空中的星芒。
<br/>
<br/>        她莫名其妙地怦然心跳，心在隐隐地悸动。
<br/>
<br/>        和梦中的那张脸，一模一样。
<br/>
<br/>        童筱颖忍不住抬起手来，当她想要触碰这张脸时，自己的手却被对方的大手给捉住。
<br/>
<br/>        “筱颖，我好想你。”傅夜沉捉住这只久违的小手后，用力一拉，将这个女人的身子紧紧地拥入怀中。
<br/>
<br/>        他高大英俊，而她因为消瘦而显得格外娇小可人。
<br/>
<br/>        这个怀抱很温暖，还有一丝安全感，让她很舒心。
<br/>
<br/>        “咕——”肚子突然叫了。
<br/>
<br/>        童筱颖尴尬地仰头看着这个男人，无厘头地问了一句：“你有吃的吗？我饿了。”
<br/>
<br/>        从实验室里逃出来后，她就没吃过东西。
<br/>
<br/>        傅夜沉看着童筱颖一脸天真无邪的样子，心里却万般无奈地回了一句：“有！”
<br/>
<br/>        在找她之前，连华生已经间接地告知了他，关于她的所有情况。
<br/>
<br/>        他以为她会很抗拒他，但是她现在表现出来的对他的信任，让他感到很欣慰。
<br/>
<br/>        一路上，他紧紧地牵着她，找了一个二十小时营业的小吃店坐下来吃东西。
<br/>
<br/>        童筱颖毫不顾忌自己的形象，大口大口地吃了起来。
<br/>
<br/>        等到碟子里的蒸饺只剩下最后一个时，她才抬眸看着他，讷讷地问道：“你不吃吗？”
<br/>
<br/>        “我不饿。”傅夜沉微微一笑。
<br/>
<br/>        童筱颖看着他的脸，只觉他的眉宇间透着一股冷峻，但是他对她却露出了温柔的一面。
<br/>
<br/>        吃饱后，童筱颖从衣兜里掏出那张素描画，递给了傅夜沉。
<br/>
<br/>        其实，她一直想知道这张画背后隐藏的答案。
<br/>
<br/>        傅夜沉看着她递过来的纸，怔了一下，才接手打开。
<br/>
<br/>        看到上面是自己的画像，他好奇地问道：“这是谁画的？”
<br/>
<br/>        “我。”童筱颖不假思索地回答。
<br/>
<br/>        傅夜沉会心一笑：“画得真丑。”
<br/>
<br/>        “我为什么会梦见你？”童筱颖十分淡定地问，她不知道自己为什么会跟这个男人自来熟。
<br/>
<br/>        明明对她来说，两个人是第一次见面。
<br/>
<br/>        傅夜沉撑着脸颊，微微歪头看着童筱颖，从容地回答道：“因为我是你的老公。”
