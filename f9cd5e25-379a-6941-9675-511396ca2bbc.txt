　　将现场处理完毕之后，刘哔方才重新回到屋内，清点着这次战斗的收获。
<br/>　　事实上，他斩杀敌人获得的积分并不是很多，基本上也都花费在修补身体的费用里了。
<br/>　　但有完成连锁任务奖励的一万积分，他现在的积分余额，已然成为了一笔巨款，足有一万零八百之多。
<br/>　　这些积分，已经足够从商城内兑换许多东西了。
<br/>　　除此之外，技能，尸气，五颗百年尸丹，鬼修少年身上的随身物品，这些收获的实际用途也不比积分相差多少。
<br/>　　坐在木质的椅子上，刘哔拿出从鬼修少年身上搜索出的几样物品，仔细端详着。
<br/>　　一共有三件，一枚骨头制成的手链，一柄漆黑匕首，一个不知道干什么用的珠子。
<br/>　　这三样物品上面都散发着丝丝缕缕的森然鬼气，尤其是那支手链上面，鬼气似乎都要凝聚成实质一般，隐约之间还有恶鬼咆哮的声音从中传出。
<br/>　　他并不知道这枚骨链的作用，但从上面散发出的波动看来，这绝对是一个不可多得的宝物。
<br/>　　手中把玩着骨链，刘哔从体内汇聚起了一道灵气，输入进了骨链之中，一般灵器也都是如此激发特性的。
<br/>　　但情况却有些出乎他的预料，当猩红色的灵气进入骨链，其内忽的传来了一阵鬼哭般的声音，灵气如同倒入热锅中的冰块一般，瞬间消融。
<br/>　　看到这样的场景，刘哔停下了手中的动作，皱着眉头思索了起来。
<br/>　　片刻，似乎是想到了什么，从体内挤出了一份低级尸气，汇入了手链之中，随后，骨链上逐渐覆盖了一层黑雾。
<br/>　　他的猜想没错，这枚骨链并不属于灵器，而是通过特殊手段炼制而成的鬼器，只有注入属性阴寒的尸气或者鬼器才能催动。
<br/>　　这种鬼器，他从资料上见过，一般炼制手段都十分残忍，大部分都需要特定生辰的活人作为祭品。
<br/>　　但同品质的鬼器，要比灵器的效用好上许多，就连很多正派的修士，都在暗地里进行祭炼。
<br/>　　看到有了效果，刘哔心中略微一喜，加大了鬼气的输出，随后，手链表面上的黑雾愈加浓密，最终所有雾气化为一道黑线，没入了刘哔的身体。
<br/>　　随着这股黑线进入，他明显察觉到自己的身体素质有了一个大幅度的提升，甚至体内之前亏损的气力，也慢慢开始恢复了起来。
<br/>　　是借用提升身体素质吗？不对，不仅仅是这样。
<br/>　　刘哔心头一动，拿起桌上的匕首，尝试着在自己胳膊上划出一道伤口。
<br/>　　一阵金铁声音响起，锋利的匕首落在刘哔胳膊上，却只留下了一道白痕，他有些惊讶，不知道自己的身体素质为什么突然变得如此强横了。
<br/>　　想到这炳黑色匕首也可能是一件鬼器，刘哔抽出些许尸气注入进了匕首中，随后黑芒大盛，匕首有些晦涩的切进了他的皮肤。
<br/>　　可惜，只是一件低级鬼器，攻击力算不上强，也只能勉强破自己防御罢了。
<br/>　　叹了口气，刘哔随手将匕首扔进空间戒指，便专心致志的查看着自己胳膊上的伤口。
<br/>　　没等待太长时间，他胳膊上的伤口以一个肉眼可见的速度迅速恢复着，看到这里，刘哔也确定了心中的想法，自己身体的恢复力也增强了许多。
<br/>　　这一切都是骨链带来的效果，虽然激发时间稍微长了一些，但对于刘哔来说，它的作用还是相当大的。
<br/>　　满意的将骨链戴在了手腕上，刘哔拿起最后一件物品，在眼前仔细端详着。
<br/>　　这是一个通体灰色的珠子，有点像尸丹，但上面的尸气却没有那般斑驳，但不管是刘哔往里注入尸气或者灵气，它都没有半点反应。
<br/>　　鼓捣了半天，依旧没有一点头绪，刘哔索性将珠子连带着五枚尸丹一起，收入了空间戒指中，不再理会。
<br/>　　毕竟是鬼修的原住地，他不想在这里停留太久时间，以免引起什么不必要的麻烦，这枚珠子，等以后有时间了再查看也不迟。
<br/>　　将战利品清点好，刘哔打开了技能面板，找出新获得的两个技能查看着。
<br/>　　名称：坚韧
<br/>　　级别：E级
<br/>　　分类：被动技能
<br/>　　介绍：来源于《僵尸先生》位面，可大幅提升宿主身体坚韧度。
<br/>　　名称：鬼步
<br/>　　级别：C级
<br/>　　分类：主动技能
<br/>　　介绍：来源于《僵尸先生》位面，可消耗能量，大幅提升宿主速度，并消除行动痕迹。
<br/>　　看到两个技能的介绍，刘哔心中有些兴奋，这两个技能，都能给他带来很大的帮助。
<br/>　　‘坚韧’技能提升身体坚韧度，大幅度增加自己的防御能力，最重要的是，这是一个被动技能，也不需要花费刘哔能量或者别的什么来进行维持。
<br/>　　而‘鬼步’技能，可以提升速度，并且消除行动痕迹，如果使用得当，这个技能的实用性就很广泛了。
<br/>　　关闭技能面板，刘哔打开了商城，他想去购置一些东西，顺便将之前的任务奖励给使用了。
<br/>　　他还记得，还有一次抽奖机会没有提取，也不知道能抽出什么东西来。
<br/>　　浏览了一遍商城，他也选定了几样物品，一万积分听起来很多，但实际使用起来，就有些不够用了。
<br/>　　先是花费一千二百积分兑换了四张的低级技能升级卷轴，将‘吞噬’‘鬼手’‘坚韧’都提升到了D级，增强了技能效果。
<br/>　　这些技能都非常实用，花费积分提升等级也很划算，而‘不死’技能属于特殊技能，无法用卷轴进行提升。
<br/>　　本来他还想要兑换一些中级技能升级卷轴，但看了看价格，无奈放弃了心中的打算。
<br/>　　中级技能卷轴可以升级B级以下的所有技能，可一张就要三千积分，剩下的九千积分全花进去也只能兑换三张。
<br/>　　提升三次中级技能，实际增加的战力并不算高，相比较，就不如用这些积分兑换其他东西更加划算了。
<br/>　　精挑细选了半天，刘哔终究选中了一张面具，一对翅膀。
<br/>　　面具只是个C级物品，价格也不算昂贵，仅仅花费了六百积分，效果也只是可以遮挡自身所有的气息。
<br/>　　但那对翅膀，就贵的离谱了，花费了他整整九千积分才兑换了出来！
<br/>(本章完)
