　　三天后，八区的一家颇具规模的拍卖行里，一场拍卖正如火如荼的进行着。
<br/>　　这是东京颇具盛名的拍卖行，很多大型的拍卖都曾在这里举行，想要在这里举办一次拍卖，不知道要花费多少金钱。
<br/>　　但所有人不知道的是，这座拍卖行背后的操纵者，实际上是一群由喰种组建起来的地下势力。
<br/>　　夜，拍卖行附近静谧无声，没有一点灯光，但拍卖行内部，却人声鼎沸，热闹无比。
<br/>　　哦不，不能说人声鼎沸，这些发出声音的，都是小有些身份的喰种们。
<br/>　　一条迂回的走廊，连接着富丽堂皇的拍卖会内部，这是一座能容纳千余人的超大型场所。
<br/>　　不知何等木材制成的雕花椅呈圆形往外扩散，正中心的圆台一片玉白。
<br/>　　细微的水流从边缘流下，几百根悬挂彩石的吊灯如同星星，绽放着光芒。
<br/>　　刘哔携着什造进入此地时，大部分的座位已经被喰种占据，嘈杂的声音此起彼伏。
<br/>　　有些没找到作为的独行者双手抱胸，面容阴翳的倚靠在墙壁上，还有主营商业的喰种，四处走动，满脸和善，到处为自己拉拢着人脉。
<br/>　　一些带着小丑面具的少年，推着盛满点心的推车，在座椅间相互穿梭着。
<br/>　　小推车从刘哔面前经过，带来了一阵甜美的血液味道，刘哔咽了口唾液，伸手想要拿上一盘，却又想到了什么，收回手来。
<br/>　　算了，什造在。
<br/>　　作为小丑组织曾经强力的盟友之一，月山习能为刘哔准备的，自然是单独的一件包厢。
<br/>　　这倒也避免了一些麻烦。
<br/>　　两人顺着过道，一路向着楼梯走去。
<br/>　　途径一些喰种的时候，受到什造身上人类气味的影响，不时有喰种投来不怀好意的目光，嘴角更是有透明的津液流淌下来。
<br/>　　对于人肉的气味，他们压根不会有一点抵抗能力。
<br/>　　明显感觉自己握着的手掌攥紧了一些，刘哔皱了皱眉头，目光冷厉的环视了一周，一股血腥的气息从身上散发出来。
<br/>　　有了这番警告，周围的喰种老实了许多，却依旧有些胆大之辈，对此毫不在意，依旧投来富有侵略性的眼光。
<br/>　　只是，这次的视线，由什造转移到刘哔身上了。
<br/>　　“这些家伙好烦啊，想杀了它们。”什造撅了撅嘴巴，脸上带着不高兴的表情，用只有两人能听到的语气说道。
<br/>　　说起来，好多天没和人战斗，她有些怀念鲜血的味道了。
<br/>　　“好啊。”刘哔微微一笑，宠溺的捋了捋什造的头发，附和道：“一堆垃圾而已，杀便杀了。”
<br/>　　瞬间，两人周遭区域内的声音立即安静了下来，所有的喰种，齐刷刷的注视着两人。
<br/>　　带着人类仆从的喰种，他们见过，毕竟总有些喰种的口味不同。
<br/>　　但刘哔和什造，明显不是那种关系，看起来，似乎像是恋人。
<br/>　　要知道，喰种世界是明令禁止喰种和人类结合的，这是很久之前就留下来规矩。
<br/>　　虽说依旧有些喰种破坏规矩，但从没见过有谁敢这么大摇大摆的显露出来。
<br/>　　就不怕引起V组织的追杀么。
<br/>　　“真的可以吗，那什造可要动手了哦。”什造双眼开始发亮，有些兴奋的看着刘哔。
<br/>　　虽说还没有动手，但是内心有些迫不及待了呢。
<br/>　　“还是不要了，毕竟你身上的伤还没好，而且，你妈妈还没找到，别耽误了正事”带着一抹微笑，刘哔牵起什造的小手，声音如同暖阳一般温柔。
<br/>　　周围的喰种一阵唏嘘，这个家伙，真当自己是什么人物了，竟然想全杀掉他们。
<br/>　　先不要说能参加拍卖会的，都是小有来头的喰种，谁没有点保命的本事，更不要说，拍卖会也不会放任这样的情况发……
<br/>　　“啪！”
<br/>　　清脆的响指声从会场响起，明明声音不是很大，却让会场内的所有人都听的清晰。
<br/>　　原本还在看着两人的喰种们，瞬间停止了所有动作，就如同被时间静止了一般，保持着原本的姿势，在座位上一动不动。
<br/>　　“蹬……蹬……蹬……”
<br/>　　牵着什造柔软的小手，两人头也不回的踏上了台阶，似乎被两人的动作带动，身后那一片区域的喰种们，瞬间裂成了一块块碎肉。
<br/>　　小溪般的鲜血瞬间倾洒在了地上，将地面染成了红色，看到这幅场景，其余区域的喰种们发出了一阵尖叫。
<br/>　　这一个区域，至少有二三十头喰种，居然就这样，被抹杀掉了？！
<br/>　　那个男人，到底有多么可怕。
<br/>　　丝毫没在意身后的动静，刘哔领着什造上了楼，却没进入自己所属的房间，而是挨个包厢转了一圈。
<br/>　　他在寻找，抚养什造的那位喰种的位置，但很遗憾，那家伙好像没资格进入包厢。
<br/>　　“二位，是找不到包厢号了吗，请出示一下您的邀请函。”一个衣着暴露的妙龄女子，含笑走到两人面前，颇为礼貌的说道。
<br/>　　不只是衣着暴露，甚至就连面具遮盖的部位都很少，也只是说堪堪遮住了双眼而已。
<br/>　　除了胡桃夹子，还有哪个喰种像她一样，穿着个胸罩和裤衩子就跑出来了。
<br/>　　额……当然，只是吐糟，不过话说起来，倒还有些性感。
<br/>　　这个身材，前凸后翘的，不去拍AV都有些可惜了。
<br/>　　“哦，我们是想找一个人，请问这位女士，你是否能告知我们一下呢？”刘哔从怀中掏出月山习之前给他准备的邀请函，递给胡桃夹子，十分客气的问道。
<br/>　　会场上这么多喰种，要一个个根本就不现实，倒还不如问问这里的服务人员，或许能找到一些线索。
<br/>　　接过刘哔递过来的邀请函，胡桃夹子查看了一下，眼中露出一抹惊异。
<br/>　　这个编号对应的喰种身份是……修罗！
<br/>　　怪不得，刚才在下面，这家伙这样肆无忌惮的出手。
<br/>　　像这样的人，估计行事也不会有那么多顾忌。
<br/>　　毕竟实力摆在了这里。
<br/>　　只是……他脸上的面具，却和资料上描述的不太一样呢。
<br/>　　是不想暴露自己的身份么。
<br/>　　可自从月山家覆灭之后，修罗的资料就被列入了各大势力的警戒名单，拍卖会还没开始的时候，小丑高层就已经下令，见到这个编码的人必须立即上报。
