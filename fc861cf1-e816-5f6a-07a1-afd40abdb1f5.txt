　　“我顶着，你去找人！”道风说完，推了徐福一把，进入过道深处，自己却堵在了入口，祭出三花聚顶，形成一道极为强大的结界，护住周身，就算是两大尊者联手，一时间也无法堪破结界……
<br/>　　建文帝也手持鱼肠剑，在他身前用剑横扫，布置出了剑阵，与道风的三花聚顶融合在一起，加强着结界的力量。
<br/>　　“道风，你做出如此违逆之事，是要做什么！！”开道尊者吼了起来。
<br/>　　“救人。”道风缓缓说道。
<br/>　　“救人，救谁？”
<br/>　　道风不再说什么。
<br/>　　明法尊者猛然想到什么，失声道：“进去那人，莫不是术士徐福？”
<br/>　　他并不认识徐福，徐福之前来闯昭狱的时候，他还不在这里当差，不过也听人描述过徐福的相貌，而且，昭狱数百年来都是很安全的，从来没发生过劫狱的事，近一个月来，却已经发生两起了，这不得不让他把两件事连在一起，自然就想到了徐福。
<br/>　　“徐福……他是要抢夺白起的元神，快！你速去通知菩萨！”明法尊者说完，双手合十，弹了几下手指，双手之间结出了一颗灵珠，朝着道风用三花聚顶布置成的结界打去，开道尊者飞身下楼，前往地藏宫。之所以他自己亲自去找地藏王菩萨，是因为地藏王菩萨的修所道场，一般人根本进不去。
<br/>　　轰的一声巨响，灵珠在结界上打出了一个缺口。道风还能支撑，建文帝却是喷出了一口鬼血，身体摇摇欲坠，只有依靠鱼肠剑的灵力勉强应付。
<br/>　　灵珠不断旋转，持续消耗着结界的法力。
<br/>　　“道风，你这么做，当真不怕上天降罪、业火焚身吗？”明法尊者冷声呵斥道。
<br/>　　“天是什么？”道风冷淡淡说道。
<br/>　　明法尊者怔住。
<br/>　　双方相持了一段时间之后，从道风身后的回廊里，一道人影冲了出来，正是徐福。
<br/>　　“走！”徐福只说了这一个字，冲出结界，手中判官笔横扫过去，击打向明法尊者的脑袋。
<br/>　　明法尊者收回灵珠，在灵珠上吹了一口气，一时间光华大放，托住了判官笔，不顾徐福也没有进一步进攻，而是借这个机会，朝楼下飞遁而去。
<br/>　　道风和建文帝紧随其后。
<br/>　　从昭狱出来，立刻被外面赶来的阴兵鬼将围住，三人冲开了一条路，徐福先遁走，道风和建文帝刚要走，突然一声怪吼，从遥远的方向传来。
<br/>　　这一声吼叫之中，充斥着一股恐怖的威势，道风和建文帝心神一震，围攻他们的那些阴兵鬼将却是多数被震翻在地，捂着两只耳朵，脸上露出痛苦的表情。
<br/>　　道风二人循着声音想起的方向望去，只见一道赤红色的火焰，从枉死城南边的方向疾飞而来，如同一朵火烧云，擦亮了整个天空。
<br/>　　明法尊者一眼看到这团火焰，立刻喜上眉梢，朗声说道：“圣兽且来，妖邪之辈在这里！”
<br/>　　“这是什么！”建文帝听到明法尊者的喊声，再看到天上的奇景，内心也是忍不住有点颤抖。
<br/>　　道风转头望着那团火焰，喃喃道：“地狱犼，地藏王菩萨的坐骑。”
<br/>　　“什么！”建文帝震惊。
<br/>　　那团火焰极快地飞来，落在枉死城的上方，化作一副巨大的身体：浑身都是火焰组成，宛如披着一层重重的铠甲，却长着一只狗头，两眼如铜铃，从长大的嘴巴里，不断流出哈喇子一样的火焰。
<br/>　　这头地狱犼的形象，极其威严和残暴，让人看上一眼就心惊肉跳。
<br/>　　在场的很多阴兵鬼将也都呆住了，他们也是第一次见到传说中的地狱犼，一个个心惊胆战而又十分出神地望着这个庞然大物。
<br/>　　地狱犼……
<br/>　　地藏王菩萨，是阴司佛门的老大，十八层地狱和轮回司、昭狱等几个衙门的实际掌控者，跟酆都大帝平起平坐，能有资格做他的坐骑，自然不可能是一般的生灵。
<br/>　　道风也是第一次见到地狱犼，不过他听说过这头凶兽的来历，传闻犼是上古凶兽，在上古之战的时候，曾经帮助过蚩尤，后来蚩尤兵败，犼被灭了肉身，只剩下元神，被打入地狱之中，万劫不复，后来地藏菩萨接管地狱，渡化犼，以地狱烈火为其重塑形体，渡化进佛门，成为自己的坐骑。
<br/>　　这犼平时镇守在地狱深层，一般不会来到阴司，不知道今天怎么会在这阴司里……
<br/>　　“速走！”徐福喝了一声，用判官笔在前方开路，将那些还在发愣的阴兵鬼将尽数荡开，自己先冲了出去，道风和建文帝紧随其后。
<br/>　　就在这个时候，地狱犼附身冲过来，一只巨大的手掌，朝道风和建文帝拍了过来。
<br/>　　道风知道避不开，反手拍出一掌，三清鬼符从掌心飞出，一上两下，形成一个三角形状，凌空旋转起来，看似十分微小，与那地狱犼的一只巨手大小完全不成正比，但结果是，居然生生托住了地狱犼的巨掌。
<br/>　　地狱犼一张犬模样的脸上，露出了狰狞的表情，有些诧异地看着道风。
<br/>　　“你也就是个子大一点。”道风冷笑。
<br/>　　地狱犼把手掌收回去，然后重重地拍打在三清鬼符上，口中发出了沉重的吼叫，手掌之上，顿时流火四射，朝着叶少阳三人打过来，叶少阳用三清鬼符死死顶住，咬牙对建文帝说道：“走！”
<br/>　　建文帝知道这地狱犼根本就不是自己能够对付的，也不跟道风客气，抽身而退，开道和明法两位尊者立刻展开了追击。
<br/>　　建文帝用鱼肠剑跟他们拼了一记，差点被搭得魂飞魄散，眼看着就要支撑不住，还是徐福回来，一把捞起他，同时用判官笔在空中写了一个“玄”字，形成一道强大的不定符，等两位尊者堪破符印，两个人已经消失了。
<br/>　　那边，地狱犼用一只手掌不断拍击着三清鬼符，将道风生生逼退了几丈远。开道和明法两位尊者丢失了目标，只好来围攻道风。
<br/>　　尊者，是佛家无上境界，地位还在罗汉之上，平均实力与道门的上仙并列，是人类几乎不能达到的修炼极限。
